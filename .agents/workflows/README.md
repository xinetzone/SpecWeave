# 工作流索引

本目录定义了多智能体协作的标准工作流，用于规范各角色在常见开发场景中的协作方式。

## 工作流清单

| 工作流 | 适用场景 | 参与角色 | 入口 |
|---|---|---|---|
| 功能开发 | 新功能开发 | 全部角色 | feature-development.md |
| 代码审查 | PR 审查 | developer, reviewer, orchestrator | code-review.md |
| 测试流程 | 测试执行 | tester, developer, reviewer | testing.md |

## 使用说明

1. **选择工作流**：根据当前任务类型，从清单中选择对应的工作流。
2. **阅读流程概览**：通过 Mermaid 流程图了解整体执行顺序。
3. **明确角色职责**：查阅角色参与表，确认各角色的输入、输出与职责。
4. **遵循步骤详情**：按步骤说明执行任务，确保每一步的完成标志达成。
5. **执行交接协议**：步骤间交接时，使用 `templates/handoff-template.md` 模板并遵循 `protocols/handoff.md` 协议。

## 与其他资产的关系

- **协议**：工作流执行过程中的交接与通信遵循 `protocols/` 目录下的协议定义。
- **模板**：任务创建与交接使用 `templates/` 目录下的标准模板。
- **角色**：各角色的能力定义与职责边界参见 `roles/` 目录。
