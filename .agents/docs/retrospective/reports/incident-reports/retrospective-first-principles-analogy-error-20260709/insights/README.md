# 洞察索引：第一性原理类比推理错误事件

本事件复盘提炼出 **3个核心洞察**，全部为方法论践行类洞察。

| 洞察ID | 标题 | 核心内容 | 相关模式 |
|--------|------|---------|---------|
| 洞察01 | 践行鸿沟：知道≠做到 | 陈述性知识→程序性知识→自动化执行存在三层差距，新学方法论不会自动变成习惯，需要强制检查点 | first-principles-prompt-pattern |
| 洞察02 | 简单任务陷阱 | 低认知负荷任务（格式更新、批量修改）因为跳过验证环节，错误率反而可能高于有流程保护的复杂任务 | 待沉淀：决策前三查检查清单 |
| 洞察03 | 提问式纠错 | 苏格拉底式提问（"这个判断哪里来的？"）比直接指正更能触发深度反思和第一性原理思考，且不触发防御心理 | adversarial-review-prompt-pattern |
