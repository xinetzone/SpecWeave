# 模式引用1：文件创建前置检查模式

**模式名称**：file-creation-precheck-pattern

**来源**：[全局模式库](../../../../patterns/methodology-patterns/governance-strategy/file-creation-precheck-pattern.md)

**核心描述**：在创建任何新文件前，强制执行三步检查流程：①查阅分类体系确定归属目录；②查阅命名规范确定文件名格式；③运行自动化验证脚本确认合规性。

**成熟度**：L2（已验证）

**适用场景**：
- 创建新文档
- 创建新代码文件
- 文档迁移和重命名
