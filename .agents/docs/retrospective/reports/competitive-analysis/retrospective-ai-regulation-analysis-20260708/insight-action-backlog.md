## 行动项详情

### CMP-001: 建立极端情绪/自残自杀干预机制
- **优先级**: 🔴最高（涉及生命安全，第13条，第三梯度罚款可达20万）
- **来源**: 报告§5.3 高风险遗漏项#1
- **法规依据**: 《办法》第13条
- **风险说明**: 涉及生命安全，未建立干预机制可能产生严重法律后果和社会责任，是公告完全未提及的最大遗漏
- **执行方案**:
  1. 建立极端情绪/自残自杀关键词词库
  2. 实现对话内容实时监测
  3. 触发后弹出心理援助资源提示
  4. 建立人工介入流程
  5. 支持紧急联系人通知机制
- **DoD**: 关键词识别→提醒→人工介入→紧急联系人全链路打通；有测试用例验证
- **技术方案**: [AI拟人化互动服务极端情绪干预机制技术实施方案](../../../../knowledge/best-practices/ai-anthropomorphic-crisis-intervention-implementation.md)
- **执行结果**: 详细技术实施方案已完成（含四层架构、三级词库、评分算法、分级响应SOP、UI设计、紧急联系人机制、人工介入流程、测试验收标准、7天上线排期），待开发团队按方案实现
- **产出物**: 技术实施方案文档、关键词词库、System Prompt安全指令、紧急干预UI、紧急联系人功能、人工坐席台、运营SOP
- **截止日期**: 2026-07-15（法规施行日）

### CMP-002: 完成算法备案手续
- **优先级**: 🔴最高（法定强制义务，第26条）
- **来源**: 报告§5.3 高风险遗漏项#2
- **法规依据**: 《办法》第26条
- **风险说明**: 算法备案是法定义务，未备案可能导致服务下架，属于"可直接检查"的硬性合规项
- **执行方案**:
  1. 登录国家网信办算法备案系统
  2. 准备服务属性、算法类型、训练数据等备案材料
  3. 提交备案申请
  4. 获取备案号后在服务显著位置公示
- **DoD**: 取得算法备案号，在应用内公示备案信息
- **执行结果**: -
- **产出物**: 算法备案公示页面、备案凭证
- **截止日期**: 2026-07-15（法规施行日）

### CMP-003: 实现2小时使用时长提醒
- **优先级**: 🔴高（可量化硬性指标，第18条）
- **来源**: 报告§5.3 高风险遗漏项#3
- **法规依据**: 《办法》第18条
- **风险说明**: 2小时是明确量化指标，技术实现成本极低但监管检测容易发现违规

  2. 满2小时弹出醒目提醒（建议休息、防止沉迷）
  3. 同时实现AI生成内容显著标识
- **DoD**: 连续使用2小时强制弹出提醒；有自动化测试验证
- **执行结果**: -
- **产出物**: 时长提醒功能
- **截止日期**: 2026-07-15（法规施行日）

### CMP-004: 建立安全评估5种触发情形识别机制
- **优先级**: 🔴高（核心合规义务，第22条）
- **来源**: 报告§5.3 高风险遗漏项#4
- **法规依据**: 《办法》第22条
- **风险说明**: 不知道何时需要评估，可能出现"上线未评""规模达标未评"等严重违规
- **5种触发情形**:
  1. 服务上线投放前
  2. 功能目的、模型类型、交互方式发生重大变化
  3. 拟增设虚拟亲属/虚拟伴侣等功能
  4. 注册用户超过100万或月活超过10万
  5. 发生违法违规事件或出现重大安全风险
- **执行方案**:
  1. 在产品运营流程中嵌入评估触发检查点
  2. 建立用户规模监控预警（90万/9万预警线）
  3. 功能变更评审增加合规评估环节
  4. 安全评估报告提交省级网信部门
- **DoD**: 有明确流程确保5种情形均触发评估；评估报告按要求提交省级网信部门
- **执行结果**: -
- **产出物**: 安全评估触发流程文档、评估报告模板
- **截止日期**: 2026-07-15（流程建立），触发时执行评估

---

### CMP-005: 按8项内容准备安全评估报告
- **优先级**: 🔴高（评估报告质量直接影响能否通过审查，第23条）
- **来源**: 报告§5.3 高风险遗漏项#5
- **法规依据**: 《办法》第23条
- **风险说明**: 评估报告缺项漏项无法通过审查，导致服务无法上线或被下架
- **8项评估内容**:
  1. 安全保障措施落实情况
  2. 训练数据处理情况
  3. 用户权益保护情况
  4. 内容安全管理情况
  5. 特殊群体保护情况
  6. 应急处置机制建设情况
  7. 日志留存情况（不少于6个月）
  8. 法律法规遵从情况
- **执行方案**:
  1. 按8项维度建立评估报告模板
  2. 上线前对照逐项自评估
  3. 评估报告由合规负责人审核签字
- **DoD**: 评估报告模板覆盖8项内容；首次上线前完成自评估报告
- **执行结果**: -
- **产出物**: 安全评估报告模板、首次自评估报告
- **截止日期**: 首次上线前（或2026-07-15前完成模板准备）

### CMP-006: 补全内容安全审核7类禁止内容
- **优先级**: 🟠中高（内容安全红线，第8条）
- **来源**: 报告§5.3 高风险遗漏项#6
- **法规依据**: 《办法》第8条
- **风险说明**: 公告仅提及3类（暴力色情、危害安全、不良内容），遗漏国家安全、恐怖主义、民族仇恨、虚假信息等高危类别
- **7类禁止内容**:
  1. 危害国家安全、荣誉和利益
  2. 颠覆国家政权、推翻社会主义制度
  3. 宣扬恐怖主义、极端主义
  4. 宣扬民族仇恨、民族歧视
  5. 传播淫秽色情、暴力信息
  6. 传播虚假信息、扰乱经济社会秩序
  7. 其他违法违规内容
- **执行方案**:
  1. 扩充内容审核词库和规则覆盖全部7类
  2. 更新内容审核模型训练数据
  3. 建立人工复核机制
- **DoD**: 内容审核规则覆盖7类禁止内容；有测试用例验证各类别拦截效果
- **执行结果**: -
- **产出物**: 更新后的内容审核规则、词库
- **截止日期**: 2026-07-15（法规施行日）
