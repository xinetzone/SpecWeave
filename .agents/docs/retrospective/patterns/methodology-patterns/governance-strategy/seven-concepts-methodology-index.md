# 七概念方法论体系索引

## 方法论体系概述

七概念方法论体系是一套基于第一性原理构建的AI智能体协作与知识沉淀方法论，围绕复盘(R)、洞察(I)、萃取(E)、原子提交(C)、原子化(A)、第一性原理(F)、对抗性审查(V)七个核心概念，建立从感知到认知、验证、执行、沉淀的完整闭环，解决智能体任务执行中经验复用率低、质量不可控、知识沉淀碎片化的问题，为SpecWeave项目提供可复制、可验证、可演进的治理框架。

## 快速入门指南

### 新手阅读路径
1. **第1步（15分钟）**：先读「复盘报告」了解这套方法论从何而来、解决什么实际问题
2. **第2步（20分钟）**：读「定位模型」建立七概念的整体认知框架，记住五个层级和七概念缩写
3. **第3步（10分钟）**：对照「决策树」判断你当前任务属于哪种场景，应该启动哪几个概念
4. **第4步（按需）**：根据场景找到「核心工作流」中的对应流程，按步骤执行
5. **第5步（执行中）**：参考「交互规范」确保概念间输入输出格式正确
6. **第6步（交付前）**：用「质量标准」检查清单逐项验收
7. **第7步（进阶）**：阅读「对抗性审查」了解方法论的已知局限，避免教条使用

### 使用建议
- 从简单场景开始：先练单一概念（原子提交C或原子化A），再练双概念组合
- 每次执行后用质量标准自检，逐步形成肌肉记忆
- 遇到问题回溯定位模型，检查是否违反了层级依赖关系
