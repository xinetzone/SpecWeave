## 快速开始

### 个人自学
1. 阅读[七概念方法论索引](../seven-concepts-methodology-index.md)了解框架
2. 打开[练习题目](seven-concepts-practice-01.md)，按步骤独立完成
3. 做完后用[评分标准](seven-concepts-rubric-01.md)自评
4. 最后对照讲师指南中的参考思路

### 团队培训（讲师）
1. 课前准备：阅读[讲师指南](seven-concepts-facilitator-guide-01.md)，亲自做一遍练习
2. 准备材料：打印练习题、评分表、大白纸+马克笔
3. 按讲师指南中的时间安排执行（90分钟快速版或3小时深度版）
