# 实战演练：「反效率」知识IP的朋友圈文案创作与审查

> **配套模式**：[正言若反写作法](../zhengyan-ruofan-writing.md) + [约束审查三轨制](../three-track-constraint-review.md)
> **评分标准**：[Rubric评分细则](zhengyan-three-track-rubric-01.md)
> **难度**：中级 | **预计时间**：60-90分钟 | **验证状态**：L1-draft

---

## 一、项目背景

你是一个主打**「反效率/慢哲学」**的个人成长博主，受众是被996压得喘不过气的职场人。你的核心理念不是"慢下来是为了更快"（这本质上还是效率逻辑），而是"工作本身可以有不同的节奏，不一定要赶时间"。

你正在为新课《不赶时间的工作日：每天工作3小时也能出成果》做朋友圈发售预热，需要写**3条核心传播文案**。

---

## 二、约束条件（严格执行）

### 2.1 哲学立场约束（反本质主义·体验导向）

| 禁止的立场 | 说明 | 正确方向 |
|-----------|------|---------|
| ❌ 本质化承诺 | 不能说"慢下来你会更高效/赚更多/更成功" | 描述体验和状态，不承诺结果 |
| ❌ 终点化暗示 | 不能说"做到X你就自由了/开悟了/解脱了" | 强调这是一种持续的实践，没有终点 |
| ❌ 权威姿态 | 不能说"听我的/我告诉你/我教你" | 用"我观察到/我陪你看见/也许你也可以试试" |
| ❌ 普世化断言 | 不能说"所有人都应该慢下来/这是唯一正确的路" | 加边界条件："对我而言/在我身上/如果你也…" |
| ❌ 科学化包装 | 不能用"神经科学证明/心理学研究表明"来背书 | 回到个人体验描述 |

### 2.2 关键词黑名单（显性违规）

| 类别 | 禁用词 | 合法例外 |
|------|--------|---------|
| 指令性语言 | 你应该、你必须、一定要、千万要、得、要 | "这不是必须的"（否定语境） |
| 承诺性语言 | 保证、一定能、必赚、包你、100%、稳了 | "不做收益承诺"（元陈述） |
| 评判性语言 | 正确/错误、好/坏、优秀/糟糕（评价他人时） | "好不好看你自己"（元回护） |
| 对立符号 | ❌、✅、✗、✓（用于对比时） | 践行卡checkbox用□、✓合法 |
| 游戏化元素 | 连续打卡、打卡天数、积分、勋章、等级、坚持、加油、突破、逆袭 | "不需要打卡"（否定语境） |
| 权威姿态 | 听我的、我告诉你、我教你、跟着我 | "我陪你看见""我们一起"合法 |

### 2.3 概念白名单（防隐性污染）

**允许使用的核心表达**：
> 体验、状态、看见、边界、慢慢来、感受、节奏、允许、自然、松、空、闲、飘、沉、飘着、待着、不赶、不着、不急、停、放、散

**禁止混入的门派及词汇**：

| 混入门派 | 禁止混入词 | 混入原因 |
|---------|-----------|---------|
| 儒家/心学 | 修身、齐家、慎独、知行合一、致良知、良知、内圣外王 | 重伦理秩序，与自然状态不同谱系 |
| 佛家/禅宗 | 觉悟、放下、解脱、涅槃、禅、般若、顿悟、空性、看破 | 求解脱 vs 贵自然，思想谱系不同 |
| 现代心理学 | 赋能、疗愈、接纳、成长、认知升级、心流、创伤、内在小孩 | 科学本质化，降格道为心理现象 |
| 成功学/励志 | 坚持、努力、加油、一定要、必赢、永不放弃、突破极限、逆袭 | 与"无为/不争"根本对立 |
| 营销话术 | 痛点、击穿、转化、收割、锁客、裂变、私域、变现闭环 | 与"信为本/柔而化之"对立 |

**混入词处理规则**：
- 描述目标体系概念时混入 → 替换为白名单词
- 描述其他体系（如"儒家讲修身"）→ 可保留但需标注
- 否定/对比语境（如"不是坚持，是自然"）→ 可保留

### 2.4 表达形式约束

- ❌ 禁止使用 ❌vs✅ 二元对立对比
- ✅ 必须使用**正言若反**辩证句式
- ✅ 每条文案读起来应该像一句有味道的话，不像模板
- ✅ 读者读完应该是"被看见"的感觉，不是"被教育"的感觉

## 三、任务要求

### 第一阶段：创作（使用正言若反写作法）

为以下3个场景各写1条朋友圈文案：

| 场景编号 | 场景主题 | 核心对立点 |
|---------|---------|-----------|
| 文案1 | 工作效率对比 | 拼命加班赶工 vs 不赶时间的工作 |
| 文案2 | 社群学习对比 | 疯狂报课囤课 vs 少即是多的慢学 |
| 文案3 | 休息vs工作 | 休息是为了更好工作 vs 休息本身就是目的 |

**每条文案必须包含**：
1. **对立式草稿**（先按❌vs✅方式写出，帮助理清核心张力）
2. **正言若反式终稿**（用"世人皆知…不知…不是不…是…"结构或其变体）
3. **回护检查记录**（回答三个问题：读者会被否定吗？回护真诚吗？节奏自然吗？）

### 第二阶段：审查（使用约束审查三轨制）

写完3条文案后，按三轨制逐一自我审查：

**第一轨：关键词黑名单检查**
- [ ] 逐词搜索黑名单6类关键词
- [ ] 记录每个命中，判断是"违规"还是"合法语境"
- [ ] 修复所有违规使用
- [ ] 记录：命中数__个，修复__个

**第二轨：概念白名单检查**
- [ ] 搜索5个混入门派的关键词
- [ ] 记录每个命中，判断上下文合法性
- [ ] 替换/移除违规混入
- [ ] 记录：混入数__个，修复__个

**第三轨：哲学立场审查**
逐句检查7个立场维度：
| 维度 | 检查问题 | 发现问题数 |
|------|---------|-----------|
| 本质化倾向 | 是否承诺"慢=更高效/更成功"？ | __个 |
| 终点化倾向 | 是否暗示有一个"做到就自由了"的终点？ | __个 |
| 权威化倾向 | 是否自居"我告诉你真相"的位置？ | __个 |
| 普世化倾向 | 是否把个人观察说成"所有人都应该"？ | __个 |
| 科学化倾向 | 是否用科学/心理学来"证明"慢的好处？ | __个 |
| 二元对立残留 | 即使正言若反，是否仍隐含高下评判？ | __个 |
| 答案植入 | 是否隐含了"正确的生活方式"答案？ | __个 |

- [ ] 修复所有立场问题
- [ ] 修复后通读一遍，感受整体语气

### 第三阶段：最终交付

输出最终版的3条文案，附：
1. 三轨审查记录表（命中数、修复数、典型问题举例）
2. 创作过程中的3个关键发现/卡点
3. 一句话总结：正言若反写作法最容易踩的坑是什么？

## 四、验收标准

| 检查项 | 通过标准 |
|--------|---------|
| 形式要求 | 3条文案全部使用正言若反句式，先对立式再辩证式 |
| 第一轨 | 黑名单零违规（所有命中均为合法语境） |
| 第二轨 | 白名单外概念零混入（副词/连接词/日常用语除外） |
| 第三轨 | 7个立场维度零问题 |
| 读者感受 | 读完没有"被教育/被否定/被营销"的感觉 |
| 文化质感 | 不像AI写的营销文案，像一个真实的人在说有味道的话 |

### 文案1思路提示
对立式：❌拼死拼活赶进度 → ✅不赶时间慢慢做
正言若反式方向：
"世人皆知快之为快，不知慢之为大快。
不是教你偷懒——
是帮你看见，哪些赶是真的需要，哪些赶只是在填焦虑的坑。"
注意："大快"不能理解为"更大的快乐/更快的速度"，而是"更快"的辩证——不赶的时候，事情反而成得更顺。但不要直接说"反而更高效"，那又落回效率逻辑了。

### 文案2思路提示
对立式：❌报课囤课焦虑学 → ✅少学慢学学透
正言若反式方向：
"世人皆知多学之为进，不知少学之为大进。
不是不学——
是你囤的那些课，有多少是真的在长，有多少只是在缓解'我在学习'的焦虑？"

### 文案3思路提示
对立式：❌休息为了工作 → ✅休息本身就是目的
正言若反式方向：
"世人皆知休息之为充电，不知不充之为大充。
不是不工作——
是你有没有试过，什么都不为，就只是坐着发会儿呆？"

### 三轨审查常见坑
1. **第一轨最容易漏**："要""得"这类口语化指令词很容易混进去
2. **第二轨最容易混**："成长""接纳"这两个词太常见了，写的时候根本意识不到是心理学混入
3. **第三轨最容易犯**：写着写着就滑向"慢下来你会更高效"——这本质上还是效率框架内的承诺，不是真的反效率

## 六、进阶挑战（选做）

如果上面的练习完成得顺利，可以尝试：

1. **变体练习**：用反问式/回环式/水喻式各写1条文案，对比效果
2. **角色互换**：假设你是一个"高效能"博主（对立面），用正言若反法写"效率"视角的文案——不否定慢，但揭示效率的边界
3. **三轨制实战**：找一篇你之前写的文案（或AI生成的文案），用三轨制审查并记录发现的问题数
