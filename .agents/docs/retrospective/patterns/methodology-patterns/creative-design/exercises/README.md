## 📋 使用建议

1. **先读模式再做练习**：练习前先通读配套模式文档，理解核心概念和操作步骤
2. **不要先看参考答案**：参考提示用HTML details折叠，先独立完成再对照
3. **记录练习过程**：使用文档中提供的练习记录模板，记录每一步的思考
4. **三轨审查不可省略**：即使觉得文案没问题，也要走完三轨审查流程——60%的问题在第三轨才会发现
