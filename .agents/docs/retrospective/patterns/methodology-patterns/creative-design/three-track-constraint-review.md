# 约束审查三轨制

## 核心概念

系统性地检查受严格约束的内容（哲学文档、品牌文案、合规材料、方法论白皮书、AI生成内容质量控制），按三轨顺序执行，同时捕获显性违规和隐性违规：
- 第一轨：关键词黑名单（搜显性违规）
- 第二轨：概念白名单（查隐性污染）
- 第三轨：立场审查（验哲学一致性）

## 解决的问题

仅靠关键词搜索审查内容会漏掉两类问题：

| 违规类型 | 能否被关键词搜索捕获 | 占比（竹简悟道项目统计） |
|---------|---------------------|------------------------|
| **显性违规**：直接使用违禁词、禁用表达 | ✅ 可以 | 40% |
| **隐性违规**：跨域概念污染、本质化/实在论化、立场偏离 | ❌ 无法 | 60% |

在"以道入商·竹简悟道视角升级"项目中，仅靠黑名单搜索会漏掉60%的硬伤。

## 三轨执行顺序

```
第一轨：关键词黑名单（搜显性违规）
  ↓
第二轨：概念白名单（查隐性污染）
  ↓
第三轨：立场审查（验哲学一致性）
```

三轨必须按顺序执行——第一轨快速扫除显性问题，第二轨排查概念混用，第三轨进行深度立场验证。跳过前两轨直接做第三轨效率低；只做前两轨不做第三轨会漏掉本质化/实在论等深层问题。

## 第一轨：关键词黑名单

**目的**：快速捕获显性违规词汇和表达。

**通用黑名单模板**（根据具体项目调整）：

| 类别 | 关键词 | 合法使用例外 |
|------|--------|-------------|
| 指令性语言 | "你应该""你必须""一定要""千万要" | "这不是必须的"（否定语境合法） |
| 承诺性语言 | "保证""一定能""必赚""包你""100%" | "不做收益承诺"（元陈述合法） |
| 评判性语言 | "正确/错误""好/坏""优秀/糟糕"（评价他人他事时） | "好不好看你自己"（元回护合法） |
| 对立符号 | "❌""✅""✗""✓"（用于对比时） | checkbox用□、打钩标记✓（践行卡checkbox合法） |
| 游戏化元素 | "连续X天打卡""打卡天数""积分""勋章""等级""坚持加油" | "不需要连续"（否定语境合法） |
| 权威姿态 | "听我的""我告诉你""我教你" | "我陪你看见""我们一起"（合法） |

**执行检查单**：
- [ ] 逐词搜索黑名单中的每个关键词
- [ ] 对每个命中，判断是"违规使用"还是"合法语境"（否定/元陈述/checkbox等）
- [ ] 对违规使用，直接修复
- [ ] 记录命中数量和修复数量，供复查参考

## 第二轨：概念白名单

**目的**：捕获跨域概念污染——那些不属于目标体系但"听起来很对"的词汇。

**为什么需要白名单**：大语言模型训练数据中各领域概念高度混合。当要求模型用特定哲学/方法论写作时，模型会自然引入相邻领域的高频词汇——这些词在原领域是"好词"，但在目标体系中是外来词，会污染概念纯粹性。

**操作步骤**：

### Step 1：明确目标体系的允许概念清单

以竹简悟道·帛书老子为例：

| 类别 | 允许使用的概念 |
|------|--------------|
| 核心概念 | 道、德、虚静、无为、柔弱、不争、玄同、见独、恒德、微明、自然 |
| 方法论概念 | 概念解缚、体道、体道链、正言若反、Open Questioning、反者道之动 |
| 四法八场景 | 虚静内观、自然无为、柔弱不争、生活实践 |
| 体道链层级 | 名→反→有无→正言若反→无为→玄同 |

### Step 2：列出常见混入门派和混入词

以竹简悟道项目为例（其他项目需根据自身情况定制）：

| 混入门派 | 常见混入词 | 为什么在帛书体系中不适用 |
|---------|-----------|----------------------|
| 儒家/心学 | 知行合一、致良知、修身齐家、仁义礼、慎独 | 儒家重伦理秩序，道家重自然状态；"知行合一"是王阳明概念，非老子 |
| 佛家/禅宗 | 觉悟、慈悲、涅槃、禅、般若、顿悟、空性、解脱 | 佛家求解脱，道家贵自然；思想谱系不同 |
| 现代心理学 | 接纳、赋能、疗愈、成长、认知科学、心流、杏仁核 | 科学本质化倾向，将道降格为心理学/神经科学现象 |
| 成功学/励志 | 坚持、加油、一定要、必赢、永不放弃、突破极限 | 与"无为/不争"根本对立 |
| 营销话术 | 痛点、击穿、转化、收割、锁客、裂变 | 与"信为本/柔而化之"对立 |

### Step 3：搜索常见混入词并处理

- 如果混入词用于描述**目标体系的概念**（如用"知行合一"命名帛书第41章的践行）→替换为目标体系内的对应概念（如"勤而行之"）
- 如果混入词用于**描述其他体系**（如"儒家讲修身齐家"）→可以保留，但需标注是他体系概念
- 如果混入词用于**否定/对比**（如"不是坚持，是无为"）→可以保留，但确保核心立场在目标体系一侧

**执行检查单**：
- [ ] 已明确目标体系的允许概念清单
- [ ] 已列出至少3个常见混入门派及其混入词
- [ ] 逐个搜索混入词，判断上下文合法性
- [ ] 违规混入已替换或移除
- [ ] 没有"这个词挺好的就留着吧"的侥幸心理

## 第三轨：立场审查

**目的**：捕获哲学立场不一致的隐性问题——这些问题不体现为特定词汇，而是体现为"怎么说"和"说什么"的立场。

**为什么需要立场审查**：即使所有词汇都在白名单内，仍可能出现立场偏离。例如"道是宇宙的客观规律"这句话里的所有词都是白名单词，但它将道本质化为"客观规律"，违反了反本质主义立场。

**通用立场审查问题清单**（以反本质主义/体验导向的哲学体系为例）：

| 审查维度 | 审查问题 | 违规表现 | 修复方向 |
|---------|---------|---------|---------|
| **本质化倾向** | 核心概念是否被表述为"客观存在/宇宙规律/科学真理"？ | "道是宇宙规律""这是客观真理" | 改为"一种体验""一种活法""某种状态""在XX情境下呈现为…" |
| **终点化倾向** | 是否暗示有一个"终点/开悟/得道"的完成状态？ | "悟道之后你就…""达到玄同境界就…" | 改为持续实践过程："道是行出来的""修行是一辈子的事" |
| **权威化倾向** | 作者/体系是否自居权威位置？ | "我告诉你真相""听我的没错" | 改为"我陪你看见""这是我观察到的" |
| **普世化倾向** | 是否把特定情境下的观察说成普遍适用的真理？ | "所有人都应该…""这是唯一正确的路" | 加边界条件："在XX场景下""对我而言""也许你也可以试试" |
| **科学化倾向** | 是否用科学/学术话语来"证明"哲学概念？ | "神经科学证明…""心理学研究表明…"（用于证道） | 删除科学证明，回到体验描述；科学数据仅作为辅助参考 |
| **二元对立残留** | 即使使用了正言若反，是否仍隐含高下评判？ | "A是低级的，B是高级的" | 改为"A和B是不同的状态/阶段/面向"，不做高下评判 |
| **答案植入** | 在开放性问题/反思工具中，是否隐含了"正确答案"？ | 自问后附"如果你回答X，说明你Y" | 纯开放式问题不附答案/解读/层级判断 |

**执行检查单**：
- [ ] 通读全文，逐段检查上述7个维度
- [ ] 标记所有立场存疑的句子
- [ ] 逐句修复：本质化→体验描述，终点化→持续过程，权威化→平等映照，普世化→加边界，科学化→回体验，二元化→去评判，答案植入→留白
- [ ] 修复后再通读一遍，感受整体语气是否一致

## 完整执行流程

```
┌─────────────────────────────────────────────┐
│  约束审查三轨制 · 执行流程                    │
├─────────────────────────────────────────────┤
│                                              │
│  1. 准备阶段                                 │
│     □ 确定目标体系的核心立场                 │
│     □ 准备黑名单（关键词）                   │
│     □ 准备白名单（允许概念+混入词）          │
│     □ 准备立场审查问题清单                   │
│                                              │
│  2. 第一轨 · 黑名单搜索（5-10分钟）          │
│     □ 逐词搜索，标记命中                     │
│     □ 判断合法/违规，修复违规                │
│                                              │
│  3. 第二轨 · 白名单检查（10-20分钟）         │
│     □ 逐词搜索混入词，标记命中               │
│     □ 判断上下文合法性，替换违规混入         │
│                                              │
│  4. 第三轨 · 立场审查（20-40分钟）           │
│     □ 通读全文，用立场问题清单逐段检查       │
│     □ 修复本质化/终点化/权威化/普世化等问题  │
│                                              │
│  5. 收尾检查                                │
│     □ 目录/编号/索引一致性                  │
│     □ 格式规范（标题层级/checkbox/引用格式） │
│     □ 最终通读一遍，感受整体语气             │
│                                              │
└─────────────────────────────────────────────┘
```

## 实战案例：竹简悟道项目Task9审查结果

在"以道入商·v2.0"项目的Task9审查中，三轨制捕获的10处问题分布：

| 轨 | 捕获的问题数 | 具体问题 |
|----|------------|---------|
| 第一轨（黑名单） | 4处 | ❌✅符号1处、"必须出声"等指令性语言2处、frontmatter字段缺失1处 |
| 第二轨（白名单） | 3处 | "知行合一"（王阳明心学混入）1处、"接纳"（心理学混入）1处、帛书索引Day编号与正文矛盾1处 |
| 第三轨（立场审查） | 3处 | "客观规律"将道本质化1处、"道是可验证的认知科学"（科学化+本质化）1处、"必须有变现闭环"断言式1处 |

**关键发现**：第二轨和第三轨捕获的隐性问题虽然数量不多，但都是伤筋动骨的硬伤——概念混用和哲学立场偏离会让整篇文档的哲学根基不成立。

## 适用边界

三轨制不是所有写作都需要：

| 场景类型 | 是否需要三轨制 | 原因 |
|---------|--------------|------|
| 哲学/方法论白皮书 | ✅ 强烈推荐 | 概念纯粹性是核心质量标准 |
| 品牌文案/品牌手册 | ✅ 推荐 | 品牌调性一致性需要立场审查 |
| 合规/法律文档 | ✅ 部分适用（第一轨+第二轨） | 关键词和术语准确性是刚需，哲学立场非重点 |
| 学术论文 | ✅ 推荐（调整立场审查维度） | 术语规范和学术立场一致性 |
| 日常博客/随笔 | ❌ 不需要 | 过度审查会扼杀自然表达 |
| 营销文案（转化导向） | ⚠️ 选择性使用 | 如需要品牌调性一致则用，纯效果广告不需要 |

## 常见陷阱

| 陷阱 | 表现 | 预防方式 |
|------|------|---------|
| **白名单变成思想警察** | 过度清洗，导致文本读起来像审查稿 | 白名单用于检查"核心概念是否被替换"，不是禁用所有非体系词汇；副词、连接词、日常用语不需要审查 |
| **立场审查变成风格统一** | 把所有句子改成同一个腔调，丢失作者声音 | 立场审查只查立场（本质化/权威化等），不查个人风格；多样性表达是好事 |
| **过度依赖黑名单** | 花大量时间扩充黑名单词库，忽视白名单和立场审查 | 黑名单是最低门槛，不是充分条件；投入时间应按第一轨<第二轨<第三轨分配 |
| **审查一次就完事** | 修改一处后引入新的违规，没有二次检查 | 每次修改后至少重新跑一次第一轨（快速搜索），确认修改没有引入新问题 |
| **为方法论而方法论** | 对1000字的短文也走完整三轨，效率极低 | 根据文档规模和重要性调整投入：短文走第一轨+抽查立场即可；重要长文档才走完整三轨 |
