# 毕业论文写作指南 Wiki

> 从选题到答辩，手把手带你从零完成一篇合格的毕业论文

## 简介

本 Wiki 系统梳理本科/硕士毕业论文写作的**完整方法论**，涵盖从选题、开题、文献综述、正文写作到修改答辩的全流程。不是给你一个模板让你填空，而是教你**围绕研究问题构建逻辑闭环**的科学写作方法。

### 核心观点

> 论文写作的本质是**问题驱动**的逻辑建构——先有一个值得研究的问题，然后用合适的方法去回答它，最后把这个过程清晰地呈现出来。
>
> 模板是结果，不是原因。好论文的框架是从研究问题中自然生长出来的，而不是从百度文库里下载来的。

## 快速回顾

### 论文写作核心公式

```
好论文 = 好选题 × 清晰逻辑 × 充分论证 × 规范格式
```

### 五步写作法

1. **定问题**：找到一个具体、可行、有价值的研究问题
2. **搭框架**：围绕问题搭建"提出→分析→解决"的逻辑链
3. **填内容**：用文献、数据、案例支撑每个论点
4. **做闭环**：确保问题与对策一一对应，结论回应引言
5. **磨细节**：格式、语言、查重、引用全面检查

### 本科 vs 硕士论文差异

| 维度 | 本科毕业论文 | 硕士毕业论文 |
|------|-------------|-------------|
| 字数 | 8000-15000字 | 20000-50000字 |
| 创新要求 | 微创新（视角/案例） | 中度创新（方法/模型/实证） |
| 文献综述 | 分类梳理即可 | 深度评述，明确研究空白 |
| 研究方法 | 2-3种基础方法 | 定性+定量结合，需实证 |
| 查重标准 | ≤20%（多数≤15%） | ≤15%（双一流≤10%） |
| 审核流程 | 导师审核+校内盲审+答辩 | 导师审核+校外双向盲审+答辩 |
