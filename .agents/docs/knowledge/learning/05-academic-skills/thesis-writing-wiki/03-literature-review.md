# 四、文献综述

> 文献综述不是文献堆砌，而是站在巨人的肩膀上看清楚自己要站在哪里。

## 4.1 什么是文献综述？

文献综述是在全面、系统地阅读相关研究文献的基础上，对该领域的研究现状、主要成果、争论焦点、存在问题及发展趋势等进行**归纳、整理、评述**的学术写作。

简单说：
- ❌ 不是："XX说了A，YY说了B，ZZ说了C……"（文献堆砌）
- ✅ 而是："关于这个问题，学术界已经研究了X、Y、Z三个方面，取得了A、B、C成果，但还存在D、E两个不足，我的研究就是为了弥补这个缺口。"

### 文献综述的三个作用

1. **证明你了解这个领域**：告诉评审老师，你不是在闭门造车，你知道前人做了什么
2. **找到你的研究定位**：通过梳理现有研究，找到缺口，明确你的研究价值
3. **为后续研究打基础**：理论基础、研究方法、分析框架都可以从文献中来

## 4.2 文献检索：在哪里找？怎么找？

### 常用文献数据库

| 类型 | 中文数据库 | 英文数据库 |
|------|-----------|-----------|
| 综合 | 中国知网（CNKI）、万方、维普 | Web of Science、Scopus、Google Scholar |
| 期刊 | 中国知网期刊库 | Elsevier、Springer、Wiley、Taylor & Francis |
| 学位论文 | 知网硕博论文库、万方学位论文 | ProQuest、WorldCat |
| 会议 | 知网会议论文 | IEEE Xplore、ACM Digital Library |

### 检索方法

#### 方法1：关键词检索

1. 确定你的核心关键词
2. 尝试不同的组合：关键词+关键词、关键词+同义词
3. 用高级检索限定时间、期刊、作者等

#### 方法2：引文追踪（滚雪球法）

找到一篇核心论文后：
- **往前追**：看这篇论文的参考文献，找到更早的经典文献
- **往后追**：看引用了这篇论文的文献，找到更新的研究

#### 方法3：综述论文法

找一篇高质量的综述论文（一般标题就叫"XX研究综述"），它已经帮你梳理了这个领域的研究脉络，你可以：
- 顺着它的参考文献找原文
- 在它的基础上补充最新的研究

### 选文献的标准

不是所有文献都要读，要学会筛选：

1. **期刊等级**：优先看核心期刊、CSSCI、SCI/SSCI论文
2. **引用量**：引用量高的论文通常更重要
3. **发表时间**：优先看近3-5年的最新研究（经典文献除外）
4. **作者团队**：领域内知名学者的论文更可靠
5. **相关性**：和你的研究主题直接相关吗？

### 需要读多少文献？

| 阶段 | 建议数量 |
|------|---------|
| 本科论文 | 20-30篇（其中核心期刊5-10篇） |
| 硕士论文 | 50-100篇（其中英文文献10-20篇） |

> 不是越多越好，关键是**精读核心文献，泛读相关文献**。

## 4.3 文献阅读：怎么读？

### 三遍阅读法

#### 第一遍：速读（5分钟/篇）
- 看什么：标题、摘要、关键词、目录/小标题、结论
- 目的：判断这篇文献值不值得读，和我的研究相关吗？
- 结果：筛选掉不相关的，标记重点文献

#### 第二遍：通读（30分钟/篇）
- 看什么：全文通读，重点看引言、研究方法、主要发现
- 目的：理解这篇文献的核心内容和主要贡献
- 结果：做简单笔记，标记重要观点和数据

#### 第三遍：精读（1-2小时/篇）
- 看什么：逐字逐句读，重点关注研究设计、论证逻辑、不足之处
- 目的：深入理解，批判性思考，找到可以借鉴和质疑的地方
- 结果：做详细笔记，写下你的思考和评论

### 文献笔记怎么做？

每篇重要文献都应该有这样一张笔记卡：

```
【基本信息】
标题：
作者：
年份：
来源（期刊/学位论文）：
关键词：

【核心内容】
研究问题：
研究方法：
主要结论：
核心观点：

【我的评论】
优点：
不足：
可借鉴之处：
和我的研究的关系：
```

## 4.4 文献综述的写作方法

### 综述的结构

```
引言 → 主体（分类评述） → 总结与展望
```

#### 引言（10%）

- 简要说明研究主题的背景和重要性
- 说明综述的目的、范围和方法
- 给读者一个预期：这篇综述要讲什么？

#### 主体（80%）

这是综述的核心，**一定要分类写**，不能按时间顺序或作者顺序罗列。

**常见的分类方式**：
1. **按研究主题分**：不同的研究方向/研究问题
2. **按理论视角分**：不同的理论流派/分析框架
3. **按研究方法分**：定性研究/定量研究/混合研究
4. **按时间阶段分**：早期研究/近期研究/最新进展
5. **按观点争鸣分**：支持派/反对派/中间派

**每一类的写法**：
- 先总述：这一类研究主要关注什么？
- 再分述：代表性的研究有哪些？主要观点是什么？
- 后评述：这类研究有什么贡献？有什么不足？

#### 总结与展望（10%）

- 总结现有研究的主要成果和共识
- 指出现有研究的不足和争议
- 提出未来的研究方向（也就是你的研究可以切入的地方！）

### 写作步骤

1. **列大纲**：先确定分类框架，每一类下面有哪些代表性文献
2. **填内容**：把你的文献笔记按分类填进去
3. **加评述**：每一部分都要有你的评述和总结
4. **找缺口**：最后总结现有研究的不足，引出你的研究价值
5. **顺逻辑**：调整段落之间的衔接，让整个综述连贯流畅

## 4.5 文献综述的常见问题

### ❌ 问题1：堆砌文献，没有评述

**表现**："张三（2020）研究了……；李四（2021）认为……；王五（2022）提出……"

**改进**：
- 每段开头先用一句话总结这一段的主题
- 中间引用不同学者的观点作为支撑
- 结尾用一段话做评述和总结

**对比**：

```
❌ 堆砌版：
  张三（2020）研究了短视频对青少年的影响，发现……
  李四（2021）认为短视频使用与注意力下降有关……
  王五（2022）提出不同看法……

✅ 评述版：
  关于短视频对青少年注意力的影响，学术界存在两种不同观点。
  一部分学者认为短视频使用会导致注意力下降（张三，2020；李四，2021），其依据是……
  另一部分学者则持不同看法，认为……（王五，2022）。
  现有研究的争议主要集中在……，未来需要从……角度进一步探讨。
```

### ❌ 问题2：分类混乱，逻辑不清

**表现**：想到哪写到哪，一会儿说方法，一会儿说结论，不知道按什么逻辑组织的。

**改进**：
- 写作前先列好分类大纲
- 每一节有明确的主题
- 同一层次的分类标准要统一

### ❌ 问题3：只说好话，没有批评

**表现**：把别人的研究吹得天花乱坠，找不到任何不足。

**改进**：
- 任何研究都有局限性
- 可以从研究方法、样本选择、理论视角、适用范围等方面找不足
- 找到不足，你的研究才有价值

### ❌ 问题4：文献太旧，没有最新研究

**表现**：引用的都是十年前的文献，没有近3年的研究。

**改进**：
- 经典文献可以用，但一定要有最新研究
- 确保你了解这个领域的最新进展
- 近3年的文献至少占三分之一

### ❌ 问题5：引用不规范

**表现**：格式乱、缺信息、找不到原文。

**改进**：
- 用文献管理软件自动生成
- 严格按照学校要求的格式（GB/T 7714、APA、MLA等）
- 每条引用都要能追溯到原文

## 4.6 文献综述的检查清单

写完文献综述后，对照这个清单检查：

- [ ] 有没有清晰的分类框架？（不是简单罗列）
- [ ] 每一部分都有评述吗？（不是只说别人研究了什么）
- [ ] 最后有没有总结研究不足和未来方向？（这是你的研究价值所在）
- [ ] 文献数量够吗？（本科20+，硕士50+）
- [ ] 最新文献够吗？（近3年至少占1/3）
- [ ] 核心文献都引用了吗？（领域内经典论文不能漏）
- [ ] 引用格式统一吗？（用文献管理软件检查）
- [ ] 逻辑连贯吗？（段落之间有衔接）
- [ ] 有没有语法错误和错别字？

> **一句话总结**：文献综述的核心不是"综"（罗列），而是"述"（评述）。你的观点和判断，才是综述的灵魂。
