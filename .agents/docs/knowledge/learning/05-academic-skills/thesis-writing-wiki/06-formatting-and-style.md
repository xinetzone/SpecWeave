# 七、格式规范与排版

> 格式是论文的脸面。内容再好，格式一塌糊涂，印象分就没了。

## 7.1 为什么格式这么重要？

很多同学不重视格式，觉得"内容好就行了"。实际上，格式非常重要，原因有三：

1. **态度问题**：格式混乱=态度不认真。老师看到格式乱七八糟，第一印象就差了。
2. **规范要求**：毕业论文有严格的格式标准，不符合要求可能被打回。
3. **阅读体验**：规范的格式让读者读起来舒服，能更好地理解你的内容。

> 记住：内容决定你的上限，格式决定你的下限。格式过不了关，内容再好也可能被毙。

## 7.2 基本格式要求

### 页面设置

| 项目 | 要求（通用） | 以你们学校模板为准 |
|------|-------------|-------------------|
| 纸张大小 | A4 | ✅ |
| 上边距 | 2.5cm | 以学校为准 |
| 下边距 | 2.5cm | 以学校为准 |
| 左边距 | 3cm（装订侧） | 以学校为准 |
| 右边距 | 2.5cm | 以学校为准 |
| 页眉 | 1.5cm | 以学校为准 |
| 页脚 | 1.75cm | 以学校为准 |

### 字体字号

| 内容 | 字体 | 字号 | 对齐方式 |
|------|------|------|---------|
| 论文题目 | 黑体/宋体 | 二号/小二号 | 居中 |
| 一级标题（章） | 黑体 | 三号/小三号 | 左对齐/居中 |
| 二级标题（节） | 黑体 | 四号 | 左对齐 |
| 三级标题（小节） | 黑体/楷体 | 小四号 | 左对齐 |
| 正文 | 宋体 | 小四号 | 两端对齐 |
| 摘要/关键词 | 宋体 | 小四号 | 两端对齐 |
| 参考文献 | 宋体 | 五号/小四号 | 左对齐 |
| 图表标题 | 黑体/宋体 | 五号 | 居中 |
| 页码 | 宋体 | 五号 | 居中 |

> **重要**：以上是通用参考，**一切以你们学校的模板为准**！

### 段落格式

- **行间距**：1.5倍行距或固定值20-22磅
- **段前距**：一级标题段前1-2行，二级标题段前0.5-1行
- **段后距**：一级标题段后1行
- **首行缩进**：正文首行缩进2字符
- **字间距**：标准

## 7.3 引用规范

### 为什么引用很重要？

1. **学术规范**：不引用就是抄袭
2. **尊重他人**：给原创作者应有的 credit
3. **体现基础**：展示你了解这个领域的研究
4. **方便读者**：读者可以顺着你的引用找到原文

### 什么时候需要引用？

- ✅ 直接引用别人的原话（要加引号）
- ✅ 转述别人的观点（用自己的话说）
- ✅ 使用别人的数据、图表、方法
- ✅ 提到某个理论、概念、模型
- ❌ 常识性的内容不需要引用（如"地球是圆的"）

### 常见引用格式

| 格式标准 | 适用领域 | 文中引用示例 |
|---------|---------|-------------|
| GB/T 7714（国标） | 中国高校通用 | 张三（2020）认为……[1] |
| APA | 心理学、社会科学 | （Zhang, 2020） |
| MLA | 文学、人文学科 | （Zhang 123） |
| Chicago | 历史、艺术史 | 脚注/尾注 |

**国内高校一般用GB/T 7714**，具体以学校要求为准。

### GB/T 7714 参考文献格式

#### 期刊论文
```
[序号] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
```
示例：
```
[1] 张三, 李四. 短视频对青少年注意力的影响研究[J]. 心理学报, 2022, 54(3): 321-330.
```

#### 图书
```
[序号] 作者. 书名[M]. 出版地: 出版社, 出版年: 起止页码.
```
示例：
```
[2] 王五. 社会心理学[M]. 北京: 北京大学出版社, 2020: 45-67.
```

#### 学位论文
```
[序号] 作者. 题名[D]. 学校所在地: 学校名称, 年.
```
示例：
```
[3] 赵六. 人工智能在教育中的应用研究[D]. 上海: 复旦大学, 2021.
```

#### 报纸文章
```
[序号] 作者. 题名[N]. 报纸名, 出版日期(版次).
```

#### 电子文献
```
[序号] 作者. 题名[EB/OL]. (发表日期)[引用日期]. 网址.
```

### 引用的常见错误

❌ 引用格式不统一（有的有卷号有的没有）
❌ 作者名字写错（中英文混排、顺序错了）
❌ 年份、卷期、页码不对
❌ 引用了没读过的文献（万一老师问起来就尴尬了）
❌ 只在文中标了，参考文献里没有
❌ 参考文献里有，文中没引用

### 工具推荐

- **文献管理软件**：Zotero、EndNote、NoteExpress（自动生成参考文献，一键切换格式）
- **格式检查**：知网研学、NoteExpress

## 7.4 图表规范

### 图的规范

1. **图号和图题**：图1 图题（在图的下方，居中）
2. **图的质量**：清晰、可读、不失真
3. **图的来源**：如果引用别人的图，要注明来源
4. **图在文中的位置**：先见文，后见图，图随文走

### 表的规范

1. **表号和表题**：表1 表题（在表的上方，居中）
2. **表格格式**：三线表（顶线、表头线、底线）
3. **表的来源**：引用的要注明来源
4. **表在文中的位置**：先见文，后见表，表随文走

### 三线表示例

表1 受访者基本信息统计

| 变量 | 类别 | 频数 | 百分比（%） |
|------|------|------|------------|
| 性别 | 男 | 120 | 48.0 |
|      | 女 | 130 | 52.0 |
| 年龄 | 18-25岁 | 80 | 32.0 |
|      | 26-35岁 | 100 | 40.0 |
|      | 36岁以上 | 70 | 28.0 |

### 图表常见问题

❌ 图太模糊，看不清楚
❌ 表格不是三线表（线条太多）
❌ 图表没有编号，或者编号不连续
❌ 图题表题写得太简单（只有"图1"，没有说明是什么）
❌ 图表和正文脱节（文中没有提到这张图/表）

## 7.5 目录与页码

### 目录

- 自动生成（不要手动打！）
- 至少包含到二级标题，最好到三级
- 目录的页码要和正文对应
- 格式：章标题加粗，节标题不加粗

### 页码

- 前置部分（摘要、目录等）：罗马数字（I, II, III...）
- 正文部分：阿拉伯数字（1, 2, 3...）
- 位置：底部居中或外侧
- 注意：封面、声明页通常不编页码

## 7.6 Word排版技巧

### 技巧1：使用样式

不要手动改格式！用Word的"样式"功能：
- 标题1、标题2、标题3……分别设置好
- 正文样式统一设置
- 好处：改一处，全文档自动更新

### 技巧2：自动生成目录

用了样式之后，目录可以自动生成：
- 引用 → 目录 → 自动目录
- 更新目录：右键 → 更新域 → 更新整个目录

### 技巧3：分节符

不同部分需要不同的页码/格式时，用分节符：
- 布局 → 分隔符 → 下一页分节符
- 每一节可以有独立的页码、页眉页脚

### 技巧4：格式刷

- 选中文本 → 点击格式刷 → 刷目标文本
- 双击格式刷可以连续用，按ESC取消

### 技巧5：查找替换

- 批量修改格式
- 批量删除空行
- 批量替换内容

## 7.7 格式检查清单

提交前，对照这个清单逐项检查：

### 整体格式
- [ ] 页面设置正确（边距、纸张大小）
- [ ] 页码正确（前置罗马数字，正文阿拉伯数字）
- [ ] 目录正确（自动生成，页码对应）
- [ ] 页眉页脚正确

### 标题格式
- [ ] 各级标题样式统一
- [ ] 标题编号正确（1, 1.1, 1.1.1……）
- [ ] 标题格式符合学校要求

### 正文格式
- [ ] 字体字号统一
- [ ] 行间距正确
- [ ] 首行缩进正确
- [ ] 段落对齐正确
- [ ] 没有奇怪的空格和换行

### 引用与参考文献
- [ ] 文中引用格式统一
- [ ] 参考文献格式统一（GB/T 7714或学校要求）
- [ ] 每条引用都能在参考文献中找到
- [ ] 参考文献在文中都有引用
- [ ] 作者、年份、卷期、页码都正确

### 图表
- [ ] 图号图题正确（在图下方）
- [ ] 表号表题正确（在表上方）
- [ ] 表格是三线表
- [ ] 图表清晰可读
- [ ] 图表在文中都有提到
- [ ] 图表编号连续

### 其他
- [ ] 没有错别字
- [ ] 标点符号正确（全角/半角）
- [ ] 英文单词拼写正确
- [ ] 公式格式正确

> **最后一招**：找同学互相检查。自己写的东西自己看不出来问题，别人一眼就能看到。
