# 勾股定理 Wiki

> 几何学的基石，数形结合的纽带，人类智慧的共同结晶

## 简介

勾股定理（Pythagorean theorem）是欧几里得平面几何中最基本、最重要的定理之一。它描述了直角三角形三条边之间的数量关系：**两条直角边的平方和等于斜边的平方**。

\[
a^2 + b^2 = c^2
\]

本 Wiki 系统梳理勾股定理的历史源流、证明方法、推广扩展和应用领域，展现这一古老定理的永恒魅力。

## 快速回顾

### 核心结论

- **定理内容**：直角三角形两直角边的平方和等于斜边的平方
- **几何意义**：两直角边上正方形面积之和等于斜边上正方形面积
- **证明数量**：已知400余种，是数学定理中证明方法最多的
- **发现文明**：古巴比伦、古埃及、中国、古希腊、古印度等多文明独立发现
- **逆定理成立**：三边满足 \(a^2 + b^2 = c^2\) 的三角形必为直角三角形

### 关键公式

| 名称 | 公式 |
|------|------|
| 勾股定理 | \(a^2 + b^2 = c^2\) |
| 欧几里得生成公式 | \(a = m^2 - n^2,\; b = 2mn,\; c = m^2 + n^2\) |
| 余弦定理（推广） | \(c^2 = a^2 + b^2 - 2ab\cos C\) |
| n维欧氏距离 | \(d = \sqrt{\sum_{i=1}^{n} (x_i - y_i)^2}\) |
