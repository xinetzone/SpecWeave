# 四、勾股数

## 4.1 什么是勾股数？

**勾股数**（Pythagorean triple），又称毕氏三元数，是指满足勾股定理 \(a^2 + b^2 = c^2\) 的三个**正整数** \((a, b, c)\)。

其中 \(a\)、\(b\) 称为"勾"和"股"，\(c\) 称为"弦"。

### 基本例子

| 勾股数 | 验证 |
|--------|------|
| (3, 4, 5) | \(3^2 + 4^2 = 9 + 16 = 25 = 5^2\) |
| (5, 12, 13) | \(5^2 + 12^2 = 25 + 144 = 169 = 13^2\) |
| (6, 8, 10) | \(6^2 + 8^2 = 36 + 64 = 100 = 10^2\) |
| (7, 24, 25) | \(7^2 + 24^2 = 49 + 576 = 625 = 25^2\) |
| (8, 15, 17) | \(8^2 + 15^2 = 64 + 225 = 289 = 17^2\) |

## 4.2 本原勾股数与非本原勾股数

### 本原勾股数（Primitive Pythagorean Triple）

如果三个数 \(a, b, c\) 的最大公约数为1（即 \(\gcd(a, b, c) = 1\)），则称为本原勾股数。

**特点**：
- 三个数两两互质（任意两个的最大公约数都是1）
- 必有一个偶数，两个奇数
- 斜边 \(c\) 必为奇数
- 必有一条直角边是3的倍数，一条是4的倍数，斜边是5的倍数

**例子**：(3,4,5)、(5,12,13)、(7,24,25)、(8,15,17)、(9,40,41)

### 非本原勾股数

如果三个数有大于1的公约数，则称为非本原勾股数。

**生成方法**：将本原勾股数乘以任意正整数 \(k\)，得到的 \((ka, kb, kc)\) 仍然是勾股数。

**例子**：
- (3,4,5) × 2 = (6, 8, 10)
- (3,4,5) × 3 = (9, 12, 15)
- (5,12,13) × 2 = (10, 24, 26)

## 4.3 勾股数生成公式

### 欧几里得公式（最经典）

对于任意正整数 \(m > n > 0\)，令：

\[
\begin{cases}
a = m^2 - n^2 \\
b = 2mn \\
c = m^2 + n^2
\end{cases}
\]

则 \((a, b, c)\) 是一组勾股数。

**验证**：
\[
a^2 + b^2 = (m^2 - n^2)^2 + (2mn)^2 = m^4 - 2m^2n^2 + n^4 + 4m^2n^2 = m^4 + 2m^2n^2 + n^4 = (m^2 + n^2)^2 = c^2
\]

### 本原勾股数的条件

当 \(m\) 和 \(n\) 满足以下条件时，生成的是本原勾股数：
1. \(m > n > 0\)
2. \(m\) 和 \(n\) 互质（\(\gcd(m, n) = 1\)）
3. \(m\) 和 \(n\) 一奇一偶

### 举例

| m | n | a = m²-n² | b = 2mn | c = m²+n² | 勾股数 | 是否本原 |
|---|---|-----------|---------|-----------|--------|---------|
| 2 | 1 | 4-1=3 | 4 | 4+1=5 | (3,4,5) | 是 |
| 3 | 2 | 9-4=5 | 12 | 9+4=13 | (5,12,13) | 是 |
| 4 | 1 | 16-1=15 | 8 | 16+1=17 | (8,15,17) | 是 |
| 4 | 3 | 16-9=7 | 24 | 16+9=25 | (7,24,25) | 是 |
| 3 | 1 | 9-1=8 | 6 | 9+1=10 | (6,8,10) | 否（×2） |

## 4.4 勾股数的性质

### 性质1：存在无穷多组勾股数

通过欧几里得公式，取不同的 \(m, n\) 可以生成无穷多组勾股数。

### 性质2：本原勾股数中，有且仅有一个偶数

在本原勾股数中，\(a, b\) 一奇一偶，\(c\) 必为奇数。

### 性质3：本原勾股数中，有且仅有一个3的倍数

### 性质4：本原勾股数中，有且仅有一个4的倍数

### 性质5：本原勾股数中，有且仅有一个5的倍数

> **综合性质3-5**：在任何一组本原勾股数中，乘积 \(abc\) 必能被 \(3 \times 4 \times 5 = 60\) 整除。

## 4.5 历史上的勾股数

### 古巴比伦：普林顿322号泥板

- 约公元前1800年的楔形文字泥板
- 记载了15组勾股数，包括复杂的组合
- 表明巴比伦人可能已掌握系统的勾股数生成方法

### 古代中国

- 《周髀算经》："勾三股四弦五"——(3, 4, 5)
- 《九章算术》：记载了多组勾股数及其应用

### 古希腊

- 毕达哥拉斯学派：系统研究勾股数
- 柏拉图：提出另一组勾股数生成公式
- 欧几里得：《几何原本》中给出完整的生成公式

## 4.6 勾股数的延伸思考

勾股数看似简单，却引出了深刻的数学问题：

### 费马大定理

勾股定理是关于二次方程 \(a^2 + b^2 = c^2\) 的整数解。法国数学家费马在1637年提出：对于大于2的整数 \(n\)，方程 \(a^n + b^n = c^n\) 没有正整数解。这就是著名的**费马大定理**，历经358年才被英国数学家安德鲁·怀尔斯于1994年证明。

### 与数论的联系

勾股数是数论的重要研究对象，与：
- 二次剩余
- 素数分解
- 椭圆曲线
- 模形式

等现代数论分支有着深刻联系。

> 从一组简单的整数三元组出发，数学的探索之路通向了人类智慧的巅峰。这就是数学的魅力——从简单的问题开始，走向深邃的真理。
