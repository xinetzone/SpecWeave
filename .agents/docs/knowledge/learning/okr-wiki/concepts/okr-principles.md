# OKR的核心原则

> **来源**: [最全OKR制定指南（2种思路+7类方法）](https://www.dingteam.com/help/584)

## 五大核心原则

### 1. 战略对齐

自上而下承接，确保个人、团队、部门与公司目标一致，力出一孔。

**关键要点**:
- 公司OKR → 部门OKR → 团队OKR → 个人OKR
- 每个层级的目标必须支撑上一层级的目标
- 避免目标脱节，形成"部门墙"

### 2. 挑战导向

理想完成率60%-70%，鼓励突破舒适区。

**关键要点**:
- 目标要有挑战性，不要设定过于保守的目标
- 60%-70%的完成率是理想状态，表示团队在挑战自我
- 即使未完全达成，也能推动团队进步

### 3. 公开透明

全员可见，促进跨部门协同，打破信息壁垒。

**关键要点**:
- 所有OKR对全员公开（除非涉及敏感信息）
- 员工可以看到公司、部门、同事的OKR
- 促进跨部门协作和信息共享

### 4. 敏捷迭代

通常按季度周期推进，定期复盘调整，适配变化。

**关键要点**:
- 按季度设定和追踪OKR
- 定期（周/月）复盘，及时调整
- 保持灵活性，适应市场变化

### 5. 与绩效脱钩

评分不直接关联薪酬，减少考核压力，鼓励设定野心目标。

**关键要点**:
- OKR是成长工具，不是考核武器
- 避免将OKR分数直接与奖金挂钩
- 让员工敢于设定有挑战性的目标

## 原则对比

| 原则 | 传统绩效考核 | OKR |
|------|--------------|-----|
| **目标设定** | 上级下达，侧重达标 | 团队自主设定，上下对齐 |
| **目标难度** | 可达成性（100%达成为合格） | 挑战性（60%-70%达成为优） |
| **评分与薪酬** | 强相关 | 不直接挂钩 |
| **透明度** | 有限公开 | 全员公开 |
| **调整频率** | 年度固定 | 季度调整 |
