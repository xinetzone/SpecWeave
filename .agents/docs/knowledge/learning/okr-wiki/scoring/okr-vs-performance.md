# OKR与绩效的关系

> **来源**: [怎么把绩效和OKR的评分结合起来？](https://www.dingteam.com/help/649)

## OKR与绩效是两回事

OKR和绩效是两个不同的体系，不应混淆。

### 案例说明

**案例1**:
- OKR完成度很低，导致OKR评分低
- 但是OKR之外的一些工作取得了超出预期的结果
- **结论**: 依然可以拿到好绩效

**案例2**:
- OKR结果大多不符合预期，有些比预期低很多，少数远远超出预期
- OKR评分不会太高
- **结论**: 但最终绩效可以很好

**案例3**:
- OKR本身难度太低，最终完成度很好，大都符合预期
- OKR评分可以很高
- **结论**: 但最终绩效不会好

---

## 如何将OKR与绩效结合？

### 结合方式

**建议**: 把OKR的评分作为绩效考核的一个指标参考项

**操作**:
1. 公司内部制定好绩效的指标，以及相应的评定人
2. 根据制定好的绩效指标，员工来确定自己的OKR内容
3. 根据OKR的执行情况，进行评分和等级评定
4. 将OKR评分作为绩效考核的参考项之一

---

## 注意事项

### OKR不等于绩效考核

**重要提醒**: OKR不等于绩效考核

**风险**: 如果有他人评分，可能会让员工有考核的概念，而导致员工不愿意去设置有挑战性的目标

### 谨慎选择

**建议**: 根据团队自身情况谨慎选择是否启用

### 员工教育

**建议**: 如果确实需要将OKR与绩效结合，建议做好员工侧的教育宣导

**目的**:
- 让员工理解OKR的真正价值
- 鼓励设定有挑战性的目标
- 避免将OKR视为考核工具

---

## 最佳实践

### 推荐模式

**模式**: OKR与绩效解耦，但OKR评分作为绩效参考

**好处**:
- 保持OKR的挑战性
- 避免员工保守设定目标
- 同时为绩效评估提供参考

### 绩效评估维度

| 维度 | 说明 |
|------|------|
| **OKR完成度** | 作为参考项，体现目标达成能力 |
| **工作成果** | 实际产出和贡献 |
| **团队协作** | 团队合作和沟通能力 |
| **创新能力** | 创新和改进能力 |
| **成长发展** | 个人成长和能力提升 |
