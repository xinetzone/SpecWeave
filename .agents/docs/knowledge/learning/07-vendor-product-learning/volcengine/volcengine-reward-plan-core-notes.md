# 火山方舟协作奖励计划核心笔记

> **来源**: https://www.volcengine.com/docs/82379/1391869?lang=zh（最近更新2026.07.01）
> **提取日期**: 2026-07-07
> **产品**: 火山方舟协作奖励计划——"数据换免费Tokens"用户增长与数据采集计划，第二期合作期至2026年7月31日
> **说明**: 控制台页面（https://console.volcengine.com/ark/region:cn-beijing/openManagement/rewardPlan）需登录认证无法直接访问，本文档基于火山引擎官方公开文档整理。

---

## 一、活动简介

### 1.1 开通前提
请确保已开通模型服务并关闭安心体验模式，未完成实名认证的用户需先完成认证。

### 1.2 活动入口
两个路径进入协作奖励计划详情页：

- **方式一**：控制台-开通管理页 → 点击「立即参与」
- **方式二**：模型广场 → 模型详情页 → 点击「火山方舟模型奖励计划」（当前支持模型：Doubao-Seed-Evolving）

进入详情页后可完成**授权**或**撤回授权**操作。

### 1.3 活动参与范围
- 完成企业认证的用户可参与，享受企业权益
- 完成个人实名认证的用户可参与并享受个人权益，可在活动页面完成企业认证并手动确认升级享受企业权益

### 1.4 活动规则说明

1. 首次授权模型及推理接入点参与本计划，可获得该模型的冷启动免费资源包，用于抵扣按tokens后付费用量，奖励额度以产品活动介绍页公示为准
2. 调用已获授权的推理接入点，每日将采集每个模型不超过500万tokens的推理数据，并于次日发放有效期为30天的等量免费资源包；未获授权的接入点数据不予采集，该部分使用量次日不返还资源包
3. 个人实名认证用户享有个人权益，完成企业认证升级后享有企业权益，采集量及免费资源包上限将提升至企业权益对应水平
4. 免费资源包可用于抵扣在线推理过程中推理输入、推理输出以及推理缓存命中的按token后付费计费用量。暂不支持抵扣批量推理、按模型单元付费、TPM保障包以及推理缓存存储等计费项
5. 平台仅针对部分模型和调用方式进行数据采集，通过不支持的方式调用时数据不会被采集，次日也不会发放奖励。当前支持的调用方式：Chat API、Responses API、Messages API、Image Generation API、Content Generation API（仅在线推理）
6. 支持采集的模型以产品活动介绍页公示为准
7. 近期参与人数较多，资源包下发可能存在延迟

### 1.5 数据安全保障

- **隐私无忧**：授权数据会被严格加密和匿名化处理
- **无授权不采集**：从未授权、撤回授权或本计划终止时，数据不会被采集
- **授权自主**：可随时在活动页面撤回授权

## 二、协作奖励计划权益说明（第二期）

- **合作期**：原2024年12月1日至2025年5月31日，延长至2026年7月31日
- **冷启动资源包**：首次成功授权某模型推理接入点后，立即获得该模型免费额度资源包
- **每日奖励包**：授权后在每日额度内自动采集数据，次日根据前一日采集量返还等量每日奖励包
- 奖励包仅可用于抵扣按Token付费的后付费推理费用
- 奖励包发放后30个自然日内有效，到期清零
- 可同时使用任意一款或几款模型服务，每款模型按推理接入点独立申请授权并提供奖励
- 同一用户唯一个账号参与（同一主体多账户不可同时申请）
- 协作奖励数据（调用模型输入和输出的所有内容）将授权给火山引擎用于模型和算法优化

---

## 三、数据授权说明

- **采集范围**：奖励包额度内调用指定模型期间输入和输出的所有内容（文本、声音、图形、图片、图像、视频等）
- **处理方式**：匿名化处理后加密存储，用于火山引擎及其关联公司的模型算法和服务优化
- **特别说明**：不在采集范围内的数据不会被采集和存储
- **授权期限**：默认永久，但撤回授权后后续调用数据不再提供，奖励计划终止，未用完的奖励包仍可在有效期内使用
- **历史数据说明**：已授权并已被使用的数据在技术上无法撤回，火山引擎仅能停止使用新输入的数据
- **撤回授权可能影响相应模型服务的继续使用**

---

## 四、特别说明

1. **同一用户判定**：多个账户通过同一手机号、证件号、账号ID或其他经排查为同一实际控制人的，均视为同一用户
2. **反作弊**：对恶意抢占资源、批量刷单及其他非常规手段影响公平性的行为，有权取消资格、回收资源包，必要时追究法律责任
3. **规则变更权**：火山引擎可根据运营情况变更活动规则、商品数量、有效期等，以页面最新内容为准；变更前已发放的权益不受影响

---

## 五、火山引擎数据授权使用协议核心要点

- **授权性质**：非独家、不可转让、不可分许可（但可分许可给关联方和第三方外包服务商）、免费
- **授权权利**：传输、存储、使用、复制、下载、修改、处理协作奖励数据
- **授权目的**：机器学习/人工智能技术和模型的优化、开发、使用、学术研究及发表
- **授权期限**：永久，但可通过撤回授权终止新数据授权
- **用户承诺**：合法持有数据、个人信息合规、不提供保密数据/商业秘密、特殊数据需提前审批
- **被禁用途**：诽谤侮辱、违法活动、侵权内容等
- **被禁行为**：反向工程、绕开限制、移除标识、分发再许可、开发竞争性产品、传播病毒等
- **争议解决**：被告所在地有管辖权法院
