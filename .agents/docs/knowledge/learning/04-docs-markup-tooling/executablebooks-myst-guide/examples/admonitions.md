# Admonitions（提示框）样式大全

本示例展示 MyST 所有内置的 admonition 样式。每种样式都有不同的颜色和图标，用于表达不同语义的提示信息。

## note（备注）

用于一般性的备注或说明信息。

渲染效果：

:::{note}
这是 note 样式的提示框，用于展示一般性的备注信息。
:::

对应代码：

````
:::{note}
这是 note 样式的提示框，用于展示一般性的备注信息。
:::
````

## Admonition 嵌套使用

Admonitions 可以嵌套使用，但建议不要嵌套过深，以免影响可读性。

渲染效果：

:::{note} 外层提示框
这是外层的 note 提示框。

:::{warning}
这是嵌套在外层内部的 warning 提示框。
:::

外层内容继续。
:::

对应代码：

`````
:::{note} 外层提示框
这是外层的 note 提示框。

:::{warning}
这是嵌套在外层内部的 warning 提示框。
:::

外层内容继续。
:::
`````
