# 术语统一表

本表用于统一跨文化比较中的关键术语，防止中英文、古今义、不同学派间的概念混用。

---

## 1. 中国哲学术语（按体系分类）

### 1.1 道家术语

| 术语 | 拼音 | 常见英译 | 本档案用法 | 核心含义 |
|------|------|---------|-----------|---------|
| 道 | Dao | Way, Logos, Reason, 音译Dao | 保留"道（Dao）"，不翻译 | 不可名状的终极实在，兼具本体/生成/规律/境界四重维度 |
| 德 | De | Virtue, Power, Excellence, 音译De | 保留"德（De）"，不简单译为virtue | 道在个体事物中的落实，物之本真本性 |
| 自然 | Ziran | Nature, spontaneity | 译为"自己如此"，首见说明"非自然界" | 自己如此、本然状态、自如其然；非对象化的自然界 |
| 无为 | Wuwei | Non-action, inaction, effortless action | 译为"不妄为/因任自然"，首次注明"非什么都不做" | 不妄加干预、辅万物之自然、不将主观意志强加于事物 |
| 朴 | Pu | Uncarved block, simplicity | 朴/素朴 | 未经人为雕琢的本真状态 |
| 玄 | Xuan | Mystery, profundity | 玄/幽深 | 深远不可测的状态 |
| 齐物 | Qiwu | Equality of things | 齐物/万物齐一 | 从道的角度看万物平等无差别的观照 |
| 逍遥 | Xiaoyao | Free and easy wandering | 逍遥（游） | 不受束缚的精神自由境界 |

## 2. 西方哲学术语（与中国概念比较时的用法规范）

| 西方术语 | 在中国哲学中可能的误植 | 本档案使用规范 |
|---------|----------------------|---------------|
| **First Principles（第一性原理）** | 直接等同于"道""本""理" | 特指西方从亚里士多德到马斯克的思想传统，中国哲学概念"在某些维度可比较"但不等同 |
| **Metaphysics（形而上学）** | 用来描述道家/儒家的本体论思想 | 谨慎使用，中国哲学没有明确的"形而上学/物理学"二分 |
| **Logic（逻辑/逻辑学）** | 说"中国哲学没有逻辑" | 墨家逻辑、因明学都是逻辑的不同形态，不是只有亚里士多德式形式逻辑 |
| **Empiricism（经验主义）** | 将墨家"三表法"等同于经验主义 | 三表法包含历史和功利维度，不同于西方经验主义 |
| **Rationalism（理性主义）** | 将朱熹"即物穷理"等同于理性主义 | 朱熹的"理"是道德本体，不仅是认知理性 |
| **Idealism（唯心主义）** | 将王阳明"心学"标签为主观唯心主义 | 避免用唯心/唯物二元框架切割中国哲学 |
| **Intuition（直觉/直观）** | 将"体道""良知""现量"等同于普通直觉 | 这些概念都与修养/修行境界相关，不是普通直觉 |
| **Pragmatism（实用主义）** | 将墨家"用之者"等同于实用主义 | 墨家"利"是"兴天下之利"，有明确的道德-政治指向 |
| **Natural Law（自然法）** | 将"天理""道法自然"等同于自然法 | 道家"自然"是"自己如此"，非西方式理性法则 |
| **Deduction/Induction（演绎/归纳）** | 将三支作法区分为演绎或归纳 | 三支是归纳与演绎结合，喻依保证归纳基础 |

## 3. 翻译/用词基本原则

1. **核心概念保留原名加音译**：道（Dao）、德（De）、理（Li）等核心概念首次出现时保留原名加括号音译，不强行翻译
2. **古今义区分**：注意"自然""诚""格物"等词的古今异义，首次出现必须说明古义
3. **学派差异标注**：同一概念在不同学派/不同注家有不同含义时，明确标注（如格物致知的朱王异同）
4. **不等同声明**：跨文化比较时，首次对应必须加"不能简单等同于"或类似限定语
5. **避免二元标签**：不使用"东方直觉/西方理性""中国整体/西方分析"这类笼统二元标签
