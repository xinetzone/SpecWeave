# 使用指南

## 1. 题库定位

本题库是 [08-methodology-framework.md](../08-methodology-framework.md) 的配套刻意练习材料，目标是帮助读者通过实际问题训练，将第一性原理六步框架（问题定义→假设列举→拆解要素→质疑验证→从头重构→现实迭代）从理论理解转化为可调用的思维习惯。

需要说明：本题库不承诺"做完就能精通第一性原理"——思维方式的改变需要长期刻意练习和真实场景打磨，这里提供的是结构化的入门训练材料。它也不是智力测验，没有唯一标准答案，重点在于思考过程的质量而非得出"正确结论"。

## 3. 使用指南

1. **建议按顺序练习**：Step 1到Step 6的专项练习有递进关系，前面的基础打好后面效果更好
2. **独立思考优先**：每道题先自己尝试，不要急于看参考答案。答案区仅提供一种参考思路，不代表唯一正确解法
3. **写下来**：思维训练必须写下来（或打字记录），仅在脑中想容易跳过关键步骤，也无法事后复盘
4. **时间盒**：每道题给自己15-30分钟，避免无限拖延或过度分析
5. **复盘**：做完后对照参考答案，重点关注：自己遗漏了什么？哪个步骤卡壳了？暴露了什么思维习惯？
6. **迁移到真实问题**：做完练习题后，尝试用同样方法分析你工作生活中遇到的真实问题，这才是训练的最终目的

## 4. 难度分级说明

题目按三个难度等级标注：

| 标识 | 等级 | 说明 |
|------|------|------|
| 🌱 | 入门 | 场景简单，有明确方法指引，主要训练单一步骤的基础操作，适合刚接触框架时练习 |
| 📚 | 进阶 | 需要多步思考，可能跨领域组合，隐含假设较多，需要识别常见陷阱，适合完成入门练习后 |
| 🔥 | 挑战 | 有认知偏差陷阱，问题边界模糊，可能涉及反常识结论，需要较强的自我质疑能力，适合有一定练习基础后 |

难度是相对的——同一道题对不同背景的人感受可能不同，不必因为某道题卡住就沮丧，重点是过程中的思考。

## 5. 练习前准备

在开始做题前，建议：

1. **通读方法论框架**：完整阅读 [08-methodology-framework.md](../08-methodology-framework.md)，理解六步每一步的目标、操作方法和常见陷阱
2. **准备笔记本**：物理笔记本或电子文档都可以，关键是能随时记录你的思考过程、假设、拆解树
3. **准备时间**：找不被打扰的时间段，每道题用时间盒约束（15-30分钟），不要在碎片时间做
4. **心态准备**：
   - 接受自己一开始会做得不好——这是正常的，思维训练需要过程
   - 不要急于得出"聪明"答案，重点是走完流程
   - 对自己诚实：发现自己在用类比而不是从原理出发时，及时停下来
   - 记住：第一性原理思维是高成本活动，不必对所有问题都用，这里是练习场景

## 7. 答案查看约定

所有参考答案均使用HTML折叠标签包裹，**请务必先独立思考并写下你的分析过程，再点击展开查看**。直接看答案会大幅降低训练效果。

参考答案仅代表一种可能的思考路径，不是唯一正确答案。如果你的思路和参考答案不同但逻辑自洽、有合理依据，那同样是有效的——甚至可能更好。重点关注自己的思考过程是否完整走完了框架的每一步。
