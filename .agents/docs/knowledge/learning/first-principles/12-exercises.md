# 第一性原理思维训练题库

本题库是 [08-methodology-framework.md](08-methodology-framework.md) 的配套刻意练习材料，帮助读者通过实际问题训练，将第一性原理六步框架从理论理解转化为可调用的思维习惯。

> ⚠️ **不做万能承诺**：思维方式的改变需要长期刻意练习和真实场景打磨，这里提供的是结构化的入门训练材料。没有唯一标准答案，重点在于思考过程的质量而非得出"正确结论"。

## 推荐练习路径

1. **首次接触**：从 [00-intro.md](exercises/00-intro.md) 开始 → 按Step 1→6顺序做所有🌱入门题 → 做07-pitfalls前3题 → 选1个📚综合案例
2. **进阶练习**：完成所有📚进阶题 → 完成07-pitfalls全部10题 → 完成剩余2个综合案例
3. **深度挑战**：完成所有🔥挑战题 → 用09-practice-guide中的方法迁移到真实问题
