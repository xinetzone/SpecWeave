## 二、Pavo 平台定位介绍

Pavo 是 Agnes AI 推出的PC端AI创作平台，其核心理念是将AI创作所需的各项能力整合到统一的工作流中，让用户无需在多个工具之间切换。

### 2.1 平台定位

Pavo 将**图片生成、视频生成、短剧创作**三大能力整合到一个工作流中，并加入Agent作为总指挥，实现从创意想法到最终成片的全流程在浏览器内完成。

### 2.2 解决的核心痛点

在Pavo出现之前，AI创作面临的普遍问题：

- **工具碎片化**：做图片用一个工具，做视频用另一个工具，写剧本还要换工具
- **流程割裂**：各环节之间需要手动导出导入素材
- **学习成本高**：需要掌握多个工具的操作方式
- **效率低下**：大量时间花在工具切换和格式转换上

Pavo 通过一站式工作流设计，系统性地解决了这些问题。

## 三、四大核心模块详解

Pavo 平台包含四个相互配合的核心模块，覆盖从需求描述到最终成片的完整创作链路。

### 3.1 Agent（智能指挥模块）

Agent 是整个平台的"大脑"和总指挥，用户只需用一句话描述需求，Agent会自动规划执行步骤、调用相应模型、产出所需素材。

**核心特点**：
- 自然语言交互，无需专业操作知识
- 自动任务拆解与规划
- 支持上下文记忆和多轮对话调整
- 基于自研Harness框架构建

### 3.2 图片生成模块

图片生成模块支持多种场景的图片创作需求：

| 应用场景 | 说明 |
|----------|------|
| **海报设计** | 商业海报、活动海报等营销物料 |
| **商品图制作** | 电商产品展示图、场景化商品图 |
| **写真创作** | 人像写真、风格化人像摄影 |

### 3.3 视频生成模块

视频生成模块提供两种核心能力：

1. **文生视频**：直接通过文字描述生成视频内容
2. **图生视频**：基于已有图片生成动态视频

支持运镜控制、画面风格调整等专业参数设置。

### 3.4 剧情短片模块（核心特色）

剧情短片模块是Pavo最具差异化的功能，提供从剧本到成片的完整短剧工作流，将原本需要多个工具配合、大量人工操作的短剧制作过程变成标准化流水线。

## 四、剧情短片全流水线（核心特色）

市面上大多数AI视频工具只能生成单个镜头，制作完整短剧需要创作者自己拼接剧本、人设、分镜、配音、剪辑等环节，门槛高、效率低。Pavo的剧情短片模块将整个生产过程变成8步标准化流水线，并在关键节点设置人机交互确认点。

### 4.1 八步生产流水线

| 步骤 | 环节名称 | 具体内容 |
|------|----------|----------|
| **第1步** | 剧本创作 | 根据用户需求自动生成完整剧本 |
| **第2步** | 人物和场景设定 | 设计剧中角色形象与故事场景 |
| **第3步** | 分镜设计 | 将剧本拆解为具体镜头序列 |
| **第4步** | 角色场景配图生成 | 生成角色设定图和场景概念图 |
| **第5步** | 分镜脚本撰写 | 详细描述每个镜头的运镜、画面、台词 |
| **第6步** | 分镜关键帧设计 | 设计每个镜头的关键画面 |
| **第7步** | 分镜视频生成 | 基于关键帧生成各镜头视频片段 |
| **第8步** | 成片渲染 | 将所有镜头合成为完整成片 |

### 4.2 六个人机交互确认点

在流水线的关键节点，系统会暂停等待用户确认，确保创作方向符合预期：

1. **需求确认**：确认视觉风格、时长等基础参数
2. **剧本确认**：审核并修改自动生成的剧本
3. **角色确认**：确认人物设定和场景设计
4. **配图确认**：审核角色和场景配图效果
5. **分镜确认**：审核分镜脚本和关键帧设计
6. **合成确认**：最终成片前的整体确认

### 4.3 实际案例演示

以一个科幻场景创作为例：

> **用户输入**："晴朗的天空突然变暗，然后大大小小的飞碟、宇宙飞船、战舰在天空中飞过，最后最大的母舰突破云层和远处大山相撞"

**系统执行流程**：
1. 首先确认视觉风格、视频时长等参数
2. 自动撰写符合描述的剧本
3. 设计出场角色（如需要）和天空、山脉、飞碟等场景
4. 生成各元素配图供用户确认
5. 撰写详细分镜脚本（包含运镜方式、画面描述、音效提示）
6. 设计每个分镜的关键帧
7. 逐镜生成视频片段
8. 渲染合成最终成片

### 4.4 适用场景

剧情短片工作流特别适合以下内容类型：

| 场景类型 | 说明 |
|----------|------|
| **热点短剧** | 紧跟热点话题的快速短剧制作 |
| **职场轻喜剧** | 办公室场景、职场吐槽类短内容 |
| **情侣剧情** | 情感类、恋爱日常短剧 |
| **萌宠拟人** | 宠物配音、拟人化萌宠故事 |
| **品牌剧情广告** | 带故事情节的品牌广告片 |

### 4.5 当前局限性

需要客观认识的是，Pavo剧情短片目前还存在一定局限：

- **电影级质感尚有差距**：追求顶级电影质感目前还无法完全实现
- **人工调整仍有必要**：复杂场景和高要求制作仍需人工介入调整
- **细节控制待提升**：某些微妙的表情、动作细节可能不够精准

## 五、Agent 指挥系统：一句话跑完创作

Agent 是 Pavo 平台的核心调度中枢，基于 Agnes AI 自研的 Harness 框架构建，能够理解用户自然语言需求并自动完成复杂创作任务。

### 5.1 技术底座：Harness 框架

Agent 运行在自研的 Harness 框架之上，该框架提供：

- 任务规划与拆解能力
- 多模型调度与协同
- 上下文记忆管理
- 工具调用编排
- 多轮对话支持

### 5.2 工作方式

用户只需用自然语言描述需求，例如：

> "帮我为一款香水做一张商业海报和一条15秒广告视频"

Agent 接收到指令后会自动执行：

1. **理解需求**：解析"香水""商业海报""15秒广告视频"等关键要素
2. **任务拆解**：拆分为海报设计和视频制作两个子任务
3. **步骤规划**：确定每个子任务的执行步骤
4. **模型调用**：自动调用图片模型生成海报，调用视频模型生成广告
5. **结果整合**：将产出的素材整理呈现给用户

整个过程无需用户手动操作各个模块，真正实现"一句话完成创作"。

### 5.3 上下文记忆与多轮调整

Agent 支持上下文记忆功能，这意味着：

- 无需每次重新描述整个项目背景
- 可以基于已有创作进行调整修改
- 多轮对话中保持对项目整体的理解
- 支持"把海报色调调暖一点""视频节奏再快些"这类迭代指令

## 六、Agnes-Video-2.5-preview 模型升级

Agnes AI 即将上线 Agnes-Video-2.5-preview 视频模型，此次升级针对批量生产场景进行了重点优化，尤其适合广告投放、短视频矩阵这类需要大量产出、不停试错的应用场景。

### 6.1 五大升级要点

| 升级项 | 具体改进 | 价值 |
|--------|----------|------|
| **生成速度** | 提升一倍 | 产出效率翻倍，批量制作成本降低 |
| **运镜流畅度** | 减少僵硬和突兀感 | 视频更自然，专业感提升 |
| **物理规律还原** | 减少漂浮、穿模、不自然碰撞 | 画面真实感增强，减少人工修正 |
| **多参考图能力** | 新增多图参考，人物和商品保持一致 | 系列视频中角色/产品形象统一 |
| **字幕乱码问题** | 显著改善字幕乱码 | 带字幕视频可用性提升 |

### 6.2 升级的战略方向

从这五项升级可以看出，Agnes-Video-2.5-preview 的目标非常明确——**冲着批量生产去的**。

对于广告投放、短视频矩阵这类场景：
- 需要大量生成不同版本进行A/B测试
- 需要人物/产品形象在多条视频中保持一致
- 需要快速迭代、快速试错
- 对生成速度和成本敏感

2.5版本的升级恰好击中了这些痛点。

## 十、常见问题解答（FAQ）

### Q1：Agnes AI 真的完全免费吗？

**A**：根据文章信息，Agnes AI 在3周前免费开放了文本、图片、视频三个核心模型的API，目前确实处于免费阶段。但需要注意的是，文章并未说明免费策略会持续多久，也未提及后续的收费计划。对于需要长期稳定使用的用户，建议关注官方公告获取最新定价信息。

### Q2：Pavo 如何访问？

**A**：Pavo 是PC端AI创作平台，通过浏览器访问即可使用，地址是：https://app.pavo-ai.work/。无需下载安装客户端，打开浏览器登录后即可使用全部功能。建议使用Chrome、Edge等现代浏览器获得最佳体验。

### Q3：剧情短片能达到什么质量水平？

### Q4：Agent 支持哪些类型的创作任务？

**A**：Agent基于Harness框架，目前支持四大模块对应的创作任务：
