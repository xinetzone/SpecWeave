# 自学质量检查清单

---

这是一个简化版的七概念掌握程度自检清单（共20项，完整版是33项）。学完前三个模块后，用这个清单快速评估自己的掌握程度。

**使用方法**：
- 对每一项，根据自己的情况打✅（掌握）、❓（部分掌握）、❌（没掌握）
- 不要骗自己，诚实地评估
- 打❓和❌的项，回去复习对应的章节

---

### 结果解读

- **🏆 90-100分：掌握优秀**
  - 你已经很好地掌握了七概念的核心内容，可以开始用它分析自己的项目了
  - 建议：去做[延伸阅读](03-further-reading.md)中的完整方法论，加深理解

- **✅ 75-89分：基本掌握**
  - 你已经理解了主要内容，但还有一些薄弱点
  - 建议：把打❓的项对应的章节复习一遍，然后独立完成[练习2](01-practice-exercises.md#练习2用七概念分析一个你熟悉的开源项目工具60-90分钟)

- **⚠️ 60-74分：部分掌握**
  - 你可能记住了一些概念，但还没有真正理解，也不会用
  - 建议：不要着急往下学，重新读一遍[理论篇](../01-seven-concepts-theory/README.md)和[融合分析篇](../03-analysis/README.md)中你打❌的部分，然后从[练习1](01-practice-exercises.md#练习1七概念快速自检30分钟个人练习)重新开始

- **❌ <60分：需要重学**
  - 目前的理解还比较零散，建议从头开始系统学习
  - 建议：按照[2小时系统学习路径](00-reading-guide.md#路径22小时系统学习)重新走一遍，不要跳步
