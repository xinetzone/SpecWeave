# DeepTutor案例 - 索引

---

## 一句话介绍

DeepTutor是一个由香港大学数据科学实验室（HKUDS）维护的开源AI学习工作空间，将辅导、解题、测验、研究、可视化、掌握练习六种学习模式塞进同一个Agent引擎，提供智能体原生的终身个性化辅导体验。

---

## 项目背景

> GitHub 25k Star，香港大学数据科学实验室的DeepTutor，开源100天左右冲到20k。
>
> ——§0.0 导语

DeepTutor由香港大学数据科学实验室（HKUDS）开源维护，是一个Agent原生的学习工作区。Agent循环写在底层，Chat、Quiz、Research、Solve、Visualize、Mastery Path六种学习模式共享一个runtime，支持本地模型和Docker一键部署，在GitHub上获得了25k Star。
