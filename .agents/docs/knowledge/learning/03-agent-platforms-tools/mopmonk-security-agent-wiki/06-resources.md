## 六、相关资源链接

### 学术论文

1. **CyberGym官方论文**
   - 链接：https://arxiv.org/pdf/2506.02548
   - 说明：UC Berkeley团队发布的CyberGym基准论文，详细介绍了评测设计、数据集构建、评估方法，是理解AI安全Agent评测的核心文献。发表于ICLR 2026。

2. **MiniMax M3技术博客**
   - 链接：https://www.minimax.io/blog/minimax-m3
   - 说明：MiniMax官方发布的M3模型技术介绍，详细解释了MSA稀疏注意力架构、各项基准成绩、训练方法等。

3. **Terminal-Bench论文**
   - 链接：https://arxiv.org/pdf/2601.11868
   - 说明：M3取得66.0%成绩的Terminal-Bench基准论文，了解命令行环境Agent评测标准。

### 代码仓库

4. **MopMonk官方GitHub仓库**
   - 链接：https://github.com/MopMonkAI/MopMonkAgent
   - 说明：MopMonk唯一的官方信息渠道。虽然代码目前没有开源，但README里有完整的技术报告，详细介绍了三大核心技术设计、七类记忆对象、评测结果，是理解MopMonk最权威的第一手资料。

5. **CyberGym官方网站**
   - 链接：https://www.cybergym.io/cybergym/
   - 说明：CyberGym基准官方网站，可以查看最新榜单、了解评测框架细节。

### 媒体报道与解读

6. **原始微信公众号文章**
   - 链接：https://mp.weixin.qq.com/s/Y_8DYQGuxgHdiw-a74ZN0w
   - 说明：最早报道MopMonk的中文文章之一，也是本教程的主要信息来源，用通俗易懂的语言介绍了MopMonk的来龙去脉和技术亮点。

7. **36氪/新智元等媒体报道**
   - 说明：MopMonk出圈后，国内多家科技媒体都进行了报道和解读，搜索"MopMonk 扫地僧"可以找到更多分析文章。

### 延伸阅读

8. **MiniMax M3独立评测**
   - 说明：M3发布后有独立研究者进行了第三方评测（如DeepSWE），可以搜索"MiniMax M3 independent benchmark"找到更多独立测试结果，帮助你全面了解模型的真实表现（注意：不同基准评测结果可能有差异，建议多方参考）。
