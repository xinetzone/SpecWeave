# 入门指南

欢迎使用 **miniTest** — 无需组建QA团队即可获得的移动端QA测试覆盖。miniTest是一款AI驱动的移动端自动化测试平台，通过AI代理Mini在虚拟设备上运行用户故事，为您的iOS和Android应用提供自动化测试保障。

## miniTest能为您做什么

- **无需QA团队**：如果您正在发布iOS或Android应用却没有自己的QA团队，miniTest正是为您量身打造
- **AI代理执行**：Mini作为AI QA工程师，选择构建版本并在虚拟设备上运行您关注的用户故事
- **明确的测试结论**：每次运行结束后给出明确的判定结果（通过/失败/警告）
- **可操作的反馈**：当发现问题时，提供视频录像、复现步骤、设备日志以及可直接粘贴到Cursor或Claude的修复提示
