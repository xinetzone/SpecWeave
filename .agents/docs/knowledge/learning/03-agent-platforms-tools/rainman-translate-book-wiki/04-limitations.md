# 局限性与注意事项

Rainman Translate Book 虽然在整书翻译场景下表现出色，但并非万能。本章客观分析其局限性，帮助读者在合适的场景下使用它。

---

## 环境依赖门槛

Rainman Translate Book 的安装需要配置多个工具链：

- Claude Code CLI（需 Anthropic 账号）
- Calibre
- Pandoc
- Python 3 + pypandoc + beautifulsoup4

这一整套环境对于**非技术用户**来说门槛较高。如果你不熟悉命令行操作、环境变量配置、Python 包管理，安装过程可能会遇到不少障碍。相比之下，Claude 网页版或 ChatGPT 等在线工具的使用门槛要低得多。

**建议**：非技术用户如果有整书翻译需求，可以考虑请技术朋友帮忙配置环境，或等待未来可能出现的在线版本。

---

## Calibre 转换格式丢失风险

Calibre 在格式转换阶段（PDF/DOCX/EPUB → HTMLZ → Markdown）可能丢失以下内容：

### 代码块

技术书籍中常见的代码块，在 Calibre 转换过程中可能出现：
- 缩进丢失
- 语法高亮信息丢失
- 代码与正文混排
- 特殊字符被错误转义

### 数学公式

LaTeX 数学公式在转换中可能：
- 渲染为乱码
- 结构被破坏
- 丢失上下标关系

### 特殊排版

以下排版元素在转换中风险较高：
- 表格（复杂表格可能结构错乱）
- 脚注与尾注（可能丢失关联关系）
- 图文混排（图片可能错位）
- 多栏排版（可能合并为单栏，内容顺序混乱）

### 应对策略

- **优先使用 EPUB 作为输入**：EPUB 的结构化程度最高，转换质量最好
- **避免使用扫描版 PDF**：扫描版 PDF 中的文字是图片，Calibre 无法提取
- **翻译完成后对照原文校对**：对格式要求高的内容（代码、公式、表格），建议人工校对

## 翻译质量依赖 Claude 模型本身

Rainman Translate Book 本质上是一个翻译流水线，翻译质量的上限取决于 Claude 模型的语言能力。以下情况可能出现翻译质量问题：

- **文学性强的文本**：诗歌、小说中的修辞手法和文学韵味难以被 AI 准确传达
- **文化特定表达**：成语、俚语、文化典故可能翻译不准确
- **领域专有术语**：如果 glossary 未能覆盖所有领域术语，可能出现翻译不一致
- **一词多义**：在没有充分上下文的情况下，AI 可能选择错误的义项

## 适用场景边界

### 最适合的场景

| 场景 | 适合原因 |
|---|---|
| 英文技术书籍 | 术语相对固定，glossary 能有效覆盖 |
| 学术论文 | 篇幅适中，术语规范 |
| 技术手册/文档 | 结构化程度高，翻译质量要求清晰 |
| 非虚构类书籍 | 语言风格相对平实，AI 翻译质量较好 |

### 不太适合的场景

| 场景 | 不适合原因 |
|---|---|
| 文学作品（小说/诗歌） | 文学修辞和韵味难以被 AI 准确传达 |
| 扫描版 PDF | 文字无法提取，需先 OCR |
| 大量代码/公式/图表的书籍 | Calibre 转换可能丢失格式 |
| 需要高度专业人工审校的文本 | 法律文件、医学文献等需要专业资质的翻译 |

## 与其他方案对比

### vs Claude 网页版直接翻译

| 维度 | Rainman Translate Book | Claude 网页版 |
|---|---|---|
| 上下文窗口 | 8 个独立窗口，互不污染 | 单个窗口，越长越容易混乱 |
| 术语一致性 | glossary.json 硬约束锁定 | 无约束，越到后面术语越乱 |
| 断点续传 | 支持，SHA-256 哈希比对 | 不支持，中断后需手动重来 |
| 多格式输出 | 5 种格式自动输出 | 需手动复制粘贴 |
| 使用门槛 | 需要配置工具链 | 打开网页即可使用 |
| 适用场景 | 整书翻译 | 短文翻译、片段翻译 |

### vs 传统 CAT 工具（如 Trados、memoQ）

| 维度 | Rainman Translate Book | 传统 CAT 工具 |
|---|---|---|
| 翻译引擎 | Claude（AI 翻译） | 翻译记忆库 + 人工翻译 |
| 术语管理 | glossary.json（自动提取） | 术语库（手动维护） |
| 翻译质量 | AI 级别，需人工校对 | 人工翻译，质量可控 |
| 翻译速度 | 快（8 并行） | 取决于人工翻译速度 |
| 费用 | API 费用 | 人工翻译费用 |
| 适用场景 | 快速获取可读译本 | 正式出版级翻译 |
