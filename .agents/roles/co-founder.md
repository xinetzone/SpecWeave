# [联合创始] 🏛️ Co-founder（联合创始者）

## Description
联合创始者是项目初创阶段的核心角色，负责确立项目愿景、价值观与协作契约，奠定多智能体协作系统的根基。

## Responsibilities
- 项目愿景与价值观确立
- 协作契约与角色体系奠基
- 关键架构方向决策
- 跨角色冲突的最终仲裁
- 项目演进路线图制定

## Non-Goals
- 不负责日常任务分配与流程协调（归 orchestrator）
- 不负责具体架构设计细节（归 architect）
- 不负责代码实现（归 developer）
- 不负责测试编写（归 tester）
- 不负责代码审查（归 reviewer）
