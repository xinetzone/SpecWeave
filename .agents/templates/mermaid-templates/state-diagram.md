# 状态图模板

```mermaid
stateDiagram-v2
    [*] --> "待处理"
    "待处理" --> "处理中" : "开始执行"
    "处理中" --> "已完成" : "执行成功"
    "处理中" --> "失败" : "执行失败"
    "失败" --> "待处理" : "重试"
    "已完成" --> [*]
```

## 状态图语法要点

- 使用 `stateDiagram-v2`（v2版本兼容性更好）
- 中文状态名直接使用，含空格的状态名需加引号
- `state 状态ID : "中文描述"` 格式为状态添加描述，描述文本加引号
- 转换箭头 `-->` 后标签含中文时用引号包裹：`-->"触发条件"`
- `[*]` 表示起始/结束状态
- 代码块内禁止空行

## 含复合状态示例

```mermaid
stateDiagram-v2
    [*] --> "待机"
    "待机" --> "运行中" : "启动"
    state "运行中" as RUNNING {
        [*] --> "初始化"
        "初始化" --> "执行任务"
        "执行任务" --> "等待"
        "等待" --> "执行任务" : "新任务到达"
    }
    RUNNING --> "待机" : "停止"
    "待机" --> [*]
```
