## 📐 统一设计系统（9张图共用）

### 1. 背景层

| 元素 | 规范 |
|------|------|
| 底色 | **统一 `#0F172A`**（不要每张图换颜色） |
| 可选纹理 | 极淡网格线（`#1E293B`，透明度8%，间距40px）增加质感 |
| 截图蒙层 | 截图上方必须盖 `rgba(15,23,42,0.75)` 半透明黑，确保文字可读 |

### 2. 卡片组件系统

整套作品只用以下5种卡片样式，不发明新样式：

| 卡片类型 | 样式 |
|---------|------|
| **标准卡片** | 背景 `#1E293B`，圆角16px，1px边框 `#334155`，无阴影 |
| **高亮卡片（蓝）** | 背景 `rgba(59,130,246,0.12)`，圆角16px，1px边框 `#3B82F6` |
| **成功卡片（绿）** | 背景 `rgba(16,185,129,0.12)`，圆角16px，1px边框 `#10B981` |
| **警示卡片（红）** | 背景 `rgba(239,68,68,0.1)`，圆角12px，1px边框 `rgba(239,68,68,0.3)` |
| **HEX/RGB小标签** | 背景 `#0F172A`，圆角6px，内边距10px 6px |

### 3. 文字层级系统

严格使用以下6种字号，不使用中间值（如30px/38px/50px）：

| 层级 | 字号(px) | 字重 | 颜色 | 用途 |
|------|---------|------|------|------|
| H1 封面大标题 | 96 | Heavy特粗 | `#F8FAFC` 白 | 仅限图1、图9使用 |
| H2 页面主标题 | 56 | Bold粗 | `#F8FAFC` 白 | 每张图顶部标题 |
| H3 卡片标题/数字 | 44 | Bold粗 | `#F8FAFC` 白 | 卡片内标题、大数字 |
| Body 正文 | 34 | Regular | 白/浅灰 | 主要说明文字 |
| Label 标签/辅助 | 28 | Regular | `#94A3B8` 浅灰 | 说明文字、备注 |
| Small 小字 | 24 | Light | `#64748B` 中灰 | 标签、分类、@、话题 |

> ⚠️ **字号纪律**：9张图总共只用这6种字号。字号统一=专业感。

### 4. 间距系统（基于8px网格）

| 间距类型 | 数值(px) |
|---------|---------|
| 页面边距 | 左右80，上下100 |
| 大标题与内容间距 | 60 |
| 卡片内边距 | 32 |
| 卡片与卡片间距 | 32 |
| 标题与正文间距 | 20 |
| 正文行间距 | 1.5倍（故事页2倍） |
