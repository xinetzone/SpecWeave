## 一、文章概述

Anthropic 发布全新企业协作工具 Claude Tag，定位为 Claude Code 的进化，使 Claude 成为可主动介入、异步执行的团队共享 AI 同事。卡帕西称其为 LLM 用户界面的第三次重大变革——第一次是网页版聊天，第二次是桌面应用，第三次是 LLM 变成独立、持续运行的系统，拥有组织内的工具和上下文，能与人类团队协同工作。
