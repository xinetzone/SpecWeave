## 6. 研究局限说明

1. **注疏传统覆盖限制**：因明部分主要引用玄奘译原典，窥基《因明大疏》等汉传因明注疏未充分展开，这是因为因明学过于专业，作为普及性比较研究已足够；专业研究需进一步阅读窥基疏、陈大齐《因明大疏蠡测》等
2. **一手文献范围**：道家、儒家、墨家均采用最权威的通行注本，但未做不同版本异文的全面校勘（如《老子》郭店楚简本、马王堆帛书本与通行本的文字差异）——本研究以概念分析为目的，非文献校勘研究
3. **佛教部分范围**：佛教因明仅涉及陈那新因明的汉传系统，对藏传因明（法称系统）涉及较少
4. **未覆盖流派**：法家、兵家、名家等流派按计划未纳入（见spec Open Questions回答）
5. **比较哲学深度**：本研究是初步的跨文化方法论比较，不是严格的比较哲学专著，更深入的比较需要比较哲学领域专家进行
