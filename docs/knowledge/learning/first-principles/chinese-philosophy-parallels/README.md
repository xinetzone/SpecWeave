# 跨文化第一性原理比较研究档案

本档案是第一性原理知识体系的v2.0扩展，系统性地搜集和整理中国古代哲学（道家、儒家、墨家、佛教因明学）中与西方"第一性原理"思维具有可比性的核心概念和方法论资源。在v1.0西方哲学→物理→商业主线的基础上，本档案增强了文化包容性，纠正了"第一性原理=西方思想"的隐性偏误。

> **🌏 核心警示**：本档案的目的不是证明"中国古代早已有第一性原理"，而是通过跨文化比较深化对根本性思维的理解。中国哲学概念不能简单等同于西方概念，差异先于共性，尊重各体系的本源性含义是第一原则。

## 4. 核心结论摘要

### 4.1 最重要的一个结论

> **第一性原理思维不是西方独有的。** 道家"返本复初"、儒家"知本""格物致知"、墨家"三表法""察类明故"、佛教因明"因三相""立宗出因"，都体现了"从根本出发进行思考和推理"的思维倾向。但四家与西方第一性原理在**问题意识、真理观、方法论核心、实践者角色**上存在深刻差异，不能简单等同。

### 4.2 中国传统的8个独特方法论贡献

1. **减法思维**（道家）：为道日损，不仅做加法也要做减法
2. **无为方法论**（道家）：辅万物之自然，不妄为也是根本方法
3. **价值优先原则**（儒家）：本立而道生，价值方向比方法更根本
4. **知行合一**（儒家）：知而不行只是未知，认知与实践不可分
5. **三维验证框架**（墨家）：历史经验+基本事实+实践效果
6. **必要条件分析**（墨家）：识别"缺了不行"比"有了就行"更重要
7. **对话论辩逻辑**（因明）：为他比量考虑语境和受众
8. **归纳演绎结合**（因明+墨家）：喻依经验基础防止空洞演绎

### 4.3 跨文化七步框架（速查）

| 步骤 | 核心问题 | 主要文化来源 |
|------|---------|-------------|
| 0. 价值定向 | 为什么做？为了谁？根本价值是什么？ | 儒家 |
| 1. 质疑假设+做减法 | 哪些是未经质疑的假设？去掉什么？ | 共通+道家 |
| 2. 溯源求故 | 根本原因是什么？必要条件是什么？ | 共通+墨家 |
| 3. 标准确立 | 用什么三维标准判断？ | 墨家+因明 |
| 4. 分析重建 | 如何拆解重构？整体是否合理？ | 西方+因明 |
| 5. 边界意识 | 适用条件是什么？无为是否更好？ | 互补 |
| 6. 知行合一共验证 | 如何在实践中检验？是否需要修正？ | 互补 |

### 4.4 阅读时必须记住的三句话

1. **差异先于共性**——先看不同，再看相似
2. **原典先于比较**——先理解概念本义，再谈跨文化对应
3. **实践先于理论**——所有方法论最终指向更好的实践，不是空谈

## 6. 使用建议与注意事项

### 6.1 使用建议

1. **不要找"中国的第一性原理"**——这种寻找本身就是西方中心的。应该研究各文化自身的根本性思维传统，平等对话
2. **不要简单比附**——看到一点相似就说"X就是Y"，这是最常见的反向语义漂移
3. **用互补视角看问题**——遇到问题时，既用西方的分析方法，也用中国的价值/边界/减法智慧
4. **从实践出发**——方法论不是用来谈论的，是用来用的，从小事开始练习七步法

### 6.2 局限说明

- 本档案聚焦四家核心概念，未覆盖法家、兵家、名家等流派（见Open Questions回答）
- 佛教因明部分侧重汉传陈那系统，对藏传因明（法称系统）涉及较少
- 古文引用以通行权威注本为依据，异文争议仅选择重大分歧标注，非专业校勘版本
- 跨文化比较是哲学上的难题，本框架是初步尝试，欢迎批评和深化
