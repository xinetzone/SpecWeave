### /zh-Hans-CN/codex/for-work (为工作打造的Codex)

**页面标题**: Codex 是你的个人工作助手

**核心价值主张**: 它可以利用你的工具完成调研、整理、起草和分析工作，并实现重复性任务的自动化

**主要功能点**:

1. **几分钟内创建文档、电子表格和幻灯片**
   - 告诉 Codex 去哪里查找，它会完成调研
   - 几分钟内生成摘要、演示文稿、邮件和报告

2. **实现重复性任务的自动化**
   - 将重复性工作交由 Codex 处理
   - 下一次更新、草稿、审查材料或跟进内容在需要时自动准备就绪

3. **专注于真正重要的工作**
   - 将繁琐工作交由 Codex 处理
   - 专注于与客户交流、做出决策和制定战略

**工作方式**:
- 提供上下文，选择输出结果
- 决定哪些内容会被使用、发送、分享或自动化
- 选择一项团队已经在做的工作（发布简报、客户总结、KPI报告或后续跟进）
- 通过每一次输出逐步建立信任

**更多资源链接**:
- [Codex 指南](https://openai.com/academy/codex-for-work/) - 了解如何使用 Codex 实现任务自动化、构建工作流

### developers.openai.com/codex (开发者文档首页)

**描述**: Codex is OpenAI's coding agent for software development. ChatGPT Plus, Pro, Business, Edu, and Enterprise plans include Codex.

**核心能力**:
- **Write code**: Describe what you want to build, and Codex generates code that matches your intent, adapting to your existing project structure and conventions.
- **Understand unfamiliar codebases**: Codex can read and explain complex or legacy code, helping you grasp how teams organize systems.
- **Review code**: Codex analyzes code to identify potential bugs, logic errors, and unhandled edge cases.
- **Debug and fix problems**: When something breaks, Codex helps trace failures, diagnose root causes, and suggest targeted fixes.
- **Automate development tasks**: Codex can run repetitive workflows such as refactoring, testing, migrations, and setup tasks so you can focus on higher-level engineering work.

## 八、关键术语与概念

- **Codex**: OpenAI的AI编程智能体，可自主完成代码编写、调试、审查、测试等任务
- **GPT-5.3-Codex / GPT-5-Codex**: 专为编程优化的模型
- **GPT-5.5 / GPT-5.5 Pro**: 最新旗舰模型
- **本地消息**: 在本地IDE/CLI/桌面应用中执行的任务
- **云端任务**: 在云端沙箱中执行的任务，可连接GitHub运行测试
- **代码审查**: Codex的PR/代码审查功能
- **审批模式**: 安全控制机制，包括Chat/ReadOnly、Agent、Full Access等级别
- **智能体模式 (Agent Mode)**: AI自主规划并执行多步骤任务
- **MCP**: Model Context Protocol，工具扩展协议
- **沙箱 (Sandbox)**: 隔离的执行环境，确保代码运行安全
