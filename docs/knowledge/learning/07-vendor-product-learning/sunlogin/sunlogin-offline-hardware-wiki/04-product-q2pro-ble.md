## 五、Q2Pro产品详解

### 产品定位

Q2Pro是向日葵无网远控硬件产品线中的工业级智能4G远控设备，定价1599元。Q2Pro以4G全网通、4K@60Hz超高清、工业级宽温设计为核心卖点，面向医疗设备、工业控制、机房维护等对可靠性和环境适应性要求较高的专业场景。

> 注：Q2Pro的官方URL路径为q2pro-ble，但实际产品页面展示的是Q2Pro工业级4G版本，"BLE"指代蓝牙低功耗（Bluetooth Low Energy）绑定功能。

### 核心特性

#### 1. 蓝牙绑定，多种连接方式

Q2Pro支持灵活的绑定和配网方式：
- 支持蓝牙5.0绑定，初始配置快速稳定
- 支持有线以太网、WiFi、4G多种上网方式
- 适应不同用户的连接需求和场景
- 蓝牙近场配网，无需网线连接即可完成配置

#### 2. 内置4G全网通模块

Q2Pro内置4G通信模块，实现真正的无网环境远控：
- 即插即用的自动锁式SIM卡槽
- 4G全网通支持（LTE-FDD/TDD）
- 无需依赖现场WiFi或有线网络
- SMA外置LTE天线接口，信号稳定
- 适合野外、矿山、无网络覆盖的工业场景

#### 3. 4K@60Hz HDMI高清传输

Q2Pro-V3版本支持超高清视频传输：
- 刷新频率60Hz
- 最高支持3840×2160（4K）分辨率
- 支持宽域分辨率覆盖
- 满足医疗、工控设备的特殊分辨率需求（CT、超声、核磁、内窥镜等）
- 高清画质便于查看和分析精密图像

#### 4. 无系统障碍，无缝兼容

Q2Pro兼容全平台操作系统：
- Windows
- macOS
- Linux
- Android
- 适配所有操作系统和软件环境
- 无需安装驱动，即插即用

#### 5. 支持KVM切换器多主机控制

与控控2类似，Q2Pro可搭配KVM切换器：
- 实现一个设备控制多台主机
- 将传统KVM升级为IPKVM
- KVM切换器需另行购买
- 适合机房服务器集群管理

#### 6. 免驱动、免客户端

纯物理外置硬件设计：
- 无需安装驱动程序
- 无需安装第三方软件
- 即插即用
- 不被杀毒软件拦截，不与系统软件冲突

#### 7. 远程文件传输

Q2Pro支持远程文件传输功能（需企业服务）：
- 接入U盘或TF卡后，可与被控设备传输文件
- 支持传输运维日志、安装包等文件
- 大幅提升运维和办公效率
- 需增购企业服务配件并完成企业实名认证

#### 8. 工业级设计，高可靠性

Q2Pro按照工业级标准设计，适应恶劣环境：

| 工业特性 | 规格 |
|---|---|
| 工作温度 | -20℃~70℃宽温范围 |
| 存储温度 | -40~85℃ |
| 外壳材质 | SECC镀锌钢板（金属外壳） |
| 电源输入 | 双电源输入：12V 2A 或 9-36V 2.7-0.7A宽电压 |
| 防护设计 | 防浪涌设计，提升电源抗雷击性能 |
| 看门狗 | 内置看门狗，保障稳定运行 |
| 安装方式 | DIN35导轨安装 + 壁挂蝴蝶扣安装 |

#### 9. 数据安全

Q2Pro采用金融级安全机制：
- 多因子验证
- 全链路金融级加密
- 国际2048位RSA非对称密钥交换
- 基于AES加密机制的自主P2P数据传输协议

#### 10. 付费功能免费赠送

Q2Pro赠送两项付费功能（免费一年）：
- **远程SSH**：实现排查网络异常，远程操作命令行界面和远程执行命令
- **远程摄像头**：搭配USB摄像头，实时监控设备运行情况，随时随地可用手机监控

### 安装方式

Q2Pro支持两种标准安装方式：

#### DIN35导轨安装（工业机柜）

1. 需自行购买DIN35mm导轨卡扣（孔距39mm，建议M3螺丝固定）
2. 将Q2Pro导轨卡扣所在面面向导轨
3. 将导轨卡扣向上拉起，将设备安装在导轨上

#### 壁挂式蝴蝶扣安装

1. 在安装面定位两个点，两点距离82mm
2. 在两个点钉上钉子，建议挂台高出壁面4~6mm
3. 将设备按孔位挂上

### 购买建议

Q2Pro适合以下用户：
- 医疗设备厂商，需要远程维护CT/超声/核磁等大型医疗设备
- 工业控制场景，需要-20~70℃宽温适应能力
- 机房运维，需要千兆网口+4G冗余、DIN导轨安装
- 野外/无网络环境，需要内置4G全网通
- 对可靠性要求高，需要双电源、看门狗、防浪涌等工业级特性
- 需要4K@60Hz超高清传输
- 需要远程文件传输、SSH、摄像头等企业级功能

如果预算有限且不需要工业级特性，可选择Q1（298元）；如果需要5G、协同远控、双向语音等更高级功能，建议选择Q5Pro；如果仅需物理隔离的近场控制，建议选择Q0.5。
