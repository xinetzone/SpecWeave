## 九、应用场景与选型指南

### 八大典型场景产品推荐

#### 场景1：企业IT机房日常运维

**场景描述**：
- 中小企业/大型企业IT部门管理机房服务器
- 服务器通常无显示器（无头部署）
- 需要远程开关机、BIOS设置、系统排障
- 预算中等，环境温度正常（室内机房）
- 网络环境：机房有线网络或WiFi覆盖

**推荐产品**：
- **首选：Q1**（298元）—— 性价比最高，满足基础机房运维需求，百兆网口足够，蓝牙绑定部署简便
- **备选：控控2** —— 如果需要更多功能、支持KVM切换器多主机控制
- **多主机场景**：Q1/控控2 + KVM切换器，一个设备管理多台服务器

**不推荐**：Q0.5（只能近场，无法远程）、Q5Pro（功能过剩）

---

#### 场景2：工业设备/PLC远程监控

**场景描述**：
- 工厂车间工控机、PLC、HMI触摸屏等设备
- 环境恶劣：高温/低温、粉尘、电磁干扰
- 可能没有有线网络或WiFi
- 需要7×24小时稳定运行
- 设备系统封闭，不允许安装软件

**推荐产品**：
- **首选：Q2Pro**（1599元）—— 工业级设计，-20~70℃宽温，SECC镀锌钢板抗干扰，DIN35导轨标准安装，双电源冗余，内置4G全网通应对无WiFi环境，防浪涌看门狗保障稳定
- **超高端场景：Q5Pro** —— 如果需要5G或更高带宽

**不推荐**：Q1（消费级温度范围，无4G，无导轨安装）、Q0.5（只能近场）

---

#### 场景3：医疗设备远程维护（CT/超声/核磁）

**场景描述**：
- 医院大型医疗设备：CT、MRI（核磁）、超声、内窥镜、DR等
- 设备价值高，分布在不同医院，厂商现场维护成本极高
- 医疗设备系统封闭，严禁安装第三方软件
- 需要高清画质查看医疗影像（4K需求）
- 手术室环境要求高，设备需稳定可靠
- 可能涉及手术示教、远程指导需求

**推荐产品**：
- **基础远程维护：Q2Pro**（1599元）—— 4K@60Hz（V3版本），工业级设计适应手术室环境，4G联网不依赖医院内网，免驱即插即用
- **手术示教/远程指导：Q5Pro** —— 5G低延迟+4K高清+双向语音+协同远控（授权确认），满足临床教学和远程会诊需求
- **备选（仅现场）：Q0.5** —— 工程师现场维护时的物理KVM工具

**不推荐**：Q1（分辨率不够，无4K，无工业级防护）

---

#### 场景4：涉密服务器/隔离网络现场运维

**场景描述**：
- 保密单位、政府、军工、金融核心系统
- 网络物理隔离，严禁任何设备联网
- 防止第三方跳板攻击、防止截屏泄露
- 必须现场物理接触才能运维
- 安全等级要求极高
- 服务器通常在封闭机房内

**推荐产品**：
- **唯一选择：Q0.5**（158元）—— 完全无网设计（无网口/WiFi/蓝牙/蜂窝），物理线缆直连，零网络攻击面，硬件级防跳板防截屏，从被控端取电无需额外电源，口袋大小随身携带
- **使用方式**：运维工程师进入机房后，用Q0.5临时连接服务器进行操作，操作完成后拔走设备，不留下任何网络接入点

**绝对禁止使用**：任何网络型产品（Q1/Q2Pro/Q5Pro/控控2）—— 因为它们带有网络模块，违反物理隔离规定

---

#### 场景5：野外/无网络环境设备远控

**场景描述**：
- 矿山、油田、电力基站、铁塔、水利设施、户外广告屏
- 无有线网络、无WiFi覆盖
- 环境温度变化大（可能高温或严寒）
- 需要远程监控状态、排查故障、升级固件
- 设备分布分散，现场出差成本高

**推荐产品**：
- **4G覆盖区：Q2Pro**（1599元）—— 内置4G全网通，-20~70℃宽温适应户外环境，SMA外置天线信号好
- **5G覆盖/高带宽需求：Q5Pro** —— 双卡5G保障网络冗余，4根天线信号更强，适合需要传输高清视频的场景
- **使用建议**：自行办理物联网流量卡（运营商4G/5G卡），根据流量需求选择套餐

**不推荐**：Q1（无4G）、Q0.5（只能近场）、控控2（4G需外接网卡，可靠性不如内置）

---

#### 场景6：远程医疗示教/手术指导

**场景描述**：
- 顶级专家远程指导基层医院手术
- 手术画面4K实时传输
- 双向语音实时沟通
- 需要医院方授权确认（医疗合规）
- 低延迟要求（操作指令实时传达）
- 可能需要录制手术过程用于教学

**推荐产品**：
- **唯一推荐：Q5Pro** —— 5G网络低延迟大带宽，4K高清内窥镜影像传输，3.5mm音频口支持双向语音，协同远控模式（显示器弹窗+语音提示+蜂鸣器，被控方按键确认授权），USB外接摄像头可拍摄手术室全景

**为什么不选Q2Pro**：无双向音频、无协同授权模式、无5G，不满足示教场景需求

---

#### 场景7：出差应急随身携带

**场景描述**：
- IT运维工程师经常出差到客户现场
- 需要一个随身携带的"万能运维工具"
- 客户现场环境未知，可能没网、可能没显示器键鼠
- 设备要小、轻、便携，最好口袋就能装下
- 应急排障用，不一定需要远程
- 预算有限

**推荐产品**：
- **首选：Q0.5**（158元）—— 70×70×18mm口袋尺寸，无需电源无需联网，即插即用，任何现场都能用，价格便宜不心疼，还能防跳板满足涉密客户要求
- **备选（需要远程）：Q1**（298元）—— 如果需要从酒店远程操作客户机房设备，Q1体积也很小（75×45×17mm，63g），但需要带电源适配器且需要联网
- **最佳组合**：Q0.5（现场应急）+ Q1（远程备用），总预算456元，覆盖绝大多数场景

---

#### 场景8：多主机机房KVM升级

**场景描述**：
- 机房已有传统KVM切换器
- 希望升级为IPKVM，实现远程访问
- 保护现有KVM投资
- 多台服务器共用一套KVM
- 可能有8口/16口/32口KVM切换器

**推荐产品**：
- **企业级：控控2** 或 **Q2Pro** + 现有KVM切换器 —— 三款产品（控控2/Q2Pro/Q5Pro）都明确支持搭配KVM切换器使用，将传统KVM的输出连接到向日葵设备的HDMI-IN和USB口即可升级为IPKVM
- **工业级环境选Q2Pro**：千兆+4G+工业宽温+导轨安装
- **标准机房选控控2**：功能全面的旗舰IPKVM
- **专业场景选Q5Pro**：如果需要5G和协同功能

**不推荐**：Q1（官方未明确支持KVM切换器，百兆网口可能不够）、Q0.5（只能一对一近场）

### 选型决策树

根据以下决策流程快速选型：

```
开始选型
   │
   ├─ 是否需要异地远程控制？
   │   ├─ 否（只能现场/物理隔离环境）→ Q0.5（158元，物理近场，极致安全便携）
   │   └─ 是（需要远程）
   │       │
   │       ├─ 预算是否<500元？
   │       │   ├─ 是 → Q1（298元，百兆+蓝牙，基础远控）
   │       │   └─ 否 → 继续判断
   │       │
   │       ├─ 是否需要双向语音/协同远控/手术示教？
   │       │   ├─ 是 → Q5Pro（专业级5G，双向语音+协同+USB映射）
   │       │   └─ 否 → 继续判断
   │       │
   │       ├─ 是否需要工业级宽温/内置4G/DIN导轨/4K@60Hz？
   │       │   ├─ 是 → Q2Pro（1599元，工业级4G，4K）
   │       │   └─ 否 → 控控2（旗舰IPKVM，功能全面）
```

### 多产品组合建议

对于专业运维团队和大型组织，建议按以下组合配置：

**入门级运维工具包（约456元）**：
- Q0.5 ×1（158元）：现场应急、涉密场景
- Q1 ×1（298元）：日常远程运维
- 覆盖：90%以上的常规运维场景

**专业级运维工具包（约2055元）**：
- Q0.5 ×1（158元）：现场应急、涉密场景
- Q2Pro ×1（1599元）：工业/医疗/4G场景
- 可能还需要Q1 ×1（298元）：办公室/小机房日常
- 覆盖：工业、医疗、机房、野外、涉密等几乎所有场景

**医疗/高端专业套件（价格需咨询）**：
- Q5Pro ×1：手术示教、高端设备远程维护
- Q2Pro ×1：常规医疗设备运维
- Q0.5 ×1：现场应急
- 覆盖：医疗设备全生命周期运维需求
