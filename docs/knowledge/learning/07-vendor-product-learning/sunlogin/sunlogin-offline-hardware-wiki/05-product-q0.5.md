## 六、Q0.5产品详解

### 产品定位

Q0.5是向日葵无网远控硬件产品线中的口袋级近场控制设备，定价158元，是五款产品中价格最低、体积最小巧、安全性最高的产品。Q0.5采用完全不同的技术路线——纯物理近场控制，无需联网、无需接电、防三方跳板远控，专门面向数据机房、工控机、涉密服务器等高安全等级场景的现场运维需求。

### 核心特性

#### 1. 画质清晰，低延迟

Q0.5提供高质量的近场视频传输：
- 视频输出：1920×1080@60帧（1080P/60帧输出）
- 低延时不卡顿
- 满足现场运维的流畅操作需求

#### 2. 小巧轻便，可随身携带

极致口袋化设计：
- 外形尺寸：仅70mm × 70mm × 18mm
- 掌心大小，口袋尺寸
- 出差随身携带毫无压力
- 适合IT运维工程师应急排障

#### 3. 防三方跳板远控

Q0.5最重要的安全特性：
- 服务器等机密设备仅允许现场控制
- 防止第三方软件远控截屏
- 物理层面杜绝网络攻击
- 满足高安全等级环境要求

#### 4. 连线即可近场控制

Q0.5的工作方式极其简单：通过线缆连接即可近场控制设备，无屏/无键鼠也能轻松驾驭。

具体连接方式：
- HDMI-IN连接被控主机的HDMI输出（视频采集）
- Type-C（被控端）连接被控主机的USB接口（USB仿真+取电）
- Type-C（主控端）连接主控设备（电脑/手机）（视频输出+控制信号）
- USB-A接口可将USB设备映射到被控电脑

连接完成后，通过向日葵主控端即可操作被控设备。

#### 5. 免驱免安装，即插即用

延续向日葵硬件的核心优势：
- 无需安装驱动程序
- 无需安装第三方软件
- 线连即可使用
- 操作系统自动识别标准USB设备

#### 6. 无需联网接电

Q0.5最独特的设计：
- 无需配网——设备本身没有网口、WiFi、蜂窝模块
- 无需单独拉线供电——从被控主机USB接口取电
- 线连被控主机和主控设备即可工作
- 完全脱离网络环境运行

#### 7. 多系统兼容

被控主机系统支持：
- Linux
- Windows
- macOS

> 注：Q0.5主要针对PC操作系统，官方未明确提及Android/iOS支持。

### 接口布局

Q0.5配备四个接口（按官方图标顺序）：

| 接口 | 标识 | 功能 |

| USB-A | USB-A图标 | 将USB设备映射到被控电脑 |

### Q0.5的安全模型

Q0.5采用了物理安全模型，与网络型产品有本质区别：

| 安全维度 | Q0.5（物理近场） | 网络型产品（Q1/Q2Pro/Q5Pro/控控2） |
|---|---|---|
| 网络攻击面 | 无（完全无网） | 有（网口/WiFi/蜂窝） |
| 远控距离 | 线缆长度限制（通常1-2米） | 全球任何有网络的地方 |
| 防跳板攻击 | 物理隔离，无法跳板 | 需要加密协议防护 |
| 防截屏 | 硬件层面防止软件截屏 | 依赖软件方案 |
| 数据传输方式 | 点对点线缆直连 | 通过网络/互联网传输 |
| 安全等级 | 最高（物理级） | 高（金融级加密） |
| 涉密场景 | 完全适配 | 需评估 |

这种"物理隔离+近场控制"的设计思路使得Q0.5成为高安全等级环境的唯一选择——不联网就不存在被网络攻击的可能性，必须物理接触才能控制。

### 典型应用场景

#### 数据机房

- 物理隔离的数据中心服务器运维
- 涉密机房设备现场维护
- 无显示器"无头"服务器的临时接控
- 高安全等级机房排障

#### 工控机现场运维

- 工厂车间工控机现场调试
- 不允许联网的PLC/工业控制设备
- 产线设备现场维护
- 工程师随身携带应急排障

#### 涉密服务器/高安全环境

- 保密单位内网服务器
- 物理隔离网络设备运维
- 严禁任何外联设备的环境
- 需要防止三方跳板远控的场景

#### 出差应急随身携带

- IT工程师出差排障的"最后一公里"工具
- 口袋大小，随身携带无负担
- 无需考虑网络环境，即插即用
- 应急场景下连接显示器键鼠不便时使用

### 与网络型产品的本质区别

Q0.5代表了无网远控的两种技术路线之一——"物理近场型"，与Q1/Q2Pro/Q5Pro/控控2代表的"网络远控型"形成互补：

| 对比维度 | Q0.5（物理近场型） | 网络型产品（远程型） |
|---|---|---|
| 控制距离 | 线缆长度（约1-2米） | 全球互联网可达 |
| 联网需求 | 完全不需要 | 需要（网口/WiFi/4G/5G） |
| 供电需求 | 从被控端取电 | 需要独立电源 |
| 安全模型 | 物理隔离 | 加密协议 |
| 典型使用方式 | 现场运维 | 远程/异地运维 |
| 价格带 | 158元 | 298元-高端 |
| 体积重量 | 最小最轻 | 更大更重 |
| BIOS控制 | 支持 | 支持 |
| 免驱免客户端 | 是 | 是 |

### 购买建议

Q0.5适合以下用户：
- 需要进入涉密机房、物理隔离环境进行现场运维的工程师
- 对安全性要求极高，不允许任何网络连接的场景
- IT运维工程师随身携带的应急排障工具
- 出差时需要一个"口袋里的KVM"应对各种现场情况
- 预算有限（158元），需要一个物理KVM替代方案
- 需要防止第三方软件跳板远控和截屏的场景

Q0.5不适合以下场景：
- 需要异地远程控制（Q0.5只能近场）
- 需要从家里/办公室远程操作机房设备（请选择Q1/Q2Pro/Q5Pro/控控2）
- 需要4G/5G蜂窝网络远控（请选择Q2Pro/Q5Pro）
- 需要4K超高清或双向语音协同（请选择Q2Pro/Q5Pro）

**最佳实践**：对于专业运维团队，建议同时配备Q0.5（现场应急）和Q2Pro/Q5Pro（远程运维），形成远近结合的完整运维工具链。
