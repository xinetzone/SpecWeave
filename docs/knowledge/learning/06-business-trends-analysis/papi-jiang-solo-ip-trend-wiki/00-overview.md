# Papi酱关闭公司回归个人IP：概述与学习目标

## 一、背景介绍

Papi酱（姜逸磊）作为中国第一代现象级网红，其创业轨迹堪称中国内容创业史的缩影。2015年凭借短视频在微博迅速走红，2016年获得1200万元融资、第一条广告拍出2200万元天价，随后成立papitube并并入泰洋川禾，走上MCN机构化道路。

十年后的2026年，Papi酱将名下6家公司全部注销，团队缩减至7人，选择回归个人IP模式。这一转变并非孤立事件，而是罗永浩、李子柒、李佳琦等超级IP共同选择的趋势缩影。

卢松松作为拥有十年创业经历的资深互联网人，以自身业务发展（松松商城、松松软文、云主机代购、自媒体代运营）为对照，对这一趋势进行了深度反思。
