## 6.3 供应链管理专业资源

### 书籍推荐

| 书名 | 作者 | 推荐理由 |
|------|------|---------|
| 《供应链管理》 | Sunil Chopra等 | 供应链管理经典教材 |
| 《物流与供应链管理》 | Martin Christopher | 物流管理基础 |
| 《采购与供应链管理》 | Robert Monczka等 | 采购管理专业书籍 |
| 《供应链风险管理》 | Stephen J. Melnyk等 | 供应链风险评估与管理 |

### 核心概念

| 概念 | 描述 |
|------|------|
| **供应链韧性** | 供应链在面对中断时的恢复能力 |
| **供应链多元化** | 通过多来源降低单一供应商依赖风险 |
| **价值增值率** | 本土生产创造的价值占总价值的比例 |
| **库存周期** | 从采购到销售的平均时间 |

### 分析工具

| 工具 | 用途 |
|------|------|
| **SWOT分析** | 优势、劣势、机会、威胁分析 |
| **PESTEL分析** | 政治、经济、社会、技术、环境、法律分析 |
| **波特五力分析** | 行业竞争格局分析 |
| **供应链地图** | 可视化供应链网络 |
