# 什么是OKR

> **来源**: [最全OKR制定指南（2种思路+7类方法）](https://www.dingteam.com/help/584)

## 定义

OKR（Objectives and Key Results，目标与关键成果法）是一套定义和跟踪目标及其完成情况的管理工具与方法。

**核心公式**: `我们将实现（目标），通过（关键结果）来衡量`

## 组成部分

### Objective（目标）

Objective是定性描述的、鼓舞人心的方向性目标。它回答"我们要去哪里？"这个问题。

**特点**:
- 定性描述，而非定量指标
- 鼓舞人心，能够激发团队热情
- 指明方向，而非具体任务
- 有时限，明确时间范围（通常一个季度）

### Key Results（关键结果）

Key Results是定量的、可衡量的指标，用于判断目标是否达成。它回答"我们如何知道到达了那里？"这个问题。

**特点**:
- 定量描述，有明确的数字指标
- 可衡量，能判断完成程度
- 强相关，直接支撑Objective的达成
- 具挑战，60%-70%完成率为最佳平衡点

## OKR的制定流程

```
公司级OKR → 部门级OKR → 团队级OKR → 个人级OKR
    ↓           ↓           ↓           ↓
  战略拆解    承接公司      承接部门      承接团队
  明确方向    结合分工      突出贡献      个人定位
```

## 关键要点

1. **聚焦重点**: 一个周期内目标不宜过多，建议控制在3-5个
2. **挑战导向**: 目标要有挑战性，60%-70%的完成率是理想状态
3. **公开透明**: OKR应全员可见，促进跨部门协同
4. **敏捷迭代**: 按季度周期推进，定期复盘调整
