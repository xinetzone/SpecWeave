# OKR对齐机制

> **来源**: [叮当OKR落地实操详细指南](https://www.dingteam.com/help/582)

## 什么是对齐？

对齐是指将自己的OKR与他人的O或KR进行关联，确保团队目标与公司目标一致，做到"心往一处想，力往一处使"。

## 为什么要对齐？

### 对齐的价值

| 角色 | 价值 |
|------|------|
| **老板** | 可以看到目标的承上启下，更好带动自己管理的部门 |
| **管理员** | 可以看到所有目标对齐情况，能快速识别目标的对齐关系是否合理 |
| **部门主管** | 可以清楚看到公司的OKR全貌，知晓战略目标如何被分解 |
| **员工** | 实现Everyone is leader，像老板一样思考 |

### 具体好处

1. **战略有支撑**: 可以清楚看到公司的OKR全貌，知晓战略目标如何被分解，更能时刻了解目标进展
2. **战术有技巧**: 可以看到所有目标对齐情况，能快速识别目标的对齐关系是否合理，更好更方便的推进OKR管理
3. **协同有方向**: 实现Everyone is leader，员工可以像老板一样，知道公司的战略是什么，部门的目标是什么，同事的目标是什么

---

## 如何操作对齐？

### 操作方式一

**操作**: 在【OKRs】页面制定好OKR后，点击上方【添加对齐】

**步骤**:
1. 制定OKR
2. 点击【添加对齐】
3. 选择需要对齐的目标

### 操作方式二

**操作**: 制定好OKR后在【对齐】页面，找到需要对齐的目标点击【+】号进行对齐

**步骤**:
1. 制定OKR
2. 进入【对齐】页面
3. 找到需要对齐的目标
4. 点击【+】号进行对齐

### 查看对齐关系

**操作**: 操作完成后可以在【对齐】页面查看对齐关系

**要点**:
- 查看自己的目标与哪些目标对齐
- 查看哪些目标对齐了自己的目标
- 了解对齐层级关系

---

## 对齐注意事项

### 对齐对象

**对齐O**: 对齐上级的目标，适用于职责不明确的情况

**对齐KR**: 对齐上级的关键结果，适用于职责明确的情况

### 对齐方向

**向上对齐**: 与上级目标对齐，确保纵向一致

**横向对齐**: 与跨部门目标对齐，确保协同顺畅

### 对齐金字塔模型

```
          公司OKR（顶层）
              ↓
          部门OKR（中层）
              ↓
          团队OKR（基层）
              ↓
          个人OKR（执行层）
```
