- 每个KR单独评分
- O的分数为其下所有KR的平均分
- 备注栏填写评分原因和说明

## 模板2：进阶评分模板

> 来自：北京希嘉创智教育科技有限公司 田林 的分享

### 评分维度

| 维度 | 权重 | 说明 |
|------|------|------|
| **完成度** | 50% | 实际完成情况 |
| **挑战性** | 30% | 目标的挑战程度 |
| **影响度** | 20% | 对团队/公司的影响 |

### 评分表

| Objective | KR | 完成度(0-1) | 挑战性(0-1) | 影响度(0-1) | 综合评分 | 备注 |
|-----------|----|-------------|-------------|-------------|----------|------|
| O1 | KR1 | [分数] | [分数] | [分数] | [分数] | [说明] |
| | KR2 | [分数] | [分数] | [分数] | [分数] | [说明] |
| | **O平均分** | | | | [分数] | |

**使用说明**:
- 考虑目标的挑战性和影响度
- 更全面地评估OKR的价值
- 鼓励设定有挑战性的目标

## 模板3：简化评分模板

> 来自：中国联合网络通信有限公司四平市分公司 的分享

### 评分标准

| 等级 | 分数 | 说明 |
|------|------|------|
| **S** | 0.9-1.0 | 超出预期，表现卓越 |
| **A** | 0.7-0.8 | 符合预期，表现优秀 |
| **B** | 0.5-0.6 | 部分完成，表现一般 |
| **C** | 0.3-0.4 | 进展较小，需要改进 |
| **D** | 0-0.2 | 未完成，需重新审视 |

### 评分表

| Objective | KR | 目标值 | 实际值 | 等级 | 分数 | 备注 |
|-----------|----|--------|--------|------|------|------|
| O1 | KR1 | [目标值] | [实际值] | [等级] | [分数] | [说明] |
| | KR2 | [目标值] | [实际值] | [等级] | [分数] | [说明] |
| | **O平均分** | | | | [分数] | |

**使用说明**:
- 使用等级制，直观易懂
- 便于团队沟通和理解
- 简化评分流程
