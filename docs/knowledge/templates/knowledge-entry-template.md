## 元数据说明

| 字段 | 说明 | 可选值 |
|------|------|--------|
| security_level | 安全级别 | public / internal / confidential |
| knowledge_type | 知识本质类型 | factual / procedural / conditional / metacognitive |
| validation_status | 验证状态 | draft / verified / peer-reviewed / deprecated |
| reuse_count | 复用次数 | 整数，每次复用后递增 |
| integrity | 完整性校验和 | 写入时自动计算，无需手动填写 |

### 知识类型选择指南

| 类型 | 含义 | 典型场景 |
|------|------|---------|
| factual | 事实性知识——是什么 | 术语定义、API文档、配置说明 |
| procedural | 程序性知识——怎么做 | 操作指南、教程、工作流 |
| conditional | 条件性知识——何时/为何 | 决策矩阵、选型指南、场景分析 |
| metacognitive | 元认知知识——如何思考 | 方法论、认知模式、经验复盘 |
