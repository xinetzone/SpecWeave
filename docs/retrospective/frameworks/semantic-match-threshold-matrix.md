# 语义匹配阈值决策矩阵

## 来源
v1.1 可配置阈值优化

## 决策矩阵

| 场景 | 推荐阈值 | 理由 |
|------|---------|------|
| 中文短文本（2-5 字） | 1 | 中文词汇短，关键词交集少 |
| 中文中文本（6-15 字） | 2 | 平衡匹配精度与召回率 |
| 英文文本 | 2-3 | 英文单词较长，关键词交集天然多 |
| CI 严格模式 | 2-3 | 减少误报，仅报告明确不匹配 |
| 开发宽松模式 | 1 | 减少漏报，捕捉可能的遗漏 |

## 原则
阈值越低，漏报越少但误报越多；阈值越高，误报越少但漏报越多。按场景选择。
