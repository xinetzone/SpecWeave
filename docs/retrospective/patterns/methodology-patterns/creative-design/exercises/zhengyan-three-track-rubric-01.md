# 评分标准：反效率知识IP朋友圈文案创作与审查

> **配套练习**：[练习01：反效率知识IP朋友圈文案创作与审查](zhengyan-three-track-practice-01.md)
> **总分**：100分 | **及格线**：60分 | **优秀线**：90分

---

## 一、等级划分

| 分数区间 | 等级 | 说明 | 后续建议 |
|---------|------|------|---------|
| 90-100 | 🏆 优秀 | 双工具熟练掌握，文案有质感，可直接作为范文 | 尝试进阶挑战，或用自己的真实场景重做一遍 |
| 75-89 | ✅ 良好 | 方法基本掌握，文案合格，但细节有瑕疵 | 针对扣分维度重读模式文档，再做1-2条文案练习 |
| 60-74 | ⚠️ 及格 | 理解了方法，但应用有明显问题，文案有硬伤 | 重新做练习，重点关注第三轨立场审查 |
| <60 | ❌ 需重做 | 方法理解不到位，或跳过了关键步骤 | 重新精读两个模式文档，从零开始再做一遍 |

---

## 二、评分维度详解

### 维度A：正言若反写作法应用（35分）

#### A1. 对立式草稿完成度（9分，每条文案3分）

| 分值 | 评分标准 |
|------|---------|
| 3分 | 对立式清晰呈现核心张力，❌和✅两侧对比鲜明，能准确看出想反对什么/主张什么 |
| 2分 | 写出了对立式，但对比不够尖锐，或对立点找得不准（表面对立非本质对立） |
| 1分 | 有对立式但很模糊，读者看不懂❌和✅的区别在哪里 |
| 0分 | 跳过了对立式草稿，直接写终稿 |

**扣分细则**：
- 对立式两侧本质是同一回事（如❌慢✅慢慢来）→ 扣2分
- 只有一侧有内容，另一侧空白 → 扣2分
- 用❌✅对比写终稿（不是草稿）→ 本项0分（违反表达形式约束）

#### A2. 核心公式应用准确度（12分，每条文案4分）

| 分值 | 评分标准 |
|------|---------|
| 4分 | 完整应用"世人皆知…不知…不是不…是…"结构，"大X"的转折有辩证深度，"不是不X"的回护真诚不虚伪 |
| 3分 | 应用了公式，但"大X"较生硬（如"大拼""大学"读起来不像自然语言），或回护部分略显敷衍 |
| 2分 | 只用了公式的一部分（如只有"世人皆知…不知…"没有"不是不…是…"），或用了变体但变体选择不恰当 |
| 1分 | 生硬套用公式，读起来像绕口令/模板，完全没有辩证感 |
| 0分 | 没有使用正言若反句式，仍是❌vs✅二元对立或直白说教 |

**扣分细则**：
- "不是不X"后面接的是"你应该Y"（假回护）→ 每条扣3分
- "大X"直接等于"更X"（如"大快=更快"）——本质化理解"大" → 每条扣2分
- 套用公式但"世人皆知"的常识不成立（读者不认同这是常识）→ 每条扣2分

#### A3. 回护检查记录质量（8分）

| 分值 | 评分标准 |
|------|---------|
| 8分 | 三个问题都有真诚的反思，不是走过场；如果发现问题并做了修改，记录了修改前后的对比 |
| 6分 | 三个问题都回答了，但回答较笼统（如"不会被否定""挺真诚的"），没有具体分析 |
| 4分 | 回答了部分问题，或回答明显敷衍（全填"是/否"没有解释） |
| 2分 | 只有一两条文案有回护检查，其余缺失 |
| 0分 | 完全没有做回护检查 |

**扣分细则**：
- 回护检查全填"没问题"但实际文案有明显问题 → 扣4分
- 先写终稿再补回护检查（后补痕迹明显）→ 扣3分

#### A4. 语言自然度与文化质感（6分，整体评分）

| 分值 | 评分标准 |
|------|---------|
| 6分 | 文案读起来像一个真实的人在说有味道的话，没有AI感/营销感，愿意截图发朋友圈 |
| 4分 | 语言通顺，没有明显模板感，但也没有特别有味道/让人想停下来想一想的句子 |
| 2分 | 有明显的模板痕迹/AI腔，读起来像SOP生成的文案，不像真人写的 |
| 0分 | 语言生硬、拗口，或有明显的语病/不通顺 |

**扣分细则**：
- 出现"家人们谁懂啊""家人们冲啊"这类过度营销的语气 → 扣3分
- 过度使用文言句式导致读不懂（玄学化）→ 扣3分
- 三条文案结构完全一样（复制感强）→ 扣2分

---

### 维度B：约束审查三轨制执行（35分）

#### B1. 第一轨：关键词黑名单检查（8分）

| 分值 | 评分标准 |
|------|---------|
| 8分 | 逐词搜索了全部6类黑名单，准确记录了命中数/修复数，对每个命中都判断了"违规/合法语境"，最终文案零违规 |
| 6分 | 搜索了大部分黑名单，有记录，最终文案零违规或少有1-2处疏漏 |
| 4分 | 只搜索了部分黑名单（如只查了❌✅符号没查指令词），或有3-4处违规未发现 |
| 2分 | 只做了象征性检查，或最终文案有5处以上显性违规 |
| 0分 | 完全跳过第一轨，或第一轨记录为零但文案中有明显违规词 |

**常见遗漏关键词**（每漏1个扣1分）：
- "要""得""应该"这类口语化指令词
- "加油""坚持"这类日常到意识不到的游戏化词
- "成长"——既是心理学混入也是成功学常用词

**扣分细则**：
- 第一轨记录"零命中"但实际有违规 → 扣4分（不诚实记录）
- 发现了违规但没修复 → 扣3分
- 只列了黑名单没说搜了哪些词 → 扣2分

#### B2. 第二轨：概念白名单检查（10分）

| 分值 | 评分标准 |
|------|---------|
| 10分 | 逐词搜索了5个混入门派的关键词，准确判断了上下文合法性，正确处理了混入词（替换/保留标注/对比保留），最终文案零污染 |
| 7分 | 搜索了大部分混入词，概念判断基本正确，有1-2处混入未发现但属于不影响理解的副词级混入 |
| 5分 | 搜索了部分门派（如只查了佛家和成功学，没查心理学），或有3-4处概念污染 |
| 3分 | 白名单检查很粗略，或有5处以上明显混入 |
| 0分 | 完全跳过第二轨，或第二轨记录为零但文案中有明显混入词 |

**高风险混入词**（每出现1个扣2分）：
- "接纳""成长""疗愈""赋能"——心理学高频混入
- "知行合一""慎独"——儒家混入
- "觉悟""放下""解脱"——佛家混入
- "坚持""努力""突破""逆袭"——成功学混入
- "痛点""转化""裂变""变现"——营销话术混入

**扣分细则**：
- 第二轨记录"零混入"但实际有混入 → 扣5分
- 用混入词来命名目标体系概念（如用"知行合一"描述帛书老子的践行）→ 扣5分（严重污染）
- 把"坚持"这类词当正面词汇使用 → 扣4分

#### B3. 第三轨：哲学立场审查（12分）

| 分值 | 评分标准 |
|------|---------|
| 12分 | 7个维度全部检查到位，准确识别了本质化/终点化/权威化/普世化/科学化/二元残留/答案植入问题并修复，最终文案立场一致 |
| 9分 | 检查了大部分维度，发现并修复了主要立场问题，有1-2处微小残留但不影响整体立场 |
| 6分 | 做了立场审查但不够系统，漏掉了2-3个维度，或发现了问题但修复不到位 |
| 3分 | 立场审查很粗略，或有4处以上明显立场问题 |
| 0分 | 完全跳过第三轨，或文案有根本性立场错误（如核心卖点就是"慢下来更高效"） |

**7个维度的常见问题**（每类问题扣2分）：

| 维度 | 典型违规示例 | 正确表达方向 |
|------|------------|------------|
| 本质化 | "慢下来你会更高效/赚更多/更成功" | "我自己慢下来之后，体验到了一种不同的状态" |
| 终点化 | "做到这三点你就自由了/开悟了" | "这是一个持续的过程，没有终点" |
| 权威化 | "听我的，按我说的做" | "我观察到…也许你也可以试试" |
| 普世化 | "所有人都应该慢下来" | "对我而言/如果你也经常感到赶…" |
| 科学化 | "神经科学证明慢下来大脑更活跃" | "我自己的体验是…"（去掉科学证明） |
| 二元残留 | "慢是高级的，快是低级的" | "快和慢是不同的节奏，适合不同的情境" |
| 答案植入 | "如果你回答X，说明你Y" | 纯开放式问题，不附解读/层级判断 |

**最严重的立场错误**（直接扣10分以上）：
- 整个文案的核心卖点就是"慢下来是为了更高效"——本质上还是效率框架，不是真的反效率
- 暗示"买了课你就能做到"——终点化+承诺性

#### B4. 审查记录完整性（5分）

| 分值 | 评分标准 |
|------|---------|
| 5分 | 三轨都有完整的数量记录（命中数/修复数），有典型问题举例，记录了修复方式 |
| 4分 | 有数量记录，但缺少典型问题举例或修复说明 |
| 2分 | 只有部分轨道有记录，或记录很笼统（如"查了，没问题"） |
| 0分 | 没有审查记录，或记录全空白 |

---

### 维度C：最终文案质量（20分）

#### C1. 黑名单零违规（5分）
- 5分：最终文案0个显性违规
- 3分：1-2个轻微违规（如口语化"要"但不是指令语境）
- 0分：3个以上显性违规

#### C2. 概念污染零混入（5分）
- 5分：最终文案0个概念混入（副词/连接词/日常用语除外）
- 3分：1个混入但不影响核心概念
- 0分：2个以上混入

#### C3. 哲学立场一致性（5分）
- 5分：7个维度全部通过，立场前后一致，没有滑回效率逻辑
- 3分：有1处立场问题但不影响核心主张
- 0分：核心主张就是立场错误（如"慢=更高效"）

#### C4. 读者感受（5分）
| 分值 | 评分标准 |
|------|---------|
| 5分 | 读完感觉"被看见"，没有被教育/被否定/被营销的不适感，想点赞或评论 |
| 3分 | 读完不反感，但也没有特别共鸣 |
| 1分 | 读完有不适感（被说教/被营销/觉得在装） |

**测试方法**：把文案发给一个正在996的朋友看，问"你看完什么感觉？"——如果朋友说"你在教我做事？"就是0分；如果朋友说"对啊我也觉得"就是5分。

---

### 维度D：练习记录与反思（10分）

#### D1. 三轨记录表完整性（4分）
- 4分：三轨的命中数/修复数/典型问题都填写完整
- 2分：部分填写
- 0分：表格空白

#### D2. 3个关键发现/卡点质量（3分，每个1分）
| 分值 | 评分标准 |
|------|---------|
| 1分 | 发现是具体的、真实的（如"我发现'要'字很容易无意识地写进去"） |
| 0.5分 | 发现较笼统（如"正言若反挺难的"） |
| 0分 | 空白或敷衍（如"没什么卡点""都挺简单"） |

#### D3. 一句话总结准确度（3分）
| 分值 | 评分标准 |
|------|---------|
| 3分 | 总结精准，点出了真正的陷阱（如"最容易踩的坑是'不是不X'后面接新指令，变成假回护"） |
| 2分 | 总结有道理但不够精准（如"最容易踩的坑是公式用得生硬"） |
| 1分 | 总结很泛（如"最容易踩的坑是写不好"） |
| 0分 | 空白 |

## 四、一票否决项

出现以下任何一种情况，直接判定为❌需重做（<60分），无论其他维度得分如何：

1. **核心立场错误**：整个文案的底层逻辑仍是"慢是为了更快/更高效/更成功"，本质上没有跳出效率框架
2. **对立式终稿**：最终文案仍使用❌vs✅二元对比，没有使用正言若反
3. **严重营销感**：文案读起来像微商/成功学，有"你必须/保证/一定能"等强烈违规
4. **跳过第三轨**：完全没有做立场审查，或立场审查全部填"没问题"但文案有明显立场错误

## 五、自评引导问题

做完练习后，先问自己以下问题，再对照评分标准给自己打分：

1. **关于正言若反**：
   - 我写的"大X"，是真的看见了X的辩证面，还是换了个说法继续否定X？
   - 读者读完我的文案，会觉得我在说"你错了"，还是"我看见了另一种可能"？

2. **关于三轨制**：
   - 第一轨搜"要/得/应该"的时候，有没有惊讶地发现自己无意识写了很多？
   - 第二轨搜"成长/接纳/坚持"的时候，有没有发现自己觉得"这个词挺好的"不想删？
   - 第三轨读自己文案的时候，有没有发现自己"一不小心"就承诺了结果？

3. **关于最终感受**：
   - 如果我是一个正在被996压得喘不过气的人，看完我自己写的文案，第一反应是什么？
   - 我愿意把这条文案发在我真实的朋友圈吗？
