# 引用块嵌套代码块渲染修复模式（Blockquote Code Block Rendering Fix）

> 📖 **深度使用指南**：本文件是 L1 精简入口（规则+模板+检查清单）。完整使用场景、5种变体修复方案、8组正反例对比、渲染器兼容性说明、10条最佳实践、常见误区，见配套 L2 深度文档 [blockquote-code-block-rendering-usage-guide.md](blockquote-code-block-rendering-usage-guide.md)。

## 模式类型
方法论模式 / 文档架构

## 成熟度
L1 实验性（1次案例：AGENTS.md 启动协议渲染修复）

## 适用场景
当 Markdown 文档中出现以下渲染问题时直接套用：
- 引用块（`>`）内嵌套代码块（` ``` `），导致代码块内的 Markdown 链接无法渲染
- 链接显示为原始文本 `[文本](URL)` 而非可点击链接
- 需要在引用块/警告框/提示框内展示结构化步骤（含链接、列表、复选框）
- 代码块内的 Markdown 语法（加粗、列表、链接）全部失效

## 问题本质

Markdown 渲染器遵循严格的嵌套规则：

```
引用块(>) > 代码块(```) → 代码块内一切视为纯文本，Markdown语法不渲染
```

**常见错误模式**：
````markdown
> **🚨 重要协议**
>
> ```
> 步骤1：读取文件
> 步骤2：访问 [链接](url)  ← 这里的链接不会渲染，显示为原始文本
> ```
````

**渲染结果**：`` `步骤2：访问 [链接](url)` `` 中链接显示为纯文本，URL重复可见。

## 修复方案

将代码块替换为引用块内的结构化 Markdown 元素（加粗标题 + 无序列表/有序列表），保留引用块的视觉突出效果，同时让 Markdown 语法正常渲染。

### Before（错误写法）

````markdown
> **🚨 启动协议**
>
> ```
> 步骤 1：读取本文件全文
> 步骤 2：按路由表确定规范
>   步骤 2.1：跨项目嵌套时读取 [vendor/AGENTS.md](vendor/AGENTS.md)
>   □ 自检项目1
>   □ 自检项目2
> 步骤 3：读取规范
> ```
````

### After（正确写法）

````markdown
> **🚨 启动协议**
>
> **步骤 1**：读取本文件全文
>
> **步骤 2**：按路由表确定规范
> - **步骤 2.1**：跨项目嵌套时读取 [vendor/AGENTS.md](vendor/AGENTS.md)
>
> **步骤 3**：读取规范
>
> **自检**：执行前逐项确认：
> - □ 自检项目1
> - □ 自检项目2
````

## 转换规则对照表

| 代码块内元素 | 替换为 | 说明 |
|---|---|---|
| `步骤 N：描述` | `**步骤 N**：描述` | 主步骤用加粗标题，前后空行分隔 |
| `  步骤 N.M：描述`（缩进） | `- **步骤 N.M**：描述` | 子步骤用无序列表 `-` + 加粗步骤号 |
| `  □ 项目`（缩进+复选框） | `- □ 项目` | 自检清单项用无序列表 `-` |
| 代码块内的 `[链接](url)` | 直接写 `[链接](url)` | 在引用块+列表中链接可正常渲染 |
| 代码块内的反引号 `` `代码` `` | 直接写 `` `代码` `` | 行内代码在列表中正常渲染 |
| 步骤间无分隔 | 主步骤间加空行 | 增强视觉层次，避免挤成一团 |

## 快速套用模板

当遇到"引用块内用代码块展示步骤"的场景时，直接套用以下结构：

```markdown
> **🚨 [协议/警告标题]**
>
> **步骤 1**：[步骤1描述]
>
> **步骤 2**：[步骤2描述]
> - **步骤 2.1**：[子步骤描述，可含链接](url)
> - **步骤 2.2**：[子步骤描述]
>
> **步骤 3**：[步骤3描述]
>
> **自检清单**：
> - □ [检查项1]
> - □ [检查项2]
> - □ [检查项3]
>
> ⚠️ **[警告说明文本]**
```

## 为什么代码块内不渲染 Markdown？

| 上下文 | Markdown语法是否渲染 | 原因 |
|---|---|---|
| 普通段落 | ✅ 是 | 标准 Markdown 解析 |
| 引用块(>)内段落 | ✅ 是 | 引用块不影响内联语法 |
| 代码块(```)内 | ❌ 否 | 代码块语义是"原样展示代码/文本" |
| 引用块(>)内代码块(```) | ❌ 否 | 代码块优先级更高，内部视为纯文本 |
| 引用块(>)内列表(-) | ✅ 是 | 列表是块级元素，在引用块内正常工作 |

## 检查清单

修复后逐项验证：

- [ ] 引用块（`>`）视觉效果保留（左侧竖线、缩进）
- [ ] 所有链接可点击且不显示原始URL文本
- [ ] 加粗（`**text**`）正常渲染
- [ ] 列表项（`-`）正常缩进显示
- [ ] 复选框（□）正常显示
- [ ] 主步骤之间有空行分隔，层次清晰
- [ ] 行内代码（`` `path` ``）正常渲染为等宽字体
- [ ] 警告图标（🚨⚠️）正常显示
