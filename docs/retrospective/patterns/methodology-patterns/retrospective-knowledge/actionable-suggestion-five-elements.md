# 可执行建议五要素（Actionable Suggestion Five Elements）

## 模式类型
方法论模式

## 成熟度
L1 实验性（1 次成功案例：断链修复复盘 5 项建议 100% 落地率）

## 适用场景
撰写复盘改进建议、行动项、任务清单、需求文档，或评估他人建议的可执行性。

## 问题背景

很多"建议无法落地"不是执行者的问题，而是建议本身不具备可执行性——只说了"应该做什么"，没说"具体怎么做、做成什么样、怎么验证做对了"。空泛建议导致：
- 执行者需要二次拆解，理解偏差造成返工
- 建议之间边界模糊，多人协作时产生重叠或遗漏
- 无法独立验证完成状态，验收标准模糊

## 核心要素

高落地率建议必须同时包含五个要素：

| 要素 | 空泛反例 | 正面示例 |
|------|---------|---------|
| **具体交付物** | "应该加个自动修复功能" | "新增 `.agents/scripts/generate-dashboard.py`"（精确到文件名） |
| **验收标准** | "让看板更准确" | "扫描 `.trae/specs/*/tasks.md` 聚合状态，自动更新 README.md"（描述清楚做什么） |
| **优先级** | 无优先级或都是高优先级 | 🔴高/🟡中/🟢低 三级分层，按治理层级逻辑排序（参见 `governance-tier-priority.md`） |
| **集成位置** | "加到CI里" | "修改 `.agents/scripts/ci-check.ps1`，在现有检查后增加步骤，断链返回非零退出码"（说明怎么加、加在哪、失败怎么办） |
| **状态追踪** | 建议写完就不管了 | 每项都有 ✅/📋/❌ 状态标记，总表+分表双重追踪（参见 `suggestion-priority-driven-execution.md`） |

**规律公式**：
```
建议落地概率 = f(具体性, 可验证性, 优先级明确度, 集成路径清晰度, 状态可见性)
```

## 关键设计特征：1:1 无冗余映射

建议与交付物应遵循严格的 1:1 映射原则：
- **原子化建议**：每项建议对应一个原子化变更（新增一个文件、修改一个集成点、增强一个工具）
- **无重叠**：建议之间边界清晰，没有模糊地带
- **独立可验证**：每完成一项都可以独立验证，不需要等其他建议完成
- **增量实施**：可以按优先级一项一项来，不必全量铺开

| 改进建议 | 交付物 | 映射类型 |
|---------|-------|---------|
| 看板自动生成 | generate-dashboard.py（新文件） | 1:1 新增工具 |
| CI集成链接检查 | ci-check.ps1（修改现有文件） | 1:1 修改集成点 |
| 原子化一键收尾 | finalize-atomization.py（新文件） | 1:1 新增工具 |

## 检查清单

- [ ] 建议是否精确到具体文件名或明确的集成点？
- [ ] 完成后如何验证是否做对了？验收标准是否可客观判断？
- [ ] 优先级是否明确（🔴/🟡/🟢），且按治理层级逻辑排序？
- [ ] 集成路径是否清晰——在哪加、怎么加、失败怎么办？
- [ ] 状态追踪机制是否建立——完成后谁更新状态、在哪更新？
- [ ] 建议之间是否 1:1 映射到交付物，边界清晰无重叠？

## 反例警示

| 问题 | 后果 |
|------|------|
| 建议只写"优化XX"没有交付物 | 执行者不知道要产出什么，理解偏差返工 |
| 没有验收标准 | 完成标准模糊，"做完了"各有各的理解 |
| 全是高优先级 | 等于没有优先级，资源无法有效分配 |
| 不说明集成位置 | 执行者需要自行判断加在哪里，容易加错位置 |
| 多个建议混在一条里 | 难以独立验证，一个问题阻塞整组建议 |

## 与现有模式的关系

- `three-part-retrospective.md`：本模式是三段式复盘"行动层"的撰写标准
- `suggestion-priority-driven-execution.md`：本模式定义建议的静态结构要素，该模式定义执行时的动态决策流程（投入估算+状态流转）
- `governance-tier-priority.md`：本模式的优先级判断依据——按治理层级而非难度排序
- `report-as-tracking.md`：状态追踪的载体是报告本身

## 成功案例

在断链修复复盘中，5 项改进建议全部落地（100%落地率）：
- A1/A2（🔴高）：防复发防线，当日完成
- B1/B2（🟡中）：流程内自动化，当日完成
- C1（🟢低）：能力拓展，当日完成
