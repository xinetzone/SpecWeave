# 元洞察 2：优先级分层的隐藏逻辑 — 按治理层级排序

> ✅ 本洞察已归档为全局方法论模式：[governance-tier-priority](../../../../../patterns/methodology-patterns/governance-strategy/governance-tier-priority.md)（L2）

优先级排序不是按难度，而是遵循治理能力的演进逻辑：🔴高优先级（防止问题再发：根除+防线）→ 🟡中优先级（提升操作效率：流程自动化+事前预防）→ 🟢低优先级（拓展能力边界）。与suggestion-priority-driven-execution（战术层执行决策）互补。

详见全局模式文档中的三级分层详解、判断口诀和反例警示。
