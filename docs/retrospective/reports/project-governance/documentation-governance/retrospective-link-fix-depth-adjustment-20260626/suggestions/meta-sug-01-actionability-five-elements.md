# 元洞察 1：建议文档的"可执行性"五要素

> ✅ 本洞察已归档为全局方法论模式：[actionable-suggestion-five-elements](../../../../../patterns/methodology-patterns/retrospective-knowledge/actionable-suggestion-five-elements.md)（L2）

对比"空泛建议"和"可落地建议"，高落地率建议必须包含五个关键要素：具体交付物、验收标准、优先级、集成位置、状态追踪。建议-交付物遵循1:1无冗余映射原则，每项建议对应一个独立可验证的变更单元。

详见全局模式文档中的五要素表格、检查清单和反例警示。
