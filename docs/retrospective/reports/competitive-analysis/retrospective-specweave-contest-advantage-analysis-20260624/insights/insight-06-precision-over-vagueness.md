# 洞察6：扶持资源的申请制——话题格式精确到空格，主动申请才能获得 ⭐⭐⭐⭐

**现象**：抖音流量扶持有两个容易踩坑的门槛：(1)形式门槛——话题格式极其精确，`#vibecoding大赏`（非`#vibe coding大赏`）和`#TRAEAI创造力大赛`（非`#TRAE AI创造力大赛`），多一个空格即无效；必须@TRAE和@抖音科技；(2)流程门槛——发布抖音≠获得流量，需填写飞书表单主动申请，2个工作日人工审核。

**规律**：大型赛事中官方提供的"扶持资源"几乎都采用申请制而非自动匹配制。设计目的有二：筛选真正认真参与的选手（愿意花时间填表）、给运营团队质量审核缓冲期。

**操作指南**：
- 话题格式按官方精确格式执行（零容忍）
- 发布后立即（30分钟内）填写飞书表单
- 将"填表"作为发布流程固定步骤列入checklist

> **一句话**：赛道大奖靠实力，抖音流量靠精确——一个空格之差可能损失50,000曝光。
