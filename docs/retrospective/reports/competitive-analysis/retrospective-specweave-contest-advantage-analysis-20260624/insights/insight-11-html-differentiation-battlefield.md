# 洞察11：标准化Prompt的隐性代价——HTML文件成为唯一差异战场 ⭐⭐⭐

**现象**：学习资料提供3套可直接复制粘贴的Prompt模板，TRAE Work从创意Doc自动生成HTML。这导致大量参赛者的报名帖结构和HTML风格高度同质化。

**规律**：当官方提供标准化"最佳实践"后，遵循最佳实践的作品会回归均值——而在均值区间的竞争中，微弱优势会被统计噪声淹没。唯一突围方式是在关键材料上脱离标准化路径。报名帖文字因同Prompt→同结构而同质化程度极高，但HTML生成方式仍存在差异空间——手动构建交互式HTML与TRAE Work自动生成的页面在视觉认知上形成跨品类结构断裂。

三层差异化空间：内容差异（最低层，边际收益低）→结构差异（中间层，评审在断裂处停留）→品类差异（最高层，完全脱离标准化路径）。

**待入库模式**：本洞察已被系统化为 [template-homogenization-escape.md](../../../../patterns/methodology-patterns/product-growth/template-homogenization-escape.md)（待正式入库）。

> **一句话**：当所有人的帖子都长得像，HTML文件就是唯一的差异战场。
