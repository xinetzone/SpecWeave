# 洞察5：报名帖是第一个评审窗口——审核只看合规，评审看内容 ⭐⭐⭐

**现象**：报名审核只看"合规"（内容完整性+表达清晰性+原创合规），不评创意好坏。但通过审核的报名帖会被初赛评审作为了解作品的第一份材料。

**规律**：官方指南要求的模板（4部分：创意名称+介绍/目标用户及痛点/价值与意义/HTML产物）恰好构成迷你pitch结构。合规填满是底线，策略化撰写是上限。大多数参赛者把报名帖当"报名表"填，少数人把它当"首印象窗口"写——这是第一个差异化优势。

**操作指南**：报名帖在合规基础上写成初赛评审的第一份pitch deck，而非仅仅凑够字数。

> **一句话**：报名帖不是报名表——它是评委对你作品的第一印象。
