**关键信号**：Community Live #13 产品经理对 TRAE Rules 特性的专门讲解，为 SpecWeave 补上了最关键的叙事环节——AGENTS.md 体系不是在 TRAE 之外另建规则，而是在 TRAE **原生支持的 Rules 特性**之上的系统化延伸。评审看到的不是一个"外部导入的方法论"，而是一个"从 TRAE 产品能力中生长出的方法论"。

**关键信号 2**：Community Live 产品介绍场（6/17）首次以正式品牌语言定义了 TRAE 的两条产品线——**TRAE IDE**（「你的专属 AI 开发工程师」）面向需要精细掌控代码的开发者，**TRAE Work**（「全新上线的智能工作助手」）面向所有人的日常办公。SpecWeave 的 AGENTS.md 方法论体系恰好位于 IDE 产品线的**最高使用深度**——它不是 Work 场景下的轻量提效，而是在 IDE 中构建 AI 协作的工程化体系。这一产品线定位为 SpecWeave 提供了「从产品能力中生长」的叙事锚点：TRAE IDE 提供了 Rules 特性，而 SpecWeave 将其系统化为四层架构。
