## 项目概览

### 核心指标

| 指标 | 数值 |
|------|------|
| 项目名称 | Ian Xiaohei Illustrations（小嘿插图） |
| 作者 | Ian（ByteDance 产品设计师） |
| 开源协议 | MIT |
| Star 数 | 5300+（23 天） |
| 核心定位 | 为中文文章中的「认知锚点」生成配图 |
| 输出格式 | 16:9 白底手绘线稿 PNG |
| 使用模式 | 3 种（配图规划 / 直接生成 / 单图生成） |
| 核心 IP | 小黑（Xiaohei）角色系统 |

**关键发现**：该项目区别于普通 AI 绘图工具的根本差异在于——不是"给文章配一张好看的图"，而是"让 AI 把文章中一个重要的认知动作画出来"。它将产品设计师的设计思维、审美品位做成了一套可复制的 AI Skill 系统。
