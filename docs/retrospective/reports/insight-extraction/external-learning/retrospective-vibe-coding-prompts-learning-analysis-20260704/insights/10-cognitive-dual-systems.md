# 洞察10:认知双系统——类比是系统1,第一性原理是系统2

- **分类**:第一性原理类
- **成熟度**:L3 可复用(validation_count=3，含本项目反面案例验证)
- **可复用性**:高 - 适用于所有复杂决策场景的认知偏差对抗

## 洞察内容

从任务合并委派决策、链接格式类比错误两个经验中，可以提炼出**认知双系统理论**这一底层第一性原理：**人类认知有双系统——系统1（快思考/类比/直觉）高效但容易出错，系统2（慢思考/第一性原理/追溯本质）准确但消耗认知资源。第一性原理思维不是"更聪明的类比"，而是主动打断系统1的自动推理，强制激活系统2去追问"这个事物的本质要求是什么"**。

最生动的证明就是：我们刚刚学习了"第一性原理要打断类比推理"，却立刻就在链接格式上犯了类比推理错误——知道和做到之间的鸿沟，正是系统1的惯性。

## 5-Whys根因追问链

| Why层级 | 问题 | 答案 |
|---------|------|------|
| Why1 | 为什么会犯类比推理错误（file:///格式误用）？ | 因为直觉上觉得"都是链接，格式应该一样" |
| Why2 | 为什么会有这种直觉？ | 因为大脑倾向于用类比快速决策（系统1） |
| Why3 | 为什么类比决策会出错？ | 因为类比只看到表面相似性，忽略了本质差异 |
| Why4 | 本质差异是什么？ | AI对话链接是单环境跳转，Markdown链接是多环境移植 |
| Why5 | **底层本质** | **认知双系统——系统1类比高效但易出错，系统2本质推导准确但耗能，需主动切换** |

## 证据支撑

- **经验E2**:任务合并委派——打破"拆分越细越好"的类比直觉，追溯上下文传递成本本质
- **经验E7**:链接格式错误——混淆AI对话输出格式与Markdown文档格式的典型类比错误
- **L3验证**:本项目学完第一性原理仅1小时即犯类比错误，成为最生动的反面教材
- **跨领域验证**:卡尼曼《思考，快与慢》系统1/2理论、芒格误判心理学

## 类比暂停法（可操作化）

当你发现自己在说"就像XX一样"、"XX也是这样做的"时，立刻暂停！这是系统1在工作。强制追问：
1. 当前场景的本质要求是什么？
2. 那个类比的前提条件在这里成立吗？
3. 有没有反例？

给自己30秒时间做"第一性原理检查"。
