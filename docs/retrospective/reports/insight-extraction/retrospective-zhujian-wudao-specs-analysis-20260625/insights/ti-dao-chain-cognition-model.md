# 洞察 G：体道链可泛化为通用认知深度模型

**发现**：竹简悟道原创的"体道链"（名→反→有无→正言若反→无为→玄同）虽然源于《老子》哲学，但其六层结构可泛化为一个通用的认知深度标尺，适用于衡量用户对任何复杂概念的理解程度。

## 体道链六层泛化

| 层级 | 竹简悟道原义 | 泛化认知定义 | 产品设计含义 |
|------|------------|------------|------------|
| L1 名 | 能说出概念名词 | 知道术语，能复述定义 | 用户文档/入门教程 |
| L2 反 | 理解对立面的辩证法 | 知道反例、边界条件、适用范围 | FAQ/误区澄清 |
| L3 有无 | 理解显隐相生 | 看见隐含假设、未言明的前提 | 深度案例/场景分析 |
| L4 正言若反 | 理解反直觉的真相 | 接受与常识相反的深层规律 | 专家视角/进阶内容 |
| L5 无为 | 在实践中不强求 | 内化为本能，不需要刻意思考 | 个性化推荐/智能辅助 |
| L6 玄同 | 超越对立看到全局 | 能同时看见矛盾双方，做出精准回应 | 社区/共创/高阶用户 |

## 用户分层产品策略

- L1-L2用户：需要明确的指引和答案（解惑模式）
- L3-L4用户：需要提问和反思空间（问道模式）
- L5-L6用户：不需要产品干预，产品是背景（自在模式）

这也正是洞察63（定位解缚）提出的"解惑→问道"过渡模式的理论基础——不同认知深度的用户需要不同的产品形态。

## 可迁移性

**中高**。可应用于教育产品、知识工具、专业软件的用户分层设计。
