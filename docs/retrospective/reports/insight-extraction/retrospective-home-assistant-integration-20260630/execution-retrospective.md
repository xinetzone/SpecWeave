## 阶段 2：API 脚本开发

### 2.1 执行内容

创建 `ha_api.py` 脚本，支持 REST API 调用。

### 2.2 关键步骤

1. 创建 Python 脚本 `.agents/scripts/ha_api.py`
2. 实现核心功能：
   - 连接状态检查（info）
   - 实体列表获取（list）
   - 实体状态查询（get）
   - 实体状态设置（set）
   - 服务调用（service）
3. 优化代码：
   - 使用 dataclass 定义数据结构（HAConfig、EntityState、HARequest、HAResponse）
   - 使用 pathlib.Path 替代 os.path
   - 实现优雅降级机制
4. 添加错误处理和安全机制

### 2.3 执行结果

- ✅ 脚本可独立运行，支持 `--help` 参数
- ✅ 状态查询功能返回正确的 JSON 数据
- ✅ 设备控制功能正确发送 POST 请求
- ✅ 错误处理机制正常工作
- ✅ 无 HA 连接时脚本优雅降级，不抛出致命错误
- ✅ 使用 dataclass 和 pathlib 优化
