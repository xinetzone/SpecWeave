# SpecWeave 项目全面萃取报告

> **项目名称**：SpecWeave 知识资产全面萃取
> **萃取日期**：2026-06-26

> **报告类型**：全面萃取与资产盘点

## 项目概览

### 1.1 萃取背景

SpecWeave 项目经过持续迭代，已积累了丰富的知识资产，包括可复用模式、决策框架、检查脚本、复盘报告和 Spec 规格文档。本次全面萃取旨在系统性盘点全部知识资产，评估模式成熟度，修复统计缺陷，升级符合条件的模式，并形成全景知识地图。

### 1.2 萃取目标

1. 全面扫描与盘点项目全部知识资产
2. 修复模式扫描脚本的递归统计缺陷
3. 评估并升级符合条件的模式成熟度
4. 验证模式库索引一致性
5. 形成全景知识地图与萃取报告
