## 八、关键指标含义解释

为帮助读者准确理解上述数据的意义，以下对核心评测指标逐一说明：

### 1. 编辑距离（Edit Distance / Levenshtein Distance）
- **含义**：衡量两个字符串之间差异程度的指标，指将一个字符串转换成另一个字符串所需的最少单字符编辑操作（插入、删除、替换）次数。
- **在 OCR 中的意义**：编辑距离越小，说明模型转录的文本与原文逐字比对的差异越小，识别准确率越高。
- **数值解读**：0 表示完全一致，数值越大错误越多。Unlimited OCR 将文本编辑距离从 0.073 降至 0.038，意味着字符级错误减少了近一半。

### 2. CDM（Character Detection Metric，公式识别指标）
- **含义**：专门用于评估数学公式识别质量的指标，综合考量公式结构、符号、上下标、分式、根式等数学元素的还原准确度。
- **在 OCR 中的意义**：公式是文档解析中最难的部分之一，CDM 分数越高，说明模型对复杂数学公式的识别和结构还原能力越强。
- **数值解读**：满分为 100，Unlimited OCR 达到 92.61，已接近人工识别水平。

### 3. TEDS（Tree Edit Distance-based Similarity，表格结构识别指标）
- **含义**：基于树编辑距离的相似度指标，将表格解析为树结构后，计算预测结果与真实标注之间的树编辑距离相似度。
- **在 OCR 中的意义**：不仅评估单元格文字识别是否正确，还评估表格的行列结构、合并单元格、层级关系等结构化信息是否还原准确。
- **数值解读**：满分为 100，分数越高说明表格结构化还原越精准。90.93 意味着复杂表格的结构还原已达到很高水平。

### 4. TPS（Tokens Per Second，每秒生成 token 数）
- **含义**：推理速度指标，衡量模型每秒能够生成的输出 token 数量。
- **在 OCR 中的意义**：直接决定文档解析的实际等待时间。TPS 越高，用户等待结果的时间越短，批量处理效率越高。
- **数值解读**：在长输出场景下（如 6144 token），TPS 的稳定性至关重要——Unlimited OCR 凭借 R-SWA 实现恒定速度，而标准注意力模型会随输出增长逐渐减速。

### 5. Distinct-n（去重 n-gram 比例）
- **含义**：衡量生成文本多样性的指标，计算所有 n-gram（连续 n 个 token 的组合）中不重复 n-gram 所占的比例。
- **在 OCR 中的意义**：Distinct-35 表示 35-gram 层面的去重比例，用于检测模型是否出现"复读"现象（即重复生成相同片段）。长文档解析中，低 Distinct-n 意味着模型进入重复循环，出现"失忆式复读"。
- **数值解读**：Distinct-35 达到 97% 意味着即使在 40+ 页超长文档中，模型输出几乎没有重复，证明 R-SWA 有效解决了长程"失忆"问题。
