- [x] 线路类型：BGP（多线）为推荐线路
- [x] 带宽规格：按量计费-按流量1-200Mbps，按量计费-按带宽/包年包月1-500Mbps，规格变更可达1000Mbps
- [x] 出入带宽规则：带宽≤10Mbps时入方向默认10Mbps，>10Mbps时与设置值一致
