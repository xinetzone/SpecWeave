# 向日葵智能远控鼠标MM110/BM110产品学习与深度洞察分析报告 - 产品需求文档

## Overview
- **Summary**: 系统性学习向日葵官方两款智能远控鼠标硬件产品（MM110扁平便携版、BM110人体工学版）的产品页面，全面梳理两款产品的技术规格、功能特性、应用场景、产品优势及市场定位，通过对比分析提炼两者的异同点，形成一份结构化的学习总结与深度洞察分析报告。
- **Purpose**: 为项目团队提供向日葵智能远控鼠标产品线的完整学习资料，系统了解向日葵在远程控制外设领域的产品布局、差异化策略、技术实现和商业逻辑，为移动办公、远程控制、智能硬件领域的产品设计与技术选型提供参考借鉴。
- **Target Users**: 移动办公人群、远程工作者、IT运维人员、智能硬件产品经理、AI Agent开发者、对远程控制外设感兴趣的技术人员、数码产品爱好者。

## Background & Context
- 向日葵是贝锐科技（Oray）旗下的国民级远程控制品牌，产品线覆盖软件远控与智能硬件
- 官方MM110产品页面：https://sunlogin.oray.com/hardware/mm110
- 官方BM110产品页面：https://sunlogin.oray.com/hardware/mouse-bm110
- 向日葵智能远控鼠标是配合向日葵远程控制软件/App使用的专用外设，解决移动设备（平板/手机）远程控制电脑时的操作效率问题
- MM110定位扁平便携，适合移动办公随身携带场景
- BM110定位人体工学舒适握感，支持双设备连接，适合长期办公场景
- 两款产品均支持蓝牙5.0连接，适配Android/iOS/Windows/macOS主流系统
- 产品通过向日葵控制端App实现指针模式下的完整鼠标操作远控
