## 📋 所需材料清单（个人备案）

### 必备材料
| 材料 | 要求 | 备注 |
|------|------|------|
| 身份证 | 有效期内的二代身份证原件 | 需要正反面照片+手持身份证照片 |
| 手机号 | 本人实名手机号 | 需要接收验证码 |
| 邮箱 | 常用邮箱 | 接收备案通知 |
| 域名 | 已实名认证的域名 | 域名所有者必须与备案主体一致 |
| 云服务器 | 中国大陆节点服务器 | 必须在备案接入商处购买，有公网IP |

### 拍照要求（关键！）
- **背景**：纯白色/浅色背景墙，无杂物
- **着装**：穿深色有领上衣，不能戴帽子、墨镜、口罩
- **姿势**：正面免冠，面部清晰可见，双手自然持身份证于胸前
- **身份证**：身份证正反面信息清晰可辨，无反光、无遮挡
- **人像**：必须是本人，不能用照片翻拍
- ⚠️ 大多数云服务商支持APP在线核验（人脸识别），比拍照更方便

### 网站信息准备
| 信息项 | 填写建议 | 注意事项 |
|--------|---------|---------|
| 网站名称 | 个人网站：如「XX的个人博客」「XXAI工具集」 | 不能含「中国/中华/全国」等字样，不能涉企业/行业/政府类词汇 |
| 网站服务内容 | 选「其他」或「个人空间/博客」 | 个人备案不能选「电子商务/付费服务」等经营性内容 |
| 前置审批 | 选「以上都不是」 | 普通网站无需前置审批 |
| 网站备注 | 简要描述网站用途，如「个人技术分享网站，提供AI工具体验」 | 不要提付费/交易/商业，个人备案是非经营性的 |

## 🚀 详细操作步骤（以阿里云为例）

> 推荐阿里云，备案流程最成熟，有APP端便捷核验。腾讯云/华为云流程类似。

### 步骤1：注册账号并实名认证（10分钟）
1. 打开 https://www.aliyun.com 注册阿里云账号
2. 完成账号实名认证（支付宝扫码即可完成个人实名认证）
3. 确保账号实名信息与后续备案主体一致

### 步骤2：购买域名（10分钟）
1. 进入 https://wanwang.aliyun.com/domain 域名注册页面
2. 搜索你想要的域名（推荐选 `.cn` 或 `.com`，备案流程相同）
3. 加入清单，完成购买（.cn域名首年通常9-29元，.com约35-55元）
4. ⚠️ **立即完成域名实名认证**：购买后进入域名控制台，提交实名认证（上传身份证），域名实名认证通常1-2个工作日通过
5. 域名实名认证通过后才能备案

### 步骤3：购买云服务器（10分钟）
1. 进入 https://www.aliyun.com/product/swas 轻量应用服务器（新手推荐）
2. 选择一个便宜的套餐（如2核2G，约60-100元/3个月，备案需要买≥3个月）
3. **地域选择**：选中国大陆节点（如华东1/华北2等），中国香港/海外节点无法备案
4. 镜像选「CentOS」或「Ubuntu」均可
5. 购买开通后，在服务器详情页可以看到「备案」入口
6. 确保服务器有公网IP

### 步骤4：进入备案系统（5分钟）
1. 方式一：服务器详情页点击「备案」按钮
2. 方式二：直接访问 https://beian.aliyun.com
3. 点击「开始备案」或「新增备案」

### 步骤5：填写主体信息（15分钟）
1. **备案主体类型**：选「个人」
2. **个人信息**：
   - 姓名：与身份证一致
   - 身份证号：准确填写
   - 证件地址：按身份证上地址填写
   - 手机号：本人实名手机号
   - 邮箱：常用邮箱
3. **通讯地址**：填写当前实际居住地址（精确到门牌号）
4. 上传身份证正反面照片

### 步骤6：填写网站信息（15分钟）
1. **网站名称**：按个人网站规范填写（见上表建议）
2. **域名**：添加你购买的域名（会自动验证是否已实名）
3. **网站服务内容**：选「其他」或个人类选项
4. **前置审批**：选无
5. **网站备注**：简要描述（见上表）
6. 上传网站负责人真实性核验照片（可通过阿里云APP人脸识别完成，比上传照片更快捷）

### 步骤7：提交初审（1-2个工作日）
1. 核对所有信息，确认无误后提交
2. 等待阿里云初审（通常1个工作日内）
3. 可能会接到阿里云的核验电话（来自杭州/北京的座机），如实回答即可
4. 如果初审被驳回，按驳回原因修改后重新提交（常见原因：网站名称不合规、照片不清晰等）

### 步骤8：短信核验（1天内）
1. 初审通过后，你的手机号会收到工信部的短信验证码
2. 在24小时内访问 https://beian.miit.gov.cn 完成短信核验
3. 短信核验通过后才会进入管局审核

### 步骤9：等待管局审核（5-20个工作日）
1. 短信核验完成后，备案材料提交到所在省通信管理局
2. **这一步无法加急**，只能等待
3. 可以在阿里云备案控制台查看进度
4. 部分省份可能会有电话核验，保持手机畅通

### 步骤10：备案成功（1-2天）
1. 收到短信和邮件通知：「您的ICP备案申请已通过审核」
2. 获得备案号（格式：京ICP备XXXXXXXX号）
3. 备案成功！

### 步骤11：备案后配置（30分钟）
1. **域名解析**：在域名控制台添加A记录，解析到你的服务器IP
2. **配置HTTPS**：
   - 在阿里云SSL证书页面申请免费证书（https://www.aliyun.com/product/cas）
   - 免费DV证书，验证域名所有权后签发（通常几分钟到几小时）
   - 下载证书，部署到Nginx/Apache
3. **网站底部悬挂备案号**：
   - 按照管局要求，网站底部必须显示备案号并链接到 https://beian.miit.gov.cn
   - 例如：`京ICP备XXXXXXXX号`
4. **配置支付宝回调地址**：
   - 在支付宝开放平台应用详情页，配置异步通知地址（notify_url）为你的HTTPS域名地址
   - 配置授权回调地址

## ⚠️ 常见问题与避坑

### Q1：备案期间网站能访问吗？
A：不能。备案期间域名不能指向中国大陆服务器，否则备案会被驳回。建议先用IP地址开发测试，备案通过后再切换域名。

### Q2：个人备案可以做付费网站吗？
A：ICP备案分为「非经营性备案」和「经营性备案」。个人能做的是非经营性备案。
- ⚠️ 严格来说，付费功能可能涉及经营性，但实际中很多个人网站都有小额付费功能
- 网站名称和备注不要写「付费/交易/商城/电商」等经营性词汇
- 建议写「个人技术分享/AI工具体验」等描述
- 支付宝AI网页应用收款审核时主要看网站是否有备案、是否合法合规，不要求经营性备案

### Q3：备案要花钱吗？
A：ICP备案本身**完全免费**（不收取任何官方费用）。你只需要花钱买域名和服务器。

### Q4：哪个省份备案最快？
A：一般浙江、上海、广东等东部省份较快（5-10天），部分省份较慢（15-20天）。与你身份证上的省份或居住地省份一致。

### Q5：可以用别人的已备案域名吗？
A：
- 方案A：让域名所有者添加你为备案网站负责人（需要对方配合）
- 方案B：将域名过户到你名下，然后做备案接入/新增备案
- ⚠️ 不建议直接解析到未备案的域名，会被拦截

### Q6：备案被驳回最常见的原因？
1. 网站名称不合规（含企业/行业/中国等字样）
2. 身份证/核验照片不清晰或不符合要求
3. 域名未完成实名认证
4. 网站内容描述涉及经营性内容
5. 电话核验未接或回答不一致

### Q7：我不想备案，有什么其他办法？
A：
1. **选AI移动应用收款**：不需要ICP备案，但需要App包名（做一个简单的WebView App也可以）
2. **用海外服务器**：但支付宝网页支付可能要求回调地址在国内备案域名，⚠️待确认是否可行
3. **用朋友的已备案域名**：需要对方配合添加子域名和备案
4. **用支付宝小程序**：⚠️待确认是否在激励覆盖范围内
