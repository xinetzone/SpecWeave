# 适用场景、优势与局限性

## 适用人群

| 人群 | 核心价值 |
|---|---|
| 个人开发者 | 白嫖整合免费额度 |
| 小团队 | 共享AI订阅（配合Quota-Share） |
| 隐私敏感用户/企业 | 本地运行，数据不出门 |
| 多模型切换者 | 一个端点灵活切换237个提供商 |

## 核心优势

- **免费额度聚合**：每月16亿token免费额度
- **Combo自动故障转移**：毫秒级切换，服务基本不中断
- **本地隐私**：AES-256-GCM加密，无遥测数据收集
- **OpenAI统一接口**：现有工具无缝接入
- **MCP/A2A生态支持**：跟进最新开放标准
- **24+工具一键配置**：主流AI开发工具开箱即用
- **多端部署**：支持npm/Docker/PWA/Electron多种部署方式

## 潜在局限

- **文档分散**：新手入门有一定门槛
- **Node版本要求较新**：需要>=22版本
- **免费额度不确定性**：免费额度可能随时变化或被平台限制
- **学习曲线**：项目功能多，需要时间熟悉全部功能
