# Checklist

## 项目骨架
- [ ] `prompt_extraction/` 目录结构完整，所有 `__init__.py` 文件存在
- [ ] `requirements.txt` 包含所有必要依赖
- [ ] `config.py` 包含评分阈值、权重、输出目录等可配置项
- [ ] `models.py` 定义 PromptRecord、FeatureSet、QualityScore、OptimizationResult 数据类

## 输入模块
- [ ] CSV 文件解析正确，支持自动字段映射
- [ ] JSON 文件解析正确（对象数组格式）
- [ ] TXT 文件按行解析正确
- [ ] Markdown 文件按标题层级拆分区块正确
- [ ] 单条手动输入可正常接收
- [ ] 空输入返回明确错误提示
- [ ] 格式错误文件返回友好错误信息

## 预处理模块
- [ ] 多余空白正确规范化为单个空格
- [ ] 首尾空白去除
- [ ] Markdown/HTML 格式标记正确去除或转换
- [ ] Markdown 语义结构（标题层级、列表、代码块）在清洗后保留
- [ ] 全角字符正确转换为半角
- [ ] 标点符号规范化
- [ ] URL/email 被识别并标记为附属信息

## 特征提取模块
- [ ] 核心指令（"请"、"要求"、"需要"等）被正确识别和分类
- [ ] 约束条件（"必须"、"不能"、"限制"等）被正确提取
- [ ] 预期输出格式（"返回"、"输出"、"格式"等）被正确识别
- [ ] 无明确特征的提示词返回空特征集
- [ ] Markdown 结构解析：标题→指令分区、列表项→约束/步骤、代码块→输出示例

## 质量评估模块
- [ ] 清晰度评分在 0-100 范围内
- [ ] 完整性评分基于指令/约束/上下文/示例/输出格式要素
- [ ] 可执行性评分基于操作性和可验证性
- [ ] 综合评分按 30%/40%/30% 加权计算
- [ ] 等级标注（优/良/中/差）正确

## 优化模块
- [ ] 综合评分 < 60 分时自动触发优化
- [ ] 缺失输出格式时补充合理格式
- [ ] 歧义表述被改写为精确表述
- [ ] 结构混乱的提示词重组为标准结构
- [ ] 优化前后对比 diff 正确展示

## 流水线编排
- [ ] 单条提示词按 输入→预处理→特征提取→质量评估→优化 顺序执行
- [ ] 批量处理所有条目正确流转
- [ ] 模块间通过 PromptRecord 数据结构传递
- [ ] 单条失败不影响后续处理
- [ ] 错误日志记录完整

## 可视化界面
- [ ] 首页展示输入入口和处理统计
- [ ] 批量上传文件功能正常
- [ ] 单条输入文本框可用

- [ ] 结果列表支持按评分排序和筛选
- [ ] CSV 导出功能正常（含原始、优化后、评分、特征）
- [ ] 纯优化提示词导出功能正常
