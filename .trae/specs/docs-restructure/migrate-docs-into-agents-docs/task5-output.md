# Task 5：.agents/docs 迁移归类抽查记录

## 说明

- 本记录按 `spec.md` 中“抽查样本预设”“抽查规则”“抽查记录模板”输出。
- 本轮抽查共选取 5 个样本，完整覆盖 spec 预设的 5 类样本类型。
- 本轮包含 1 个混合型样本，用于验证“主要读者 + 主要触发方式”的升级判断规则。
- 本记录只新增任务产出文件，不回填 `tasks.md`。

## 样本选择口径

- 优先选择已完成迁移并能直接复核的真实路径，不使用虚构占位样本。
- `默认归属` 先按来源继承规则判断，再结合主要读者、主要用途、发现方式交叉验证。
- `实际用途` 以样本正文定位、目录声明以及仓库内现有引用关系为依据。
- 若样本可能同时服务人工阅读与 agent 路由，则标记为混合型，并检查是否已有显式边界说明。
