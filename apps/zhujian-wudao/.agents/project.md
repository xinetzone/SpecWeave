# .agents/project.md — 项目概述与核心概念词典

## 项目简述

竹简悟道是一款以帛书《老子》（1973年马王堆汉墓出土）为哲学根基的 AI 反思引导工具。产品不做知识翻译者、不做建议者，只做提问者——通过对话帮助用户自行识破概念陷阱（概念解缚），而非提供答案。

- **产品形态**：多端（Web+平板+手机）AI 对话应用
- **视觉风格**：古典竹简卷轴风格，追求"能让人静下来"
- **对话模型**：Open Questioning（区别于苏格拉底法），只提问题、不引导方向
- **当前阶段**：设计文档与洞察库建设期，HTML 单文件原型验证期

## 核心概念词典

### 概念解缚
产品最核心的差异化定位（洞察 1）。人最大的桎梏不是无知，而是被自己创造的概念困住了——"我必须成功""辞职就是失败""应该比别人强"都不是真相，只是概念。竹简悟道帮用户在对话中自行识破这些概念陷阱。

### 体道四法
四条通往概念解缚的实践路径（spec §4.1，洞察 49/51/42+48/52 系统化展开）：

| 法门 | 标签 | 核心动作 | 帛书根源 |
|------|------|---------|---------|
| 虚静内观 | 收 | 收敛外求→澄明内心→体证本然 | 第 16 章"致虚极，守静笃" |
| 自然无为 | 守 | 识别强求→放下控制→让事物自行演化 | 第 25 章"道法自然" |
| 柔弱不争 | 柔 | 辨认争心→守柔处下→以退为进 | 第 36 章"柔弱胜刚强" |
| 生活实践 | 行 | 承认知道→落实行动→行无行 | 第 41 章"上士闻道，勤而行之" |

### 体道链
衡量用户理解深度的六层标尺（spec §4.2，洞察 15）：
`名(L1) → 反(L2) → 有无(L3) → 正言若反(L4) → 无为(L5) → 玄同(L6)`

### 三层递进
每个法门的三层深度分布（spec §4.3，洞察 37/49/51/52）：
- **第一层**：入门 — 识别问题，收敛外求
- **第二层**：深入 — 澄明内心，以柔承刚
- **第三层**：本然 — 体证本然，行无行

### 玄同
产品核心承诺（洞察 8/35/36）。超越亲疏、利害、贵贱的对立，回到事情的本然状态。不是"怎么都行"，而是"看清全局之后的精准回应"。

### 恒德
帛书独有的概念（洞察 33/34）。"恒德"强调不因外在条件变化而动摇的内在稳定性，区别于通行版的"德"。

### 微明
帛书第 36 章"将欲微明"（洞察 25）。让模糊逐渐清晰，不做预设、允许事情在过程中自行明朗化——是产品的核心迭代哲学。

### Open Questioning
产品对话方法论（洞察 22）。区别于传统苏格拉底法——问题不是引导到预设答案的手段，而是打开思考空间的本身。

### 四句产品宣言
（洞察 6）
1. 概念解缚 — 不被自己脑中的概念支配
2. 回到原典 — 一切思考都从帛书《老子》出发
3. 只问不答 — 不做建议者，只做提问者
4. 玄同自在 — 超越对立，回到本然

### 四法×九景矩阵
体道四法 × 场景九景的交叉推荐逻辑（洞察 26），为 AI 回复提供更多可能方向的基础框架。矩阵默认对齐 Spec 心境标签（决策前/迷茫时/冲突中/变革期/日常修养/团队管理/个人成长/行无行/且问且行），HTML 已实现双轨并存——用户可自由选用场景困局标签或心境状态标签，后台推荐引擎统一使用 Spec 标签做交叉匹配。

### 场景标签（合并为一行 · Spec 主导 + HTML 辅助）

```
[且问且行]  ← 独立全宽入口
─────────────────────────────────────────────
[决策前]  [迷茫时]  [冲突中]  [变革期]  [日常修养] [团队管理] [个人成长]
 重大决策   方向探索   人际关系   职业困惑   创作瓶颈   管理困境   编程困境
─────────────────────────────────────────────
[行无行]  ← 知行合一入口（R16新增）
 知行合一
```

每枚标签以 Spec §2.4 心境状态为主名、HTML 场景处境为辅助小字。`data-scenario` 统一使用 Spec 键名（`deciding`/`lost`/`conflict`/`changing`/`cultivate`/`team`/`growing`/`action`），AI 回复按对应 `aiResponses[key]` 匹配。原 HTML 九景数据池保留，通过未选标签的自由输入随机触达。

### 每日一问
产品核心习惯养成机制（洞察 53）。以帛书第 48 章"为道者日损"为哲学根基，通过每日一个问题帮助用户建立定期反思习惯。当前实现 8 题（date % 8 轮转），体道四法各覆盖：虚静内观 3 题、柔弱不争 2 题、自然无为 2 题、生活实践 1 题。
