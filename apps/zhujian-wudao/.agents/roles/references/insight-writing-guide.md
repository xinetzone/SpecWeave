## 核心概念词典

### 概念解缚

产品最核心的差异化定位（洞察 1）。人最大的桎梏不是无知，而是被自己创造的概念困住了——"我必须成功""辞职就是失败""应该比别人强"都不是真相，只是概念。竹简悟道帮用户在对话中自行识破这些概念陷阱。

### 体道四法

四条通往概念解缚的实践路径（spec §4.1，洞察 49/51/42+48/52 系统化展开）：

| 法门 | 标签 | 核心动作 | 帛书根源 |
|------|------|---------|---------|
| 虚静内观 | 收 | 收敛外求→澄明内心→体证本然 | 第 16 章"致虚极，守静笃" |
| 自然无为 | 守 | 识别强求→放下控制→让事物自行演化 | 第 25 章"道法自然" |
| 柔弱不争 | 柔 | 辨认争心→守柔处下→以退为进 | 第 36 章"柔弱胜刚强" |
| 生活实践 | 行 | 承认知道→落实行动→行无行 | 第 41 章"上士闻道，勤而行之" |

### 体道链

衡量用户理解深度的六层标尺（spec §4.2，洞察 15）：
`名(L1) → 反(L2) → 有无(L3) → 正言若反(L4) → 无为(L5) → 玄同(L6)`

### 三层递进

每个法门的三层深度分布（spec §4.3，洞察 37/49/51/52）：
- **第一层**：入门 — 识别问题，收敛外求
- **第二层**：深入 — 澄明内心，以柔承刚
- **第三层**：本然 — 体证本然，行无行

### 玄同

产品核心承诺（洞察 8/35/36）。超越亲疏、利害、贵贱的对立，回到事情的本然状态。不是"怎么都行"，而是"看清全局之后的精准回应"。

### 恒德

帛书独有的概念（洞察 33/34）。"恒德"强调不因外在条件变化而动摇的内在稳定性，区别于通行版的"德"。

### 微明

帛书第 36 章"将欲微明"（洞察 25）。让模糊逐渐清晰，不做预设、允许事情在过程中自行明朗化——是产品的核心迭代哲学。

### Open Questioning

产品对话方法论（洞察 22）。区别于传统苏格拉底法——问题不是引导到预设答案的手段，而是打开思考空间的本身。

## 洞察撰写标准结构

### 七节完整操作手册（体道四法系统化洞察）

适用于体道四法相关洞察，必须为七节完整结构：

```
## 洞察{N}：{标题}

**来源**：[§{章节}]

**核心内容**：

{简短摘要}

---

### 一、{第一子节}
### 二、{第二子节}
### 三、{第三子节}
### 四、{第四子节}
### 五、{第五子节}
### 六、{第六子节}
### 七、{第七子节}
```

### 基本结构（非系统化洞察）

适用于非体道四法的洞察：

```
## 洞察{N}：{标题}

**来源**：[§{章节}]

**核心内容**：

{简短摘要}
```

## 编号规则

- 全局递增，从 1 开始，当前最高 65
- 不能跳号、插号
- 按层级分段：1-15 产品层、16-30 架构层、31+ 哲学层（含元层 55-65）
