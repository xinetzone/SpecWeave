# QA 检查清单

## 必过项

- [ ] 是 16:9 横版。
- [ ] 背景是干净纯白底。
- [ ] 有学者。
- [ ] 学者参与核心哲学动作，不只是站在旁边。
- [ ] 没有复刻旧案例构图，为当前文章生成了新的自然隐喻。
- [ ] 画面安静、深远、有哲思感。
- [ ] 简洁清爽，主体不超过画面约 40%-50%。
- [ ] 一张图只讲一个核心哲学结构。
- [ ] 中文标注少、短、能读（3-5 个词，不是句子）。
- [ ] 朱砂红只用于核心概念或主路径（≤ 3 处）。
- [ ] 石青只用于补充说明或反馈状态（≤ 3 处）。
- [ ] 至少 40% 空白。
- [ ] 无道教符号（八卦、拂尘、符箓、丹炉、葫芦）。
- [ ] 无具体历史人物特征。
- [ ] 学者没有看向画面外、没有教育/指示状。

## 失败信号

出现以下情况，重生成或局部编辑：

- [ ] 左上角有"概念图 / 体道四法 / 三层递进 / 体道链"等标题。
- [ ] 学者像道教神仙、历史名人、武术家、禅师。
- [ ] 学者只是在旁边站着看——没有参与隐喻动作。
- [ ] 画面像 PPT、课程课件、国学课堂、传统文化海报。
- [ ] 元素太多、箭头太多、结构太密集。
- [ ] 背景有纸纹、阴影、渐变、米色、印章、落款。
- [ ] 文字变成大段解释或完整句子。
- [ ] 画面太像水墨画、工笔画、CG 古风。
- [ ] 出现了 八卦 / 拂尘 / 丹炉 / 符箓 / 葫芦 / 太极拳 等道教符号。
- [ ] 中文错字严重或标注不可读。
- [ ] 画面太死板、太空洞——空白有，但没有"道"的感觉。
- [ ] 和 `assets/examples/` 里的旧案例构图过于相似。

## 迭代方法

- [ ] **太普通**：让学者变小，让自然元素变大。一个小人在大空中，就有意味了。
- [ ] **太复杂**：删节点，只保留一个哲学动作和一个自然意象。
- [ ] **太像国画**：强调 minimalist hand-drawn line art, not Chinese ink painting, not gongbi style。
- [ ] **太宗教**：强调 not a Daoist priest, not religious, secular ancient scholar, simple robes。
- [ ] **太 PPT**：去掉所有标题、边框、整齐网格。
- [ ] **太像旧案例**：保留核心意思，换一个完全不同的自然意象（水→石、云→竹、山谷→溪）。
- [ ] **学者太装饰**：重写提示词，让学者成为隐喻的主体——他在做那个动作，不是站在动作旁边。
- [ ] **文字太多**：删到 3-5 个关键词。图自己说话。
- [ ] **空白不足**：缩小主体，把背景打开。

## 交付判断

高质量图应该让读者先静下来，然后 1 秒内感受到一个哲学隐喻——不是"看懂"，是"感到"。

如果第一眼像国学课堂配图，或者传统文化展览海报，就不合格。

好的标准：空白很多，学者很小，线条很简，但看完之后心里有一个"东西"停在那里。
