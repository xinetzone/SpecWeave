# 道德经学者 IP

## 角色定义

「学者」是竹简悟道正文配图的固定视觉 IP。形象来源于帛书《老子》的守藏史传统——一个在竹简之间安静思考、观察自然、记录感悟的古代学者。他不是一个具体的历史人物，而是一个姿态的凝结：人在天地之间，面对大道，保持沉默的追问。

默认每张图都要出现学者。学者不是吉祥物，不是装饰，不是古风贴纸，而是正在参与画面核心隐喻的思考者与观察者。

## 外形

- 极简手绘的古代学者轮廓。线条带有竹简刻写的质感——略有不规则的起伏，仿佛用刀笔在竹片上划过。
- 宽袖长袍，衣纹用两三笔断线勾勒，大量留白。不要精细的服饰细节，不要刺绣、花纹、玉佩、冠冕。
- 头部简化为一笔圆弧，面部留白或仅有一个墨点表示垂目。不是写实人物，不是工笔白描，更接近简笔写意。
- 身形可以是站、坐、倚、蹲、背影、侧影。可以是竹简前的剪影、山石旁的轮廓、天地间的一个小点。
- 体量感介于庄重与轻逸之间——像一阵风就能吹动的竹简，又像千年不变的远山。
- 手部简化，持笔时笔与手融为一体，持简时简与手融为一体。

## 性格

- 安静、专注、不急不躁。
- 像一块石头坐在溪边——存在，但不喧嚣。
- 在观察、在思考、在写字，但从不看向画面外的读者。他在自己的世界里。
- 不"教育"读者，不指向文字，不做解释状。他只是存在、观察、记录。
- 偶尔露出一种"说不清"的表情——不是困惑，不是顿悟，是一种悬置的、正在发生的理解。

## 常见姿态

让学者承担画面中的核心哲学动作：

- 独坐山石，望云观水（观道）
- 弯腰拾起一片落叶（微明）
- 竹简前伏案写字（记录感悟）
- 立于岔路口，两侧风景不同（概念解缚）
- 水边照见自己倒影（虚静内观）
- 风吹过，衣袂微动但人不动（无为）
- 一棵弯曲的树旁，学者站得笔直（柔弱胜刚强）
- 竹简散落一地，学者只拾起一片（为道日损）
- 融入远山轮廓，几乎分辨不出（玄同）
- 在流水边放下手中的石头（放下执念）

## 禁止

- 不要把学者画成道教神仙、道士、炼丹者。
- 不要画成孔子、老子或其他具体历史人物。
- 不要画成佛像、罗汉、修行者。
- 不要画成武术大师、太极拳师。
- 不要添加法器、拂尘、葫芦、八卦等道教符号。
- 不要画成装饰性古风插图（水墨渲染、题诗落款、印章）。
- 不要让学者看向画面外、教育状地指着文字。
- 不要让学者变成画面中的大块主体——他常常是一个小点，一个剪影，一个融入场景的存在。
- 不要把学者画得太精致、太工笔、太商业插画感。

## 判断标准

如果把学者从画面中移除，图的哲学隐喻仍然完全成立，说明学者太装饰了——要重写提示词，让学者成为隐喻参与的必有要素。

学者不应该"解释"概念，学者本身就是概念在空间中的凝结。
