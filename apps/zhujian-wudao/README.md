# 竹简悟道 —— 陪你慢慢想透的智慧伙伴

> 以帛书《老子》（1973 年马王堆汉墓出土）为哲学根基的 AI 反思引导工具。
> 不做知识翻译者，不做建议者，只做提问者——通过对话帮助用户自行识破概念陷阱（概念解缚），而非提供答案。

## 项目定位

在所有工具追求"更快更准更多"的时代，竹简悟道刻意追求**更慢、更深、更静**。现代人面对职业转型、重大决策、人际关系困惑时，常常"想不清、无人诉"——内心困惑不是效率问题，回答越快人反而越焦虑。用户需要的不是"给我一个答案"，而是"陪我静下来，把问题慢慢想透"。

## 使用方式

直接在浏览器中打开 `竹简悟道_完整版.html` 即可体验完整原型，无需任何依赖安装。

## 核心设计

- **概念解缚**：AI 对话围绕帛书四条体道路径，帮用户自行识破概念陷阱，而非直接给建议
- **竹简 + 仪式感**：竹简展开那一刻，就是"我要认真思考了"的心理暗示
- **帛书版完整哲学链**：从"反者道之动"到"玄同"境界，让一条完整的思考之路在对话中自然展开
- **每日一问**：以帛书第 48 章"为道者日损"为哲学根基的习惯养成机制

## 初赛 Demo 范围

帛书 81 章原文展示（竹简卷轴 UI）+ AI 反思对话核心功能 + 每日一问机制。
